Gzip 压缩

本页面（HTML）、CSS 和 JS 文件都启用了 Gzip 压缩。打开浏览器开发者工具的 Network 标签，查看 Content-Encoding: gzip 响应头，对比"传输大小"和"实际大小"的巨大差异。

浏览器缓存

CSS 和 JS 文件设置了 7 天强缓存，图片设置了 30 天强缓存。刷新页面，注意 Network 标签中静态资源的请求状态会变成 304 或 "from disk cache"。

而 HTML 页面使用的是协商缓存（no-cache），每次都会先向服务器验证是否有更新。

这张图片缓存了 30 天（demo.png）

访问控制

管理后台 /admin/ 启用了 IP 白名单 + HTTP Basic 认证。内网 IP 范围的用户可以免密码访问，外部用户需要输入用户名和密码。

访问管理后台（需要认证）

限流保护

API 接口 /api/ 启用了请求频率限制：每个 IP 每秒最多 5 个请求（burst=10）。如果短时间内发送大量请求，会收到 503 错误。

点击按钮查看限流效果...

JSON 格式日志

所有访问日志以 JSON 格式输出到 /var/log/nginx/access.log。API 请求额外记录到 /var/log/nginx/api_access.log。健康检查和静态资源请求不记录日志。

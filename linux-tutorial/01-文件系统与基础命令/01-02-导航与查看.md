# 01-02 导航与查看

## 本节你会学到什么

- 用 `pwd` 知道"我在哪里"
- 用 `cd` 在目录树中自由穿梭
- 掌握 `ls` 的常用选项（`-l`、`-a`、`-h`、`-t`、`-r`）
- 理解绝对路径和相对路径的区别
- 学会看懂 `ls -l` 输出的每一列含义

---

上一节我们把 Linux 的目录树比作一个小区。现在的问题是：你站在小区里，怎么知道自己在哪里？怎么走去别的地方？怎么看周围有什么？

在 Windows 里，你打开"文件资源管理器"，上面有个地址栏显示 `C:\Users\张三\文档\`，你一看就知道自己在哪。想换地方，点一下左边的"此电脑"，再双击 D 盘，再进某个文件夹。

在 Linux 的命令行里，没有鼠标可以点。你需要用命令来完成同样的事情。

## pwd -- 我在哪里

`pwd` 是 "print working directory"（打印当前工作目录）的缩写。它只做一件事：告诉你现在在哪个目录下。

```bash
pwd
# 输出: /home/zhangsan
```

这跟你在商场里看楼层指南地图上的"你在这里"红点是一回事。每当你"迷路"了（忘了自己在哪个目录），敲一下 `pwd` 就知道了。

**类比**：`pwd` 就是 GPS 定位。你打开手机地图，按下"定位"按钮，地图上出现一个蓝点--那就是你的当前位置。`pwd` 就是命令行世界的"定位"按钮。

## cd -- 怎么走

`cd` 是 "change directory"（切换目录）的缩写。它让你在目录树的不同节点之间移动。

### 基础移动：绝对路径和相对路径

```bash
# 绝对路径：从根 / 出发，完整写出目的地
cd /home/zhangsan/Documents

# 相对路径：从当前位置出发，走到目的地
# 假设当前在 /home/zhangsan
cd Documents          # 进入当前目录下的 Documents
cd ./Documents       # ./ 明确表示"从当前目录出发"，和上面等价
```

**类比**：绝对路径就像快递单上的完整地址："中国广东省深圳市南山区科技园路 100 号"。相对路径就像你在问路时说的"往前走两个路口左转"。前者不管你在哪，都能找到目的地；后者取决于你现在在哪。

在 Windows 命令行（CMD）里也有 `cd` 命令，但有个关键区别：Windows 用反斜杠 `\`，Linux 用正斜杠 `/`。

```
Windows: cd C:\Users\zhangsan\Documents
Linux:   cd /home/zhangsan/Documents
```

注意看斜杠的方向！这是刚从 Windows 转过来的新手最容易犯的错误--在 Linux 里敲 `cd \home\zhangsan`，系统会以为你要去一个叫 `\home\zhangsan` 的奇怪地方。

### 快捷导航符号

Linux 有几个特殊的目录简写，非常好用：

| 符号 | 含义 | 类比 |
|------|------|------|
| `~` | 当前用户的家目录 | "我家" |
| `/` | 根目录 | "小区大门口" |
| `.` | 当前目录 | "这里" |
| `..` | 上一级目录 | "上一层楼" |
| `-` | 上一次所在目录 | "刚才那个地方" |

```bash
cd ~           # 无论你在哪里，一键回家
cd ~/Documents # 回家后进入 Documents 目录
cd /           # 去根目录（小区大门口）
cd ..          # 返回上一级目录
cd ../..       # 返回上两级目录
cd -           # 回到上一次所在的目录（在两个目录间来回切换的神器）
```

**类比**：`cd -` 就像电视遥控器上的"返回上一频道"按钮。你正在看 CCTV-5 体育频道，切换到 CCTV-1 看新闻，按一下"返回"键就回到 CCTV-5。在命令行里你也经常需要在两个目录之间来回切换（比如在项目代码目录和日志目录之间），`cd -` 就是你的"返回键"。

### 常见场景

```bash
# 场景1：你在 /home/zhangsan，想去 /etc
cd /etc                    # 用绝对路径，一步到位

# 场景2：你在 /home/zhangsan，想去 /home/zhangsan/Downloads
cd Downloads               # 用相对路径，因为 Downloads 就在脚下

# 场景3：你刚从 /etc 出来，想回去
cd -                       # 一键回去

# 场景4：你迷失在某个深层目录里，想回家
cd ~                       # 或直接输入 cd（不带参数也是回家）
```

**注意**：在 Windows 的 CMD 里，输入 `cd` 不带参数会显示当前目录（相当于 `pwd`）。在 Linux 里，`cd` 不带参数等于 `cd ~`，直接回家。这个行为差异需要适应。

## ls -- 这里有什么

`ls`（list，列出）是你最常用、使用频率最高的命令。它每时每刻都在回答一个问题："这个目录里有什么？"

### 基本用法

```bash
ls              # 列出当前目录的文件和子目录
ls /etc         # 列出 /etc 目录下的内容
ls /home /var   # 同时列出多个目录的内容
```

**类比**：`ls` 就像你走到一个房间门口，探头往里看了一眼。房间里有桌子（文件）、有柜子（目录），你用眼睛扫一遍就知道里面有什么。

### 常用选项组合

`ls` 的强大之处在于它的选项（也叫"参数"或"flag"）。选项通常是一个减号后面跟一个字母，比如 `-l`、`-a`。

#### `ls -l` -- 详细列表模式

```bash
ls -l
# 输出类似这样:
# drwxr-xr-x 2 zhangsan users 4096 May 15 10:30 Documents
# -rw-r--r-- 1 zhangsan users  220 May 15 09:15 notes.txt
```

`-l` 是 "long format"（长格式）的缩写。它不再是只显示文件名，而是给每个文件列出一行"身份证信息"。我们来看看每一列的含义：

```
-rw-r--r--  1  zhangsan  users  220  May 15 09:15  notes.txt
(A)         (B) (C)       (D)    (E)  (F)          (G)
```

| 列 | 含义 | 说明 |
|----|------|------|
| A | 文件类型和权限 | 第一个字符 `-`（普通文件）/ `d`（目录）/ `l`（链接），后面 9 个是权限 |
| B | 硬链接数 | 指向这个文件的数据有多少个名字（后面章节详讲） |
| C | 所有者 | 这个文件属于谁 |
| D | 所属组 | 这个文件属于哪个用户组 |
| E | 文件大小 | 默认单位是字节（byte） |
| F | 修改时间 | 最后一次修改的日期和时间 |
| G | 文件名 | 文件叫什么 |

**类比**：`ls` 不加选项相当于你扫了一眼房间里的东西，看到几个模糊的轮廓（只看到文件名）。`ls -l` 相当于你走到每个物品前面，拿起来仔细看标签：所有者是谁、多大、什么时候放的。就像你去图书馆，`ls` 告诉你书架上有什么书（书名），`ls -l` 告诉你每本书的作者、页数、出版日期。

#### `ls -a` -- 显示隐藏文件

```bash
ls -a
```

在 Linux 里，以 `.` 开头的文件和目录是隐藏的。比如 `.bashrc`（Bash 的配置文件）、`.gitignore`（Git 忽略规则）、`.ssh/`（SSH 密钥目录）。默认的 `ls` 不会显示它们，`-a` 让它们现身。

```bash
ls          # 只显示普通文件，不显示隐藏文件
ls -a       # 显示所有文件，包括以 . 开头的隐藏文件
ls -A       # 显示几乎所有文件，但不显示 . 和 ..（推荐）
```

**类比**：Windows 也有隐藏文件（比如 `C:\Users\zhangsan\AppData` 默认是隐藏的），你需要去"文件夹选项"里勾选"显示隐藏的文件"才能看到。Linux 的做法更简单：文件名前面加个点就隐藏了，加 `-a` 就全部显示。

`ls -A` 比 `ls -a` 更常用，因为它排除了 `.`（当前目录）和 `..`（上级目录）这两个"目录自己指向自己"的条目，输出更干净。

#### `ls -h` -- 人类可读的文件大小

```bash
ls -lh
# -rw-r--r-- 1 zhangsan users 2.3K May 15 09:15 notes.txt
# -rw-r--r-- 1 zhangsan users 1.5M May 15 09:15 report.pdf
```

`-h` 是 "human-readable" 的缩写。默认 `ls -l` 显示的文件大小单位是字节，一个大文件可能显示成 `5368709120`，你完全看不清这是多大。加上 `-h`，它会自动转换成 K（千字节）、M（兆字节）、G（吉字节）这些你熟悉的单位。

`-h` 必须和 `-l` 一起使用才有意义（单独使用 `-h` 会被忽略）。

#### `ls -t` -- 按修改时间排序

```bash
ls -lt       # 最新修改的文件排在最前面
ls -ltr      # 最早修改的文件排在最前面（-r 反转排序）
```

`-t` 按修改时间从新到旧排序。这在你找"刚才改了哪个文件"的时候特别有用。配合 `-l` 使用可以看到具体时间。

```bash
# 看看 /var/log 下最近有哪些日志被更新了
ls -lt /var/log | head -10
```

#### `ls -r` -- 反转排序

```bash
ls -lr       # 按字母顺序从 Z 到 A
ls -ltr      # 按时间从旧到新（最旧的在最上面）
```

#### `ls -R` -- 递归列出所有子目录

```bash
ls -R        # 不仅列出当前目录，还列出所有子目录里的内容
```

这个命令会把整个目录树的内容都显示出来。对一个很大的目录（比如 `/etc`）使用 `ls -R` 可能会在屏幕上疯狂滚动好几分钟。一般建议限定范围或配合 `less` 使用（后面章节会讲）。

### 这些选项可以组合

Linux 命令的选项通常可以合并写在一起：

```bash
ls -l -a -h          # 分开写，啰嗦但清晰
ls -lah              # 合并写，简洁
ls -alh              # 顺序无所谓，效果一样
ls -lh -a            # 混合写也没问题
```

上面四条命令的效果完全一样。最常用的组合是 `ls -lah`，它给你：

```
total 32K
drwxr-xr-x 2 zhangsan users 4.0K May 15 10:30 .
drwxr-xr-x 5 zhangsan users 4.0K May 15 09:00 ..
-rw-r--r-- 1 zhangsan users  220 May 15 09:15 .bashrc
-rw-r--r-- 1 zhangsan users 2.3K May 15 09:20 notes.txt
drwxr-xr-x 2 zhangsan users 4.0K May 15 10:30 Projects
```

这条命令是 Linux 用户的肌肉记忆：每到一个新目录，先敲 `ls -lah` 看看里面到底有什么。

## 动手试试

在你的 Linux 终端中完成以下操作：

1. 输入 `pwd`，记下你当前的位置
2. 输入 `cd /`，去到根目录。再输入 `pwd` 确认
3. 输入 `ls`（不带参数）看看根目录有什么
4. 输入 `ls -l` 看看详细的列表
5. 输入 `ls -lah` 看看包括隐藏文件在内的完整信息
6. 输入 `ls -lt /var/log | head -20` 看看最近更新的日志文件
7. 输入 `cd ~` 回家
8. 输入 `cd -` 回到根目录，再输入 `cd -` 回到家。感受一下"返回键"的魅力
9. 输入 `cd /etc`，然后 `ls -d */` 只列出目录而不列出文件
10. 最后输入 `history`，看看你刚才执行了哪些命令

## 本节小结

`pwd` 定位你在哪，`cd` 让你随便走（`~` 回家，`-` 回上一个地方），`ls -lah` 是查看目录内容的万能命令。绝对路径从 `/` 出发，相对路径从当前位置出发。

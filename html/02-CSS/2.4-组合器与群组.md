# 2.4 组合器与群组

## 本节你会学到什么

- 后代选择器（空格）：选中某元素内部的所有后代
- 子代选择器（`>`）：只选直接子元素
- 相邻兄弟选择器（`+`）：选紧挨着的下一个兄弟
- 通用兄弟选择器（`~`）：选后面所有兄弟
- 选择器列表（`,`）：一次选中多个不相关的选择器

## 正文

### 家族树：HTML 的"辈分"关系

在学组合器之前，我们得先理解 HTML 元素之间的关系。看这段代码：

```html
<div>
  <p>第一段</p>
  <p>第二段</p>
  <span>
    <p>第三段（在 span 里面）</p>
  </span>
</div>
```

用家族树来理解这段结构：

```
div （老祖宗）
├── p  （儿子）
├── p  （儿子）
└── span （儿子）
    └── p （孙子——span 的儿子，div 的孙子）
```

- 元素包含另一个元素 → **父子**关系（div 是 p 的父，p 是 div 的子）
- 包含关系可以跨代 → **祖先-后代**关系（div 是所有三个 p 的祖先）
- 同一个爸爸的元素 → **兄弟**关系（前两个 p 和 span 是兄弟）

CSS 组合器就是利用这些"辈分"关系来精确定位元素。

### 后代选择器 —— "div 里面所有的 p"

语法：`A B`（中间用空格隔开）

选中的是 A 里面所有的 B，不管隔了多少代——儿子、孙子、曾孙……只要在 A 里面，统统算数。

```css
div p {
  color: red;  /* div 里面所有的 p，不论深浅 */
}
```

类比：班主任说"一年级的教室里所有的学生"，不管在哪个角落，只要在这个教室里的学生都算。

### 子代选择器 —— "div 的亲儿子 p"

语法：`A > B`（中间用 `>`）

只选直接子元素，孙子辈的不要。

```css
div > p {
  color: blue;  /* 只选 div 的直接儿子 p，孙子辈的不管 */
}
```

类比：班主任说"我直接带的班级的学生"——管不到隔壁班，也管不到孙子辈。

### 相邻兄弟选择器 —— "紧挨着 h1 后面那个 p"

语法：`A + B`（中间用 `+`）

只选紧跟在 A 后面的第一个 B 兄弟。

```css
h1 + p {
  font-size: 18px;  /* 只选紧挨在 h1 后面的那个 p */
}
```

类比：军训的时候"张三右边紧挨着的那个人"——只有那一个。

### 通用兄弟选择器 —— "h1 后面所有的 p 兄弟"

语法：`A ~ B`（中间用 `~`）

选 A 后面所有的 B 兄弟（同一个爸爸，且在 A 后面）。

```css
h1 ~ p {
  color: gray;  /* h1 后面所有的 p 兄弟 */
}
```

类比："张三后面站的所有人"——一排人都算。

### 选择器列表 —— "h1, h2, h3 你们仨，一起过来"

语法：`A, B, C`（用逗号隔开）

如果你想让好几个不相关的选择器共用一套样式，可以用逗号把它们串起来：

```css
h1, h2, h3 {
  font-family: "Georgia", serif;
  color: #333;
}
```

这比写成三遍要省事得多：

```css
/* 又臭又长的写法 */
h1 { font-family: "Georgia", serif; color: #333; }
h2 { font-family: "Georgia", serif; color: #333; }
h3 { font-family: "Georgia", serif; color: #333; }
```

### 四个组合器速查表

| 组合器 | 符号 | 含义 | 记忆口诀 |
|---|---|---|---|
| 后代 | `A B` | A 里所有 B（儿子孙子全算） | 空格最宽，全兜住 |
| 子代 | `A > B` | A 的亲儿子 B | 箭头只指下一代 |
| 相邻兄弟 | `A + B` | 紧挨 A 后面的一个 B | 加号就是"紧接着" |
| 通用兄弟 | `A ~ B` | A 后面所有 B | 波浪打完一排 |

### 组合器也可以叠用

```css
/* div 里面的 ul 直接子元素 li 里面的第一个 a 标签 */
div ul > li a {
  text-decoration: none;
}
```

从左往右读：在 div 里面 → 找到 ul → 它的直接儿子 li → li 里面的 a。虽然初学时不一定要写这么复杂，但读别人代码时要知道怎么解析。

## 动手试试

1. 打开示例文件 `2.4-combinators.html`，观察不同组合器的选中效果
2. 尝试把 `div > p` 改成 `div p`（去掉 `>`），观察多了哪些被选中的元素
3. 在一个列表里写多个 `li`，用 `li + li` 选中"除第一个外的所有 li"（这是一个经典技巧！）
4. 用选择器列表给 h2 和 h3 设置相同的颜色

## 本节小结

组合器让我们能利用 HTML 的家族树关系精准选元素：空格选后代，箭头选儿子，加号紧随兄弟，波浪一排兄弟，逗号分组集合。

# 2.8 背景

## 本节你会学到什么

- `background-color`——背景色
- `background-image`——背景图（`url()` 引用）
- `background-repeat`——平铺方式，像墙纸的铺贴
- `background-position`——图片定位，像相框里照片的位置
- `background-size`——`cover`（铺满）vs `contain`（完整显示）
- `background-attachment`——`fixed` 让背景固定不动
- 简写 `background` 属性

## 正文

### 背景——给盒子铺"墙纸"

你已经知道怎么用 `background-color` 涂色了。但背景远不止涂色——你可以贴图、控制图怎么铺、图放哪里、图跟不跟滚动，就像装修时挑选和铺贴墙纸一样。

### background-color：底色

最基础的操作。给元素涂一个底色：

```css
div {
  background-color: #f0f0f0;
}
```

### background-image：贴上背景图

用 `url()` 函数引入图片，就像给墙贴上带花纹的壁纸：

```css
.hero {
  background-image: url("bg.jpg");
  /* 默认情况下，背景图会像瓷砖一样平铺填满整个元素 */
}
```

> 注意：`url()` 里的路径是相对于 CSS 文件的位置（或者 HTML 文件的位置，如果是内部样式表）。和 `<img>` 的 `src` 规则一样。

### background-repeat：怎么铺

这是"墙纸的铺贴方式"。一个背景图默认会自动重复——横铺满、竖也铺满。

```css
background-repeat: repeat;      /* 默认值：横竖都铺，像地砖 */
background-repeat: repeat-x;    /* 只在水平方向铺 */
background-repeat: repeat-y;    /* 只在垂直方向铺 */
background-repeat: no-repeat;   /* 不铺！只显示一张 */
```

类比：`repeat-x` 就像一排完全一样的画框从左挂到右。`no-repeat` 就是只挂一张画。

### background-position：图片放在哪里

当图片不铺满时，它默认放在元素的**左上角**。你可以指定它放在哪里：

```css
background-position: center;        /* 居中 */
background-position: top right;     /* 右上角 */
background-position: bottom left;   /* 左下角 */
background-position: 50% 50%;       /* 水平 50% + 垂直 50% = 居中 */
background-position: 20px 40px;     /* 距左 20px，距顶 40px */
```

第一个值管水平（left/center/right/百分比/像素），第二个值管垂直（top/center/bottom/百分比/像素）。这个操作就像调整相框里照片的位置。

### background-size：图片尺寸

你贴的墙纸可能太大或太小，可以缩放它：

```css
background-size: auto;        /* 默认：原图多大就多大 */
background-size: cover;       /* 铺满整个容器，可能裁剪图片边缘 */
background-size: contain;     /* 完整显示整张图，可能留白 */
background-size: 200px 100px; /* 精确控制宽高 */
background-size: 50% 50%;     /* 相对于容器的百分比 */
```

`cover` 和 `contain` 是最常用的两个值：

- **cover**：像把一张大照片剪裁后贴满整个相框——"宁可裁边也要铺满"。适合全屏背景图。
- **contain**：像把整张照片缩放到相框里完整显示——"宁可留白也不裁边"。适合产品展示。

### background-attachment：背景是否跟随滚动

当你滚动页面时，背景图的行为有两种：

```css
background-attachment: scroll;  /* 默认：背景跟着元素一起滚动 */
background-attachment: fixed;   /* 背景固定，滚动时背景不动 */
background-attachment: local;   /* 背景跟着元素内容滚动（用于有滚动条的容器内部） */
```

`fixed` 最有用——它让背景固定在视口（浏览器窗口）上，你滚动页面，背景纹丝不动。很多官网的首页大图就是这么做出来的：

```css
.hero-section {
  background-image: url("hero.jpg");
  background-size: cover;
  background-position: center;
  background-attachment: fixed;  /* 背景固定，制造视差效果 */
}
```

### 简写 background 属性

上面这一堆属性，你当然可以一个一个写。但有一个**简写属性** `background`，一次搞定所有：

```css
/* 分开写（冗余） */
.hero {
  background-color: #333;
  background-image: url("bg.jpg");
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
  background-attachment: fixed;
}

/* 简写（简洁） */
.hero {
  background: #333 url("bg.jpg") no-repeat center / cover fixed;
  /*                   色    图          铺       位置  / 大小    滚动 */
}
```

简写格式（各部分用空格隔开，顺序不强制但推荐这个顺序）：

```
background: [color] [image] [repeat] [position] / [size] [attachment]
```

> 注意：`position` 和 `size` 之间要用 `/` 隔开，因为二者都可以用数字表示，不加 `/` 浏览器分不清。
> `size` 必须写在 `position` 后面，且用 `/` 分隔。

### 多背景——一个元素贴多层背景

CSS3 支持一个元素同时有多层背景（用逗号隔开），先写的在最上层：

```css
.multi-bg {
  background:
    url("overlay.png") no-repeat center,
    url("pattern.png") repeat,
    #3498db;
}
```

就像在一面墙上先刷一层漆（底色），再贴一层花纹墙纸（pattern），最后在中间贴一张海报（overlay）。

1. 打开示例文件 `2.8-background.html`，观察不同的背景效果
2. 把 `background-attachment: fixed` 的那段滚动一下，体验固定背景
3. 修改 `background-size` 在 `cover` 和 `contain` 之间切换，感受区别
4. 用简写 `background` 属性重写一个元素的所有背景设置
5. 找一个你喜欢的图片 URL，替换示例中的背景图

## 本节小结

背景五个维度：颜色打底、图片贴图、平铺方式、图片位置、图片尺寸。`fixed` 做视差，`cover` 做全屏，简写 `background` 一次搞定所有。

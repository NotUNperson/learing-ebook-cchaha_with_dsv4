float 浮动演示 —— 从报纸排版到高度塌陷

一、float 的本意 —— 文字环绕图片

浮动图片

这是一段围绕在浮动元素旁边的文字。就像报纸上经常看到的排版：左边放一张图片，右边的文字自然地环绕着图片排列。这段文字会一直往下排，直到超过图片的高度，然后才会延伸到图片下方的整个宽度。

float 最初就是为了实现这种"文字环绕"效果而设计的。你看到的这个蓝色方块用 float:left 浮到了左边，它后面的文字就自动绕到右边去了。

注意看：父容器的虚线边框是不是缩成一条线了？这就是"高度塌陷"！往下看。

总结

概念 作用 比喻
float 元素浮动，文字环绕 叶子浮在水面上，水绕道流
高度塌陷 父元素高度变成 0 气球里的小球飞走了，气球瘪了
clear 强制换到浮动元素下方 "前面有人漂着，我等他们漂完再过"
clearfix 自动修复塌陷 自动在最后放一个隐形托底木板

二、@font-face 自定义字体（语法演示）

如果你的项目需要用到自己下载的字体文件（不用 Google Fonts），可以这样写：

@font-face {
  font-family: "MyCustomFont";          /* 给字体起名 */
  src: url("fonts/MyCustomFont.woff2")  /* 字体文件路径 */
       format("woff2"),
       url("fonts/MyCustomFont.woff")   /* 备选格式 */
       format("woff");
  font-weight: normal;
  font-style: normal;
}

/* 用起来和普通字体一样 */
body {
  font-family: "MyCustomFont", sans-serif;
}

本示例没有自带字体文件，所以这里只是展示语法。实际使用时把 url() 里的路径换成你真实的字体文件即可。

盒模型（上）—— 万物皆盒子

一、盒模型四层结构（从内到外）

下面用嵌套结构展示 CSS 盒模型的四层——颜色对应 Chrome DevTools 中的盒模型面板：

margin（外边距）区域 — 橙色
border（边框）区域 — 黄色
padding（内边距）区域 — 绿色
content（内容区）— 蓝色
padding: 35px（背景色会延伸到 padding 区域）
border 宽度: 8px
margin: 30px（此处在整体布局中表现为间距）

二、content（内容区）

内容区是元素真正"装东西"的地方——文字、图片、子元素。

内容区：width=300px, height=80px

三、padding（内边距）—— 背景色会延伸过来

没有 padding（内容贴着边框）：

没有 padding —— 文字紧贴边缘

加了 padding: 25px：

有 padding: 25px —— 文字和边框之间有"泡沫层"，而且背景色也填满了 padding 区域！

四、border（边框）—— 可见的盒壁

没有 border：

有 padding，但没有边框（边界模糊）

加了 6px 黑色实线 border：

有 6px 黑色边框 —— 边界清晰可见！

五、margin（外边距）—— 盒子之间的间距

下面两个盒子在同一个容器（虚线框）里：

这个盒子 margin: 5px —— 和虚线框、和下面的盒子都贴得很近
这个盒子 margin: 40px —— 和周围的距离大了很多！

六、margin: 0 auto —— 水平居中

我水平居中了！
关键 CSS：width: 400px; margin: 20px auto;
（必须有固定宽度，auto 才能平分剩余空间）

padding / margin 值简写规则

写法含义记忆
padding: 20px; 上下左右都是 20px 1 个值管四边
padding: 20px 40px; 上下 20px, 左右 40px 2 个值：上下、左右
padding: 10px 20px 30px; 上 10, 左右 20, 下 30 3 个值：上、左右、下
padding: 10px 20px 30px 40px; 上 10, 右 20, 下 30, 左 40 4 个值：顺时针（上右下左）

margin 的简写规则和 padding 一模一样。

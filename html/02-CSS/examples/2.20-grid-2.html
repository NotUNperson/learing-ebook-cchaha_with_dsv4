四、Flexbox vs Grid —— 选用原则

用 Flexbox 当...

  • 导航栏（一排链接）
  • 居中对齐一个元素
  • 卡片自动换行列表
  • 表单里的输入框+按钮
  • 不确定数量的项目
  • 只需要一维排列

用 Grid 当...

  • 整页布局（页头+侧栏+内容）
  • 照片墙/产品网格
  • 仪表盘（不规则格子）
  • 表单的多列布局
  • 行列需要精确对齐
  • 需要在行列两个维度控制

核心原则：一维用 Flexbox，二维用 Grid。两者可以嵌套，没有排他性。

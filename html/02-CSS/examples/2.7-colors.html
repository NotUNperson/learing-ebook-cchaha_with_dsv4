CSS 颜色表示法演示

一、颜色关键字

red tomato dodgerblue gold transparent

CSS 支持约 140 种颜色关键字。简单直观，但不够精细。

二、十六进制颜色（#RRGGBB）

#ff0000 纯红 #00ff00 纯绿 #0000ff 纯蓝 #cccccc 灰 #f0f (缩写)

最常用的颜色表示法。每两位十六进制数代表一个通道（红绿蓝），00 最暗 FF 最亮。

缩写规则：当每通道两位相同时可缩写成一位。#ff00ff → #f0f

三、RGB / RGBA

rgb(255, 0, 0) rgb(0, 255, 0) rgb(0, 0, 255) rgb(100,100,100)
rgba(255, 0, 0, 0.3) —— 半透明红色

像操控三盏灯：R 红灯、G 绿灯、B 蓝灯，每盏 0~255。A 是透明度，0 全透明 ~ 1 不透明。

四、HSL / HSLA —— 更符合直觉的调色

色相条（0~360，在色环上转一圈）：

红(0) → 黄(60) → 绿(120) → 青(180) → 蓝(240) → 洋红(300) → 红(360)

hsl(0, 100%, 50%) 纯红 hsl(120, 100%, 50%) 纯绿 hsl(240, 100%, 50%) 纯蓝
hsl(240, 100%, 90%) 浅蓝 —— 只调亮度 hsl(240, 100%, 20%) 深蓝 —— 降亮度 hsl(240, 30%, 50%) 灰蓝 —— 降饱和度
hsla(240, 100%, 50%, 0.3) 半透明蓝

HSL 优势：想调"亮一点的蓝"？改 L。想调"灰一点的绿"？降 S。直觉操作。

五、currentColor —— 跟随当前文字颜色

我的边框颜色用的是 currentColor！
把鼠标移到我上面——文字变红，边框自动跟着变红。

currentColor 让其他属性能跟随 color 属性变化，统一配色超方便。

六、opacity 属性 vs rgba 颜色透明 —— 重要区别！

使用 opacity: 0.5（整体透明）

文字也变透明了

注意：文字也变模糊了！opacity 把整个元素（包括里面的文字）都变透明。

使用 rgba(255,0,0,0.5)（只背景透明）

文字依然清晰！

注意：背景半透明但文字仍然清晰！rgba 只影响这一个颜色属性。

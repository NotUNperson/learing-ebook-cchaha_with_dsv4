class 选择器

这是一段普通文字，但里面有一个 被高亮标记 的词语， 还有一个 重要的强调。

我同时拥有 highlight 和 important 两个 class——背景是黄色，文字是红色粗体。 class 就像贴标签贴纸，想贴几张贴几张。

这是一个普通的 box（class="box"），有边框、内边距和圆角。
我是同时有 box 和 highlight 两个类的元素——边框变成了橙色加粗。 在 CSS 中用 .box.highlight（两个 class 连写，中间没空格）来选中我。

id 选择器

我的 id 是 special-box。id 选择器权重=100，即使 class 里写了其他背景色，id 的背景色也会胜出。 注意：同一个 id 在页面里只能出现一次，就像身份证号。

属性选择器还支持匹配模式：

  • [attr="val"] — 精确等于
  • [attr^="val"] — 以 val 开头
  • [attr$="val"] — 以 val 结尾
  • [attr*="val"] — 包含 val

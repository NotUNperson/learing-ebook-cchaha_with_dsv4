层叠与优先级演示

一、权重谁高听谁的

我是一段普通文字。所有 p 元素的默认颜色是灰色（标签选择器，权重=1）。

我是"highlight"类。class 选择器权重=10，高于标签选择器的 1，所以我是绿色。

我同时有 highlight 和 special 类。p.special 的权重是 11（1个标签+1个class），高于 .highlight 的 10，所以我是紫色。

我的 id 是 special-text。id 选择器权重=100，即使我也有 class，id 权重最高——最终我是红色。

我用了内联样式。内联权重=1000，无视所有选择器规则——我是 dodgerblue（道奇蓝）。

二、同样权重，后来居上

  • 第一项：应该显示什么颜色？
  • 第二项：观察 style 标签里 li 的规则顺序
  • 第三项：后面写的 li { color: darkorange; } 覆盖了前面的 li { color: blue; }

上面三个列表项现在是橙色（darkorange）——因为两条 li 规则权重相同，后面的覆盖前面的。

三、!important —— 无视权重，直接碾压

我同时有内联样式（权重=1000，红色）和 !important（绿色）。内联样式也敌不过 !important——最终我是绿色！

注意：!important 是核武器，日常开发中极少使用，尽量别碰。

优先级从高到低总结

  1. !important —— 终极核武器
  2. 内联样式 style="" —— 权重 1000
  3. id 选择器 #xxx —— 权重 100
  4. class 选择器 .xxx / 属性选择器 / 伪类 —— 权重 10
  5. 标签选择器 / 伪元素 —— 权重 1
  6. 通配符 * —— 权重 0
  7. 继承来的样式 —— 比 0 还低

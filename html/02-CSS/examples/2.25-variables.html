一、按钮使用 CSS 变量

主要按钮（--accent-color） 危险按钮（--danger-color）

按钮的颜色、圆角、内边距全都来自 CSS 变量。改 :root 里的 --accent-color，所有蓝色按钮同时变色。

二、卡片使用 CSS 变量

卡片标题

卡片的背景色、边框、圆角、阴影、内边距全部来自变量。在 :root 里改一个 --card-bg，所有卡片背景同时变。

三、局部覆盖 —— 变量的层叠

蓝色主题盒子

这个盒子用 --box-bg: #eaf2fd 覆盖了全局背景色。里面的文字颜色用了局部 --box-text。

红色主题盒子

同样的 HTML 结构，不同的局部变量。这就是 CSS 变量的层叠——"地方规定"覆盖"国家法律"。

四、calc() 配合 CSS 变量

这个项目的 padding 是 var(--spacing-sm) var(--spacing-md)
整个盒子 padding 是 calc(var(--spacing-md) * 2) = 32px
圆角是 calc(var(--border-radius) / 2) = 4px

CSS 变量可以在 calc() 里做加减乘除。间距变量统一管理后，所有间距保持节奏一致——这就是设计系统的基础。

五、暗色模式

这个页面支持系统暗色模式。如果你的操作系统设置了深色主题，页面会自动切换配色。

原理：在 @media (prefers-color-scheme: dark) 里重新定义 :root 的变量。

切换系统深色/浅色模式后刷新本页面，你会发现 —— CSS 一行不改，颜色全换了。这就是 CSS 变量的威力。

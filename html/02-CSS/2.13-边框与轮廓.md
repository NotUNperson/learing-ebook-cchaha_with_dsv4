# 2.13 边框与轮廓

## 本节你会学到什么

- border 三要素：宽度（width）、样式（style）、颜色（color）
- border-style 的多种样式：solid、dashed、dotted、double 等
- border-radius：从 0 到圆的艺术
- 单边控制：border-top、border-right 等
- outline：边框外的"贴纸"，不占空间，常用于 :focus
- border vs outline 的核心区别

## 正文

### 边框——盒子的"墙"

前面两节我们反复提到 border，现在来深入看看这层"盒壁"到底能玩出多少花样。

border 有三个必不可少的要素：

```css
border: 2px solid #333;
/*       粗细  样式  颜色  */
```

三个要素缺一不可（尤其是 `style`——如果不写样式，边框不会显示，因为默认样式是 `none`）。

### border-style：边框的样子

CSS 提供了 8 种边框样式，但日常用到的也就四五种：

```css
border-style: solid;   /* 实线 —— 最常用，干净利落 */
border-style: dashed;  /* 虚线 —— 像裁纸线 */
border-style: dotted;  /* 点线 —— 圆点连成的线 */
border-style: double;  /* 双线 —— 两条实线，中间留白 */
border-style: groove;  /* 凹槽 —— 3D 凹陷效果（很少用） */
border-style: ridge;   /* 凸脊 —— 3D 凸起效果（很少用） */
border-style: inset;   /* 内嵌 —— 3D 内嵌（很少用） */
border-style: outset;  /* 外凸 —— 3D 外凸（很少用） */
```

类比：
- solid = 画笔画的一条直线
- dashed = 剪刀剪出来的虚线
- dotted = 缝纫机踩出来的点线
- double = 两条平行线，像信封边缘

### border-width：粗细

```css
border-width: 1px;    /* 最常用：细线 */
border-width: 3px;    /* 明显但不夸张 */
border-width: 5px;    /* 粗线，有分量 */
border-width: thin;   /* 细（约 1px） */
border-width: medium; /* 中（约 3px，默认值） */
border-width: thick;  /* 粗（约 5px） */
```

### border-color：颜色

```css
border-color: #333;
border-color: rgba(0, 0, 0, 0.2);  /* 半透明边框 */
border-color: currentColor;         /* 跟文字颜色走 */
```

`currentColor` 作为边框色特别有用——只需改 `color` 就能统一调整边框颜色。

### 单边控制

你不一定要四边相同。CSS 让你分别控制每条边：

```css
/* 方式一：属性名指定 */
border-top: 3px solid #e74c3c;     /* 只有上边框 */
border-right: 2px dashed #3498db;   /* 只有右边框 */
border-bottom: 1px solid #ccc;      /* 只有下边框 */
border-left: 5px solid #27ae60;     /* 只有左边框 */

/* 方式二：在简写里指定 */
border-style: solid none dashed none;   /* 上实线 右无 下虚线 左无 */
border-width: 3px 0 1px 0;
```

最经典的单边用法是**左边框强调**——给引用块或提示信息加一条加粗的有色左边框：

```css
.tip {
  border-left: 4px solid #3498db;
  padding-left: 15px;
  background-color: #f0f7fb;
}
```

### border-radius：圆角

`border-radius` 让盒子的角从直角变成圆弧。这是 CSS 里最"看得见效果"的属性之一——数字变一下，风格大不同。

```css
border-radius: 0;        /* 无圆角（默认）—— 直角，硬朗 */
border-radius: 4px;      /* 微小圆角 —— 软化边缘，最常见 */
border-radius: 8px;      /* 明显圆角 —— 友好圆润 */
border-radius: 16px;     /* 大圆角 —— 胶囊/药片 */
border-radius: 50%;      /* 正圆/椭圆 —— 如果宽高相等就是正圆！ */
```

**做成正圆**的秘诀：元素的 `width == height`，然后设置 `border-radius: 50%`。

```css
.circle {
  width: 80px;
  height: 80px;
  border-radius: 50%;     /* 正圆！ */
  background-color: #3498db;
}
```

`border-radius` 也可以单独控制四个角（顺时针：左上、右上、右下、左下）：

```css
border-radius: 10px 20px 30px 40px;  /* 四个角各不相同 */
border-top-left-radius: 15px;         /* 只改左上角 */
```

### outline：边框外面的"贴纸"

`outline` 和 `border` 很像——都是一圈线。但有本质区别：

| 对比 | border | outline |
|---|---|---|
| 占空间 | 占用布局空间（计算宽度时算上） | **不占空间**（叠在元素外面） |
| 各边独立 | 可以四边不一样 | 四边相同 |
| 圆角 | 受 border-radius 影响 | **不受** border-radius 影响（始终直角） |
| 位置 | 在 padding 外面 | 在 border 外面 |
| 常用场景 | 正常边框 | `:focus` 聚焦指示器 |

outline 最常用的场景是**表单聚焦时的蓝色框**。浏览器默认的 `:focus` 效果就是用 outline 实现的：

```css
input:focus {
  outline: 2px solid #3498db;        /* 聚焦时出现蓝色轮廓 */
  outline-offset: 2px;               /* 轮廓向外偏移一点（与边框之间有空隙） */
}
```

```
┌──────────────────────────────┐
│       outline（轮廓）        │  ← 在 border 外面，不占空间
│  ┌───────────────────────┐  │
│  │    border（边框）     │  │  ← 占空间
│  │  ┌───────────────┐   │  │
│  │  │   padding     │   │  │
│  │  │  ┌───────┐   │   │  │
│  │  │  │content│   │   │  │
│  │  │  └───────┘   │   │  │
│  │  └───────────────┘   │  │
│  └───────────────────────┘  │
└──────────────────────────────┘
```

用比喻来记：border 是"墙"，outline 是"贴在墙外的便利贴"——它存在，但不影响房间（布局）的大小。

### outline-offset：轮廓偏移

让 outline 和 border 之间有空隙：

```css
button:focus {
  outline: 2px dashed #3498db;
  outline-offset: 3px;  /* outline 向外偏移 3px，出现一条可爱的间隙 */
}
```

## 本节小结

border 三要素：width（粗细）、style（样式）、color（颜色）。border-radius 让直角变圆，50% 就是正圆。outline 是"不占空间的额外边界"，专为 :focus 生。

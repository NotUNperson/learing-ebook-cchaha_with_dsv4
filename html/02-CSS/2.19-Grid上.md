# 2.19 Grid（上）—— 二维网格布局

## 本节你会学到什么

- Grid 是什么，和 Flexbox 的本质区别
- 网格容器与网格项目
- `grid-template-columns` / `grid-template-rows` 定义行列
- `fr` 单位（fraction，按比例分配）
- `repeat()` 函数简化重复定义
- `gap` 在 Grid 中的用法
- `minmax()` 函数设置尺寸范围

## 正文

### 从"排队"到"棋盘"

回想一下 Flexbox：它像一个排队系统，一次只能管一个方向——要么横着排，要么竖着排。即使启用了换行，行与行之间也是独立的，互不影响。

Grid 不一样。Grid 是一张**棋盘**——行和列**同时定义**，每个格子都有明确的坐标位置。你把元素放在"第 2 行第 3 列"，它就钉在那。

用一个直白的类比：
- **Flexbox = 火车站的一排售票窗口**：人们排成一排，是单方向的
- **Grid = 电影院的座位**：第几排第几号，行列同时定位，是个二维的网

### 为什么需要 Grid

有些布局用 Flexbox 做起来很别扭。比如你要做一个"三行两列"的仪表盘：
- 第 1 行第 1 列是大图表 A（占两行）
- 第 1 行第 2 列是小图表 B
- 第 2 行第 2 列是小图表 C
- 第 3 行横跨两列是全宽图表 D

用 Flexbox 做这个布局需要嵌套多层，调来调去。用 Grid，直接在"棋盘"上把每个元素放到指定格子就行了。

### 一、开启 Grid

```css
.container {
  display: grid;
}
```

就这么一行。和 Flexbox 一样，父元素变成网格容器，子元素自动变成网格项目。

### 二、grid-template-columns / grid-template-rows —— 画棋盘

开启 Grid 之后，默认只有一列。你需要用 `grid-template-columns` 和 `grid-template-rows` 画出格子的行列：

```css
.container {
  display: grid;
  grid-template-columns: 200px 200px 200px;   /* 三列，每列 200px */
  grid-template-rows: 100px 100px;            /* 两行，每行 100px */
}
```

上面的代码画了一个 **3 列 x 2 行** 的棋盘，共 6 个格子。

### 三、fr 单位 —— 按比例分配

`fr` 是 fraction（碎片/分数）的缩写，是 Grid 里最有用的单位。它表示**按比例分配剩余空间**：

```css
.grid {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;  /* 三列，宽度比 1:2:1 */
}
```

**类比**：三个人分一张饼，1fr + 2fr + 1fr = 总共 4 份。老二分到一半（2份），老大和老三各分四分之一（各1份）。

`fr` 相比固定 `px` 的好处：容器宽度变化时，比例自动保持。这是响应式布局的基础。

### 四、repeat() 函数 —— 不用重复写

当你需要定义很多列时，手动写 `1fr 1fr 1fr 1fr 1fr` 很烦。`repeat()` 就是来做这个的：

```css
/* 下面两行完全等价 */
grid-template-columns: 1fr 1fr 1fr 1fr 1fr;
grid-template-columns: repeat(5, 1fr);

/* repeat 也支持混合 */
grid-template-columns: 200px repeat(3, 1fr) 100px;
/* 第一列 200px，中间三列等分，最后一列 100px */
```

### 五、gap —— 格子之间的间距

`gap` 在 Grid 里的表现和 Flexbox 中一样（它是 CSS 的通用属性）：

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;           /* 行间距和列间距都是 20px */
  gap: 30px 15px;      /* 行间距 30px，列间距 15px */
}
```

也可以用 `row-gap` 和 `column-gap` 分别设置。

### 六、minmax() —— "最小这么大，最大不超那么多"

`minmax()` 让你设定一个列的尺寸范围：

```css
.grid {
  grid-template-columns: repeat(3, minmax(150px, 1fr));
}
```

意思是：每列最少 150px，最多按比例均分。当容器很窄时列不会小到看不见，当容器很宽时列会按比例撑满。

这在响应式设计中非常实用——不用写媒体查询就能实现简单的自适应。

### 七、隐式网格 —— 元素多了怎么办

如果你定义了 3 列 x 2 行（6 个格子），但放了 9 个元素，多出的 3 个元素会怎么处理？

Grid 会自动创建**隐式行**来容纳它们。默认情况下，隐式行的高度由内容决定。你可以用 `grid-auto-rows` 控制隐式行的高度：

```css
.grid {
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 100px 80px;        /* 前两行明确高度 */
  grid-auto-rows: 60px;                  /* 多出来的行默认 60px */
}
```

1. 打开 `examples/2.19-grid-1.html`，观察不同 `grid-template-columns` 定义的效果。
2. 找到 `.grid-fr` 的 CSS，把 `1fr 2fr 1fr` 改成 `1fr 1fr 1fr`（三列均分），刷新观察变化。
3. 试试把 `repeat(3, 1fr)` 改成 `repeat(4, 1fr)`，看看变成四列的样子。
4. 找到 `.grid-minmax`，把浏览器窗口宽度从很窄拖到很宽，观察 `minmax(150px, 1fr)` 的效果——窄时列不会小于 150px，宽时列自动伸展。
5. 数一数 `.grid-implicit` 里有多少个项目，再数一数 CSS 定义了几行。多出的项目的行高由什么决定？

## 本节小结

Grid 是二维棋盘——用 `display: grid` 开启，`grid-template-columns`/`rows` 画格子，`fr` 按比例分配，`repeat()` 减少重复，`gap` 设间距。和 Flexbox 是一维排队、Grid 是二维棋盘。

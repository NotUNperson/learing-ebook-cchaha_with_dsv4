# 2.3 基本选择器

## 本节你会学到什么

- 类型选择器：按标签名选中元素
- class 选择器：按类名选中，可复用
- id 选择器：按唯一标识选中
- 通配符选择器：选中所有元素
- 属性选择器：按属性特征筛选
- 设计原则：多用 class，少用 id

## 正文

### 选择器就是"点名"

你在军训时，教官是怎么叫人的？

- "全体都有！" —— 叫所有人
- "第三排的，出列！" —— 按位置叫
- "张三！" —— 叫某一个人
- "戴眼镜的，举手！" —— 按特征叫

CSS 选择器就是你在样式表里"点名"的方式。你想给谁加样式，就先选中谁。

### 类型选择器 —— 按"物种"选中

最直接的方式：直接叫标签的名字。

```css
p    { color: gray; }      /* 所有段落 */
h1   { font-size: 32px; }  /* 所有一级标题 */
a    { text-decoration: none; }  /* 所有链接 */
```

这就像教官喊"所有男兵出列"——不管你个子高低、哪个班的，只要是男兵都出来。

**适用场景**：给所有同类元素设置基础样式。比如所有段落的行高、所有链接去掉下划线。

### class 选择器 —— 像标签贴纸

class 选择器用 `.` 开头。同一个 class 可以贴到任意多个元素身上——就像便利店的标签贴纸，你可以在矿泉水上贴"特价"，也可以在薯片上贴"特价"。

```css
.highlight {
  background-color: yellow;
  font-weight: bold;
}
```

```html
<p class="highlight">这段高亮</p>
<span class="highlight">这个 span 也高亮</span>
<div class="highlight">这个 div 也高亮</div>
```

一个元素还可以贴多张标签——用空格分开：

```html
<p class="highlight important large">我贴了三张标签</p>
```

这就像一个人可以同时是"学生"、"篮球队员"、"志愿者"。

### id 选择器 —— 像身份证号

id 选择器用 `#` 开头。一个 id 在一个页面里只能出现一次——就像身份证号，每个人的号码是唯一的。

```css
#header {
  background-color: #333;
  color: white;
}
```

```html
<header id="header">这是页面头部</header>
<!-- 下面的写法是错误的！同一个 id 不能出现两次 -->
<!-- <footer id="header">这不行！id 必须唯一</footer> -->
```

### 为什么"多用 class，少用 id"

| 对比维度 | class | id |
|---|---|---|
| 复用性 | 一个 class 可用在多个元素上 | 一个 id 只能用一次 |
| 权重 | 适中（10），容易覆盖 | 太高（100），不便维护 |
| 职责 | 纯粹的样式标识 | 样式 + JS 操作 + 锚点定位 |
| 灵活性 | 一个元素可以有多个 class | 一个元素只能有一个 id |

简单说：**class 像昵称，可以很多人用；id 像身份证号，每人一个，还不好改。** 作为初学者，养成这个习惯：样式全用 class，id 留给 JavaScript 和锚点链接。

### 通配符选择器 —— 选中所有人

`*` 可以选中页面里的**每一个**元素：

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
```

这就像教官喊"全体都有"。常用的场景是**全局重置**——把浏览器自带的各种默认间距清掉，让所有元素从同一起跑线出发。

> 通配符权重为 0，任何其他选择器都能覆盖它。

### 属性选择器 —— 按特征筛选

这就像警察筛选"所有穿红色上衣的人"。你可以根据元素的属性和属性值来选中它：

```css
/* 所有有 type 属性的 input */
input[type] { border: 1px solid gray; }

/* type 属性值正好是 text 的 input */
input[type="text"] { background-color: #f0f0f0; }

/* href 以 "https" 开头的链接 */
a[href^="https"] { color: green; }

/* src 以 ".jpg" 结尾的图片 */
img[src$=".jpg"] { border-radius: 4px; }

/* class 属性中包含 "btn" 的元素 */
*[class*="btn"] { cursor: pointer; }
```

四个常用匹配符记忆法：

- `[attr="val"]` —— 精确等于（exact）
- `[attr^="val"]` —— 以此开头（starts with，尖号 `^` 像一个向上的箭头指着开头）
- `[attr$="val"]` —— 以此结尾（ends with，美元号 `$` 像一个向下的钩子指着结尾）
- `[attr*="val"]` —— 包含这个值（contains）

## 动手试试

1. 打开示例文件 `2.3-selectors.html`，观察不同选择器的效果
2. 试着给某个元素同时加两个 class，然后在 CSS 里分别写出对应样式，看它们如何叠加
3. 用属性选择器给所有 `[target="_blank"]` 的链接加一个特殊样式
4. 思考：为什么说"多用 class，少用 id"？用自己的话写一条理由

## 本节小结

选择器是 CSS 的"点名册"：标签选种类、class 选标签组、id 选唯一、属性按特征筛选、通配符选全部。日常开发中，class 是你的主力工具。

# 2.25 CSS 变量

## 本节你会学到什么

- CSS 自定义属性（变量）的定义和使用
- `:root` 中定义全局变量
- `var()` 函数读取变量值
- 变量的层叠与覆盖（局部覆盖全局）
- 在媒体查询中覆盖变量实现暗色模式
- CSS 变量和 Sass/Less 变量的本质区别

## 正文

### 从"全局搜索替换"到"改一处全站生效"

想象你在做一个网站，品牌色是深蓝色 `#2c3e50`。按钮用这个颜色，标题用这个颜色，导航栏用这个颜色，链接也用这个颜色……这个颜色值在 CSS 里散落了 30 处。

有一天老板说："我觉得深蓝色太闷了，改成墨绿色吧。"

你打开 CSS，全局搜索 `#2c3e50`，一个一个确认，生怕漏掉一个。改完还得检查有没有错改的地方。

这就是 CSS 变量要解决的问题。你可以把颜色定义为一个变量，30 处全用这个变量。改品牌色？改一行就行。

### 一、定义变量 —— `--变量名: 值`

CSS 变量的正式名称叫**自定义属性（Custom Properties）**。变量名以 `--` 开头：

```css
:root {
  --primary-color: #2c3e50;      /* 主色 */
  --secondary-color: #e74c3c;    /* 辅色 */
  --spacing-unit: 8px;           /* 间距单位 */
  --font-size-base: 16px;        /* 基础字号 */
  --border-radius: 8px;          /* 圆角 */
}
```

`--` 就像给一个属性贴了个标签："这个叫 `--primary-color`，值是 `#2c3e50`"。

`::-root` 是什么？它是 CSS 里的"根伪类"，指的就是 `<html>` 元素。在 `:root` 里定义的变量是**全局变量**，页面里任何元素都可以用。

### 二、使用变量 —— `var()`

定义好以后，用 `var()` 函数来读：

```css
.button {
  background-color: var(--primary-color);
  padding: var(--spacing-unit) calc(var(--spacing-unit) * 2);
  border-radius: var(--border-radius);
}
```

注意 `calc(var(--spacing-unit) * 2)` 这种写法——`var()` 可以在 `calc()` 里参与数学运算，非常灵活。

### 三、带默认值的 var()

如果变量可能不存在，可以给一个默认值（fallback）：

```css
.button {
  color: var(--text-color, #333);  /* 如果 --text-color 没定义，就用 #333 */
}
```

### 四、变量的作用域 —— 局部覆盖全局

变量遵循 CSS 的**层叠规则**。在更具体的元素上重新定义变量，会覆盖上级的定义：

```css
:root {
  --box-bg: #fff;      /* 全局：白色背景 */
}

.dark-section {
  --box-bg: #333;      /* 局部：这个区块内覆盖为深色 */
  background: var(--box-bg);
}
```

**类比**：全局变量是"国家的法律"，局部变量是"地方的规定"。地方规定如果和国家法律不一样，在当地按地方的规定来。

### 五、在媒体查询中覆盖变量 —— 暗色模式的原理

这是 CSS 变量的一个杀手级用法。你可以在媒体查询中重新定义变量，实现**主题切换**：

```css
:root {
  --bg-color: #fff;
  --text-color: #333;
  --card-bg: #f5f5f5;
}

/* 用户系统开了暗色模式时 */
@media (prefers-color-scheme: dark) {
  :root {
    --bg-color: #1a1a2e;
    --text-color: #eee;
    --card-bg: #16213e;
  }
}

body {
  background: var(--bg-color);
  color: var(--text-color);
}
```

所有用了这些变量的元素，在暗色模式下自动切换颜色——不用重复写样式！

### 六、CSS 变量 vs Sass/Less 变量

你要是听说过 Sass 或 Less，可能会问：它们也有变量，和 CSS 变量有什么区别？

| 维度 | Sass/Less 变量 | CSS 自定义属性 |
|------|----------------|----------------|
| 定义方式 | `$primary: blue;` | `--primary: blue;` |
| 存在时机 | 编译后消失（变成具体值） | 一直存在于浏览器中 |
| 能否运行时修改 | 不能（已经是静态值了） | 能！JS 可修改，实时生效 |
| 能否被媒体查询覆盖 | 不能（编译时已确定） | 能！层叠规则实时生效 |
| 能否继承 | 不能 | 能！遵循 CSS 层叠规则 |

核心差异：Sass 变量是"编译时烘焙到代码里的"，面包烤好了就不能改。CSS 变量是"运行时的活变量"，浏览器里随时可以改。

**一句话**：CSS 变量是浏览器原生支持的、运行时可动态改变的、遵循层叠规则的真正变量。Sass 变量更像是"编译时的查找替换宏"。

### 七、JavaScript 修改 CSS 变量

这是 CSS 变量最强大的能力之一——用 JS 动态修改：

```javascript
// 读取 CSS 变量
getComputedStyle(document.documentElement).getPropertyValue('--primary-color');

// 修改 CSS 变量 —— 全站即时生效！
document.documentElement.style.setProperty('--primary-color', '#e74c3c');
```

改一行 JS，整个页面的配色全换——这就是 CSS 变量的魅力。

1. 打开 `examples/2.25-variables.html`，观察页面配色。所有颜色、间距都来自 `:root` 中定义的变量。

5. 试着在卡片组件里用 `var(--card-padding, 20px)` 这样的带默认值的写法，然后再在 `:root` 里定义 `--card-padding: 30px`，对比变化。

## 本节小结

CSS 变量用 `--名字` 定义、`var()` 使用。在 `:root` 定义全局变量，遵循层叠规则可局部覆盖。配合媒体查询实现暗色模式，是浏览器原生支持、运行时可动态修改的真正变量。

## 正文

### 从毛坯到精装 —— CSS 篇收官

回顾一下你从第一节走到这里，已经学了：
- 选择器、层叠与优先级
- 盒模型、背景、边框、圆角
- 文字排版、颜色、单位
- display、定位、浮动
- Flexbox 和 Grid 两大布局神器
- 响应式设计、媒体查询
- 过渡、动画、CSS 变量

现在该把它们**全部用上**了。本节是一个完整的综合练习：把一个朴素得像 1998 年的新闻博客页面，打扮成 2026 年应有的样子。

### 设计思路

在动笔之前，先想清楚设计思路——这就是"设计系统"的雏形：

1. **配色方案（CSS 变量）**：选择一个主色（深蓝灰`#2c3e50`）、辅色（蓝色`#3498db`）、背景色、文字色。全部定义为 CSS 变量。
2. **布局方案（Grid + Flexbox）**：整页用 Grid 搭骨架（页头 / 导航 / 内容 / 侧栏 / 页脚），局部用 Flexbox 排细节。
3. **响应式方案**：移动优先，600px 和 900px 两个断点。手机上单列，平板上导航展开，桌面上出现侧边栏。
4. **交互细节**：链接 hover 下划线滑动，卡片 hover 上浮，按钮 hover 变色——全部用 transition。
5. **排版**：rem 统一字号，合适的行高和段落间距，让文字"呼吸"。

### 代码结构

整个页面放在一个文件里（`examples/2.26-final.html`），CSS 写在 `<style>` 标签中。结构分为几个区块：

```
┌─────────────────────────────────────────┐
│  Header（网站标题 + 副标题）             │
├─────────────────────────────────────────┤
│  Nav（导航栏 —— Flexbox）                │
├──────────────────────┬──────────────────┤
│                      │                  │
│  Main Content        │  Sidebar         │
│  - 文章卡片 x 3      │  - 热门文章      │
│  - 每张卡片有标题、  │  - 标签云        │
│    摘要、时间、链接  │                  │
│                      │                  │
├──────────────────────┴──────────────────┤
│  Footer（页脚信息）                      │
└─────────────────────────────────────────┘
```

### 关键 CSS 技术清单

在阅读代码时，请留意以下技术点的实际运用：

- [x] **CSS 变量**：`:root` 中定义了 10+ 个变量，统一配色和间距
- [x] **Grid 布局**：整页骨架用 Grid，`grid-template-areas` 画了地图
- [x] **Flexbox 布局**：导航栏、文章卡片内的元信息用 Flexbox 排列
- [x] **响应式设计**：移动优先，两个断点（600px、900px）切换布局
- [x] **媒体查询**：`@media (min-width: 600px)` 和 `@media (min-width: 900px)`
- [x] **过渡效果**：链接下划线滑动、卡片 hover 上浮、按钮 hover 变色
- [x] **rem 单位**：所有字号和间距使用 rem，便于全局缩放
- [x] **box-shadow**：卡片和导航加了阴影，营造层次感
- [x] **border-radius**：圆角让整体感觉更柔和
- [x] **max-width + margin auto**：内容区居中且有宽度上限

## 动手试试

1. 打开 `examples/2.26-final.html`，通读整个 CSS（约 200 行），对照上面的技术清单，找到每个技术点对应的代码位置。
2. 修改 `:root` 里的 `--primary-color`，比如改成 `#8e44ad`（紫色），刷新看看整个页面的色调变化——这就是 CSS 变量的威力。
3. 新增一篇文章卡片（复制粘贴一篇），观察 Grid 布局是否自动处理了新卡片的位置。
4. 在 600px 断点的媒体查询里，试着把 `grid-template-columns` 改成三列看看效果。
5. 给 `.card` 加一个 `animation: fadeInUp 0.5s ease`，然后自己写一个 `@keyframes fadeInUp`（从下方 30px 淡入）——让文章卡片入场有动画。
6. 这是你自己的作品了！自由修改配色、字体大小、间距，把它变成你喜欢的风格。

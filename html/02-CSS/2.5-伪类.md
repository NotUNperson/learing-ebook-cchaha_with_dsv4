# 2.5 伪类

## 本节你会学到什么

- 什么是伪类——元素在特定状态下的样式
- 动态伪类：`:hover`, `:focus`, `:active`
- 结构伪类：`:first-child`, `:last-child`, `:nth-child()`
- 否定伪类：`:not()` 排除特定元素
- 用 `:nth-child` 做斑马纹表格

## 正文

### 什么是伪类

你之前学的选择器（标签、class、id）选的是元素的**身份**。但有些时候你想根据元素的**状态**来选——"这个链接被鼠标指上去的时候"、"这个输入框正在被输入的时候"、"作为父亲的第一个儿子"。

这些"状态"无法通过 class 或 id 来标记，因为它们随时会变。CSS 提供了**伪类**（pseudo-class）来应对。

"伪类"这个名字有点吓人，拆开想：**"伪"就是"不是真正的 class，但表现得像 class"**。它的语法是冒号 `:` 开头挂在选择器后面：

```css
a:hover {
  color: red;  /* 当鼠标悬停在链接上时，链接变红 */
}
```

### 动态伪类——交互状态

这几个是你日后最常用、也是和用户交互最密切的：

| 伪类 | 触发时机 | 类比 |
|---|---|---|
| `:hover` | 鼠标悬停在元素上 | 你把手放在一本书上，书皮发热的反应 |
| `:focus` | 元素获得焦点（如 input 被点击） | 舞台上聚光灯照到谁 |
| `:active` | 元素被点击的那一瞬间 | 你按下按钮的那一刻 |

```css
/* 基础状态 */
a {
  color: blue;
  text-decoration: none;
}

/* 悬停时——换颜色加下划线 */
a:hover {
  color: red;
  text-decoration: underline;
}

/* 点击瞬间——颜色再变深一点 */
a:active {
  color: darkred;
}

/* 表单获得焦点时——加个蓝色边框，告诉用户"你正在这里输入" */
input:focus {
  outline: 2px solid #3498db;
  background-color: #eef7ff;
}
```

> 提示：`:hover` 对手机触摸屏无效（手指不能"悬停"），移动端开发时要注意。

### 结构伪类——按出生顺序

现在我们回到家族树，根据元素在其父元素中的排位来选中它。

```css
/* 作为第一个子元素 */
li:first-child {
  font-weight: bold;  /* 大儿子有优待 */
}

/* 作为最后一个子元素 */
li:last-child {
  border-bottom: none;  /* 小儿子也有特权 */
}
```

但真正强大的是 `:nth-child()`，它是结构伪类里的瑞士军刀：

```css
/* 偶数项——第 2, 4, 6, 8... 个 */
li:nth-child(even) {
  background-color: #f0f0f0;
}

/* 奇数项——第 1, 3, 5, 7... 个 */
li:nth-child(odd) {
  background-color: white;
}

/* 第 3 项 */
li:nth-child(3) {
  color: red;
}

/* 从第 3 项开始，每 2 项选一次：3, 5, 7... */
li:nth-child(2n+3) {
  color: red;
}
```

`nth-child` 里的公式 `an + b` 怎么理解？

- `n` 从 0 开始，每次加 1：0, 1, 2, 3...
- `2n` 就是 0, 2, 4, 6...（偶数）
- `2n+1` 就是 1, 3, 5, 7...（奇数，等同 odd）
- `3n+1` 就是 1, 4, 7, 10...（每 3 个取第 1 个）

**斑马纹表格**就是 `:nth-child` 最经典的应用：

```css
/* 表格偶数行加灰背景——就像斑马的黑白条纹，引导视线不串行 */
tr:nth-child(even) {
  background-color: #f5f5f5;
}
```

### 否定伪类 —— :not()

"除了……之外，都选"——这就是排除法。

```css
/* 所有段落都缩进，除了最后一个 */
p:not(:last-child) {
  margin-bottom: 20px;
}

/* 所有按钮都圆角，除了取消按钮 */
button:not(.cancel) {
  border-radius: 4px;
}
```

### 其他常用伪类一览

| 伪类 | 含义 |
|---|---|
| `:first-of-type` | 同类型兄弟中的第一个 |
| `:last-of-type` | 同类型兄弟中的最后一个 |
| `:nth-of-type(n)` | 同类型兄弟中的第 n 个 |
| `:empty` | 没有子元素（连文本也没有）的元素 |
| `:checked` | 被选中的复选框或单选按钮 |
| `:disabled` | 被禁用的表单元素 |

> `:first-child` 和 `:first-of-type` 的区别：
> - `p:first-child` —— 选中"作为父元素第一个孩子"的 p。如果第一个孩子不是 p，那就啥也不选！
> - `p:first-of-type` —— 选中"同类型中第一个 p"，不管前面有没有其他类型的兄弟。
>
> 初学时容易踩坑，建议先用 `:first-of-type` 更安全。

## 本节小结

伪类让我们能根据元素的状态（悬停、聚焦、排位等）来添加样式，它不修改 HTML，却让网页的交互感和结构性大幅提升。

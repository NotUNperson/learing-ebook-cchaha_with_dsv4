# 2.9 文字样式

## 本节你会学到什么

- `font-family`：字体系列，后备字体列表
- `font-size`：字号，px / em / rem 单位初步介绍
- `font-weight`：字体粗细（100~900 或 bold/normal）
- `font-style`：斜体
- `line-height`：行高（推荐 1.5~1.8）
- 文字排版：`text-align`, `text-decoration`, `text-transform`
- 字间距、词间距
- web-safe 字体 vs 自定义字体

## 正文

### 文字——网页的灵魂

一个网页 80% 以上的内容是文字。即使是最炫酷的网站，用户最终还是在"读"。所以把文字弄得好看、好读，是 CSS 最重要的基本功之一。

打个比方：文字样式就像你写一封信——选什么字体（毛笔还是钢笔）、写多大（蝇头小楷还是海报大字）、行距多宽（逼仄还是疏朗）、对齐方式（靠左还是居中）。

好，我们逐个来看。

### font-family：用什么字体

你选的字体，别人的电脑上不一定有。怎么办？写一个"备选列表"——就像你推荐餐厅：

> "咱们去 A 餐厅吧。如果 A 没位子就去 B。B 也不行就去 C。实在不行路边随便找家火锅。"

```css
body {
  font-family: "PingFang SC", "Microsoft YaHei", "Hiragino Sans GB", sans-serif;
  /*           苹果苹方       Windows 微软雅黑    Mac 旧版黑体     通用无衬线字体 */
}
```

规则：
- 字体名含空格或中文 → 用引号包起来（`"Microsoft YaHei"`）
- 多个字体用逗号隔开 → 浏览器从左到右找，找到第一个有的就用
- 最后一定要放一个**通用字体族**做保底：`serif`（衬线）、`sans-serif`（无衬线）、`monospace`（等宽）、`cursive`（手写）、`fantasy`（装饰）

简单区分衬线与无衬线：
- **serif（衬线）**：笔画末端有小脚脚装饰，像宋体、Times New Roman。适合打印、长篇阅读
- **sans-serif（无衬线）**：笔画干干净净没有脚脚，像黑体、微软雅黑、Arial。适合屏幕阅读（更清爽）

### font-size：字号

```css
font-size: 16px;   /* 像素：绝对单位，固定大小 */
font-size: 1.2em;  /* em：相对于父元素字号的倍数 */
font-size: 1.2rem; /* rem：相对于根元素（html）字号的倍数 */
```

三个单位的区别是新手常晕的点，用一个例子讲清楚：

```html
<div style="font-size: 20px;">         <!-- 父元素：20px -->
  <p style="font-size: 1.5em;">        <!-- 1.5 × 20px = 30px -->
    <span style="font-size: 1.5em;">   <!-- 1.5 × 30px = 45px（em 层层叠上去，容易失控） -->
    </span>
  </p>
</div>
```

- `px`：绝对值，简单直接。但用户如果调浏览器默认字号，px 的不会跟着变（可访问性稍差）
- `em`：相对父元素。容易层层相乘失控
- `rem`：相对**根元素**（`<html>`），层层嵌套也不怕——推荐！

> 浏览器默认根字号是 16px，所以 `1rem = 16px`，`1.5rem = 24px`。

### font-weight：粗细

```css
font-weight: normal;  /* 常规 = 400 */
font-weight: bold;    /* 粗体 = 700 */
font-weight: 300;     /* 细（Light） */
font-weight: 900;     /* 超粗（Black） */
```

范围是 100 到 900，但能不能用具体数值取决于你的字体文件是否包含对应粗细。大多数 web-safe 字体只有 `normal(400)` 和 `bold(700)` 两种。自定义字体（如下一节要讲的 Google Fonts）通常会提供多种粗细。

### font-style：斜体

```css
font-style: normal;   /* 正常 */
font-style: italic;   /* 真斜体（字体设计师专门设计的斜体字形） */
font-style: oblique;  /* 假斜体（浏览器把正常字体硬生生倾斜） */
```

大多数情况下用 `italic` 就够了。如果字体没有真正的斜体版本，浏览器会自动退化成 oblique。

### line-height：行高——读得舒服的关键

行高是两行文字**基线（baseline）**之间的距离。推荐值：**1.5 ~ 1.8**。

```css
p {
  font-size: 16px;
  line-height: 1.8;  /* 无单位数字：1.8 × 当前字号的 16px = 28.8px */
}
```

为什么用无单位数字（1.8）而不是 px？因为无单位值会继承这个**比例**，子元素不管字号多大，行高都按比例自动调整——省心。

类比：行高就像笔记本的横线格——小学生用宽格（行高大），大学生用窄格（行高小）。网页正文通常在 1.6~1.8 比较舒服。

### 文字排版：对齐、装饰、转换

```css
text-align: left;       /* 左对齐（默认） */
text-align: center;     /* 居中 */
text-align: right;      /* 右对齐 */
text-align: justify;    /* 两端对齐（像报纸栏） */

text-decoration: none;          /* 无装饰 */
text-decoration: underline;     /* 下划线 */
text-decoration: line-through;  /* 删除线 */

text-transform: uppercase;   /* 全部大写 */
text-transform: lowercase;   /* 全部小写 */
text-transform: capitalize;  /* 每个单词首字母大写 */
```

### word-spacing 和 letter-spacing：字间距

```css
letter-spacing: 2px;    /* 字母/汉字之间的间距 */
word-spacing: 4px;      /* 单词之间的间距（对中文无效） */
```

适量的 `letter-spacing` 能让标题更大气，但过大会影响阅读。正文字间距一般不动。

### web-safe 字体 vs 自定义字体

**web-safe 字体**：几乎所有操作系统都预装的字体——Arial, Verdana, Georgia, Times New Roman, Courier New。这些字体你直接写名字就能用，不用担心用户有没有。

**自定义字体**：你自己下载或从 Google Fonts 引入的字体文件（`.woff2`、`.ttf` 等），下一节专门讲。

总结：web-safe 字体是"全国各地都有的连锁店"，自定义字体是"只在当地有的特色小店"——前者可靠，后者独特。

## 动手试试

1. 打开示例文件 `2.9-text.html`，观察不同字体、字号、行高的效果
2. 修改 `font-family` 把 `sans-serif` 换成 `serif`，感受衬线的区别
3. 把 `line-height` 调成 1.2 再调成 2.5，感觉哪个读起来最舒服
4. 用 em 和 rem 分别写一个嵌套结构，看字号的差异
5. 用 `text-align: justify` 给一段长文字两端对齐，看效果

## 本节小结

文字样式七大件：字体（全家+备选）、字号（rem 最稳）、粗细、斜体、行高（1.5-1.8）、对齐、装饰。好读的文字才是好网页。

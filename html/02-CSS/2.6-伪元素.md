# 2.6 伪元素

## 本节你会学到什么

- 什么是伪元素——创建"虚拟的子元素"
- `::before` 和 `::after` 的用法（必须配合 `content` 属性）
- `::first-line` 首行特殊样式
- `::selection` 被选中文字的样式
- 伪类（:）vs 伪元素（::）——单冒号 vs 双冒号的区别

## 正文

### 什么是伪元素

还记得上一节的伪类吗？伪类是选"状态"，伪元素是**造"元素"**——它能在 HTML 里不存在的虚拟元素上施加样式。听起来玄乎，但我们一步步来。

伪元素的语法是**双冒号** `::`（虽然单冒号也常能工作，但规范推荐双冒号以区分伪类）。

```css
/* 伪类：单冒号，选状态 */
a:hover { }

/* 伪元素：双冒号，造元素 */
p::before { }
```

### ::before 和 ::after —— 一对黄金搭档

这是 CSS 里最常用的两个伪元素。它们分别在元素的**内容前后**插入"虚拟内容"。关键规则：

> `::before` 和 `::after` **必须配合 `content` 属性使用**，否则什么都不会显示。

```css
.quote::before {
  content: "\201C";  /* 左双引号的 Unicode 编码 */
  font-size: 2em;
  color: #ccc;
}

.quote::after {
  content: "\201D";  /* 右双引号 */
  font-size: 2em;
  color: #ccc;
}
```

```html
<p class="quote">学而不思则罔，思而不学则殆。</p>
```

最终显示效果：

> "学而不思则罔，思而不学则殆。"

——注意这两个引号不是写在 HTML 里的，而是 CSS 通过 `::before` 和 `::after` 自动生成的。

`content` 属性可以放什么？

| 值 | 示例 | 说明 |
|---|---|---|
| 字符串 | `content: ">> ";` | 直接显示文字 |
| 空字符串 | `content: "";` | 什么都不显示，但创建了伪元素（常用于纯装饰） |
| Unicode | `content: "\00BB";` | 特殊符号，用反斜杠 + 编码 |
| 图片 | `content: url(icon.png);` | 插入图片（但不常用，不如用 background） |
| 属性值 | `content: attr(href);` | 读取元素的某个属性值 |

`attr()` 尤其有用——你可以把元素的属性"挖出来"显示：

```css
a[href^="http"]::after {
  content: " (" attr(href) ")";  /* 链接后面自动显示 URL */
  font-size: 0.8em;
  color: #999;
}
```

为什么用伪元素而不用真实 HTML 元素？

- **不污染 DOM 结构**：纯装饰性的东西（如引号、图标、分隔符）不应该出现在 HTML 里，它们不是内容，是"装扮"
- **动态生成**：比如 `attr()` 读取属性值，内容随属性变化
- **无需修改 HTML**：有时你无法修改 HTML（例如使用第三方系统），但可以加 CSS

### ::first-line —— 首行特殊样式

报纸的第一段首行常有特殊格式化（如小型大写字母）。`::first-line` 让它变得简单：

```css
.article p::first-line {
  font-weight: bold;
  font-size: 1.1em;
  color: #2c3e50;
}
```

> 注意：`::first-line` 只对块级元素有效。哪些属性可以用在 `::first-line` 上有一定限制（字体、颜色、背景等），但不影响日常使用。

### ::selection —— 被选中的文字

用户用鼠标划选文字时的样式：

```css
::selection {
  background-color: #3498db;
  color: white;
}
```

在示例里试一试——选中任意文字，看看背景色是不是变了。

### ::placeholder —— 输入框提示文字

这是表单输入框里 placeholder 文字的样式（注意它用双冒号，虽然写单冒号也行）：

```css
input::placeholder {
  color: #aaa;
  font-style: italic;
}
```

### 伪类 vs 伪元素 总结

| 对比维度 | 伪类 | 伪元素 |
|---|---|---|
| 符号 | 单冒号 `:` | 双冒号 `::` |
| 作用 | 选**状态** | 造**虚拟元素** |
| 数量 | 可以叠多个 | `::before` 和 `::after` 各只能有一个 |
| 依赖 | 取决于用户交互或结构位置 | `::before/::after` 必须配合 `content` |
| 常用例子 | `:hover`, `:nth-child()`, `:focus` | `::before`, `::after`, `::selection` |

> 历史原因：CSS2 时代伪类和伪元素都用单冒号。CSS3 规范把伪元素改成双冒号（以区分），但浏览器为了兼容旧代码，仍然接受单冒号写法。**新项目请用双冒号写伪元素。**

## 动手试试

1. 打开示例文件 `2.6-pseudo-element.html`，观察 `::before` 和 `::after` 插入的内容
2. 修改某个 `::before` 的 `content` 值，换成你想要的文字
3. 缩小浏览器窗口让一段文字换行，观察 `::first-line` 的效果是否仍然只作用在第一行
4. 选中文段文字，看 `::selection` 的效果
5. 尝试用 `content: attr(class);` 在某个元素后面显示它自己的 class 属性值

## 本节小结

伪元素让你在不写 HTML 的情况下凭空创建装饰性元素。`::before/::after` 是最常用的工具，记住必须配合 `content` 属性。伪类选状态，伪元素造元素——二者配合，如虎添翼。

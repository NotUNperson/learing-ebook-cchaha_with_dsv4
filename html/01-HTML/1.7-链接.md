## 正文

### 没有链接的网页，不叫"超文本"

还记得 HTML 全称里的那个 **HyperText（超文本）** 吗？"超"就超在：文字可以变成传送门，点一下就能跳到另一个页面。这是网页和纸质书最本质的区别。

制造传送门的标签叫 `<a>`，a 是 anchor（锚）的缩写。你可以把它想象成一条绳子，一头栓在你的文字上，另一头栓在目标页面上。

### a 标签基本写法

```html
<a href="https://www.bilibili.com">去 B 站看视频</a>
```

- `a` 是标签名。
- `href` 是属性，全称 hypertext reference（超文本引用），说白了就是"跳哪里去"。
- `去 B 站看视频` 是链接文字，是用户看到的、可以点击的部分。

点击这个链接，浏览器就会从当前页面跳转到 B 站。**href 的值就是目标地址。**

### 绝对路径 vs 相对路径

路径就是"怎么找到目标"。分两种：

**绝对路径** -- 像写快递地址：

```html
<a href="https://www.example.com/news/page.html">新闻</a>
```

从协议（https）、域名（www.example.com）、到具体文件路径（/news/page.html），一个不落全写出来。无论你在世界哪个角落，按这个地址都能找到。就像快递单上写"中国北京市朝阳区 XX 路 XX 号"--全球唯一。

**相对路径** -- 像指路：

```html
<!-- 同目录下的文件 -->
<a href="about.html">关于我们</a>

<!-- 子目录下的文件 -->
<a href="pages/contact.html">联系我们</a>

<!-- 上级目录的文件（.. 表示"上一层"） -->
<a href="../index.html">返回首页</a>

<!-- 根目录（从网站根出发） -->
<a href="/help/faq.html">帮助</a>
```

相对路径以**当前文件所在目录**为起点。"出门左转"这种指路方式只在当前位置附近有效。好处是：如果你的整个网站搬了家（换域名），相对路径不用改，绝对路径全得改。

### target="_blank"：在新标签页打开

```html
<a href="https://example.com" target="_blank">打开（新标签页）</a>
```

默认点击链接，浏览器会在**当前标签页**中跳转，原来的页面就没了。加上 `target="_blank"`，浏览器会新开一个标签页，原页面保留。

什么时候用？用户可能还要回来看你的页面时。比如文章里引用了一篇外部资料，用户看完外部资料还要回来继续读文章。

### 锚点：页面内跳转

一本几百页的书，目录旁边都标着页码。想看第三章，翻到第 87 页就行。

HTML 的锚点就干这事：**在长页面里快速跳转到指定位置。**

两步走：

**第一步：在目标位置放一个"书签"（用 id 属性）：**

```html
<h2 id="chapter3">第三章：CSS 入门</h2>
```

**第二步：做一个指向这个书签的链接：**

```html
<a href="#chapter3">跳转到第三章</a>
```

`#` 号 + 目标元素的 id 值。点击这个链接，页面会瞬间滚到 `<h2 id="chapter3">` 的位置。

甚至可以回到页面最顶端：

```html
<a href="#top">回到顶部</a>
```

浏览器默认认识 `#top`，滚动到页面顶部。

### 链接的三种状态

浏览器会给链接自动上色，表示它的状态：

- **未访问（link）**：默认蓝色 + 下划线。你还没点过这个链接。
- **已访问（visited）**：默认紫色。你点过了，浏览器记着呢。
- **悬停（hover）**：鼠标放上去时的样子（颜色往往不变，但指针变成小手）。

这些颜色以后可以用 CSS 自定义。

### a 标签的其他用法

```html
<!-- 下载链接（提示浏览器下载而非打开） -->
<a href="file.pdf" download>下载 PDF</a>

<!-- 邮件链接 -->
<a href="mailto:hello@example.com">给我们发邮件</a>

<!-- 电话链接（手机上点击直接拨号） -->
<a href="tel:13800000000">拨打电话</a>
```

## 动手试试

1. 创建一个 `index.html` 和 `about.html`，让两个页面互相链接。
2. 在其中一个页面里写一个很长的段落（复制粘贴几十行文字），然后做三个锚点链接，分别跳到段落的开头、中间、结尾。
3. 分别用绝对路径和相对路径写链接，感受区别。

---

## 本节小结

a 标签是网页的"传送门"。绝对路径写全址，相对路径说方向。锚点让你在长页面里跳来跳去，像书签一样好用。

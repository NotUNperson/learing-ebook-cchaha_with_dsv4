## 正文

### 网页不能只"展示"，还要"收集"

前面学的都是"输出"：把文字、图片、表格展示给用户看。但网页还需要"输入"：让用户填信息、点按钮、选选项。

这就是表单（form）干的事。登录框、搜索框、评论区、购物车结算……全是表单。

回到填表格那个比喻（1.1 节）：如果你把一份纸质表格给别人填，你需要什么？
- 表格本身
- 每个格子前面的说明（"姓名：_____"）
- 填好之后交给谁

在 HTML 里：
- `form` = 整张表格
- `input` = 每个填写格
- `label` = 填写格前面的说明
- `button` = 交卷按钮

### form 标签：整个表单的容器

```html
<form action="/login" method="POST">
  <!-- 输入控件们都放在这里 -->
</form>
```

两个核心属性：

- `action`：数据提交到哪里。填一个 URL，比如 `/login`（网站服务器上的登录处理程序）。可以理解为"填好的表格交给谁"。
- `method`：数据怎么传过去。有 GET 和 POST 两种方式。

**GET 像明信片**：你把要传的数据写在 URL 后面，所有人都能看到（包括浏览器地址栏、历史记录、服务器日志）。适合搜索（`?q=HTML`）这类不敏感的数据。

**POST 像密封信封**：数据藏在 HTTP 请求体里，URL 上看不到。适合密码、银行卡号等敏感数据。

```html
<!-- 搜索表单：用 GET，数据会出现在 URL 上 -->
<form action="/search" method="GET">
  <input type="text" name="q">
  <button>搜索</button>
</form>

<!-- 登录表单：用 POST，数据藏在信封里 -->
<form action="/login" method="POST">
  <input type="text" name="username">
  <input type="password" name="password">
  <button>登录</button>
</form>
```

### input 标签：各种输入的基础

```html
<input type="text" name="username" placeholder="请输入用户名">
```

input 是空标签。几个关键属性：

- `type`：指定输入框的类型。`text` 是普通文本，后面还会有 `password`、`email` 等。
- `name`：给这个输入框起名字。提交表单时，服务器通过 name 来识别"这个值是哪个字段的"。**name 必须写！**
- `placeholder`：占位提示文字。用户还没输入时，框里显示的灰色提示。不能替代 label！

### label 标签：说明书

```html
<label for="username">用户名：</label>
<input type="text" id="username" name="username">
```

- label 的 `for` 属性要和 input 的 `id` 属性相同，这样它们就"绑定"了。
- 点击 label 文字，浏览器会自动把光标聚焦到对应的输入框。

这有什么用？对于手指粗的手机用户和鼠标操作不精准的老年人来说，点击文字比点击小小的输入框容易得多。

另一种写法：把 input 直接包在 label 里面（不用 for/id）：

```html
<label>
  用户名：
  <input type="text" name="username">
</label>
```

两种都行，推荐第一种（用 for/id），因为更灵活，label 和 input 可以分开放。

### button 标签：提交按钮

```html
<button type="submit">登录</button>
```

button 的 type 有三种：
- `submit`：点击后提交表单，把数据发给服务器。
- `reset`：清空用户填的所有内容（极少用）。
- `button`：普通按钮，自己什么都不干（留给 JavaScript 绑定行为）。

### 一个完整的登录表单

```html
<form action="/login" method="POST">
  <p>
    <label for="username">用户名：</label>
    <input type="text" id="username" name="username" placeholder="请输入用户名">
  </p>
  <p>
    <label for="password">密码：</label>
    <input type="password" id="password" name="password" placeholder="请输入密码">
  </p>
  <p>
    <button type="submit">登录</button>
  </p>
</form>
```

## 本节小结

form（整个表单）+ input（输入格）+ label（说明文字）+ button（提交按钮）构成了表单的基本骨架。GET 是明信片，POST 是密封信封。

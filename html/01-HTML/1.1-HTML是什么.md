# 1.1 HTML 是什么

## 本节你会学到什么

## 正文

### 从"建房子"说起

想象你要盖一栋房子。你需要什么？砖块、水泥、门窗、管道。把这些东西按图纸垒在一起，房子就出来了。

HTML 就是网页的"砖块和图纸"。它的全称是 **HyperText Markup Language**（超文本标记语言）。别被名字吓到，拆开看很简单：

- **HyperText（超文本）**：不只是文字，还能包含链接、图片、视频，点一下就能跳到别的地方。
- **Markup（标记）**：用特定的符号把内容"标记"出来，告诉浏览器"这一段是标题""那一块是段落"。
- **Language（语言）**：它是一套约定好的规则，大家都按规则写，浏览器就都能看懂。

一句话总结：**HTML 是用来描述网页内容和结构的语言。**

### 标签、元素、属性 -- 用"填表格"来理解

这三个词是 HTML 最基础的概念。我们用填表格来类比。

假设你去银行办事，柜员递给你一张表格：

```
┌─────────────────────────────┐
│ 姓名：___________________   │
│ 电话：___________________   │
│ （请用黑色水笔填写）         │
└─────────────────────────────┘
```

这张表格里，"姓名"和"电话"是**要填什么的说明**。表格底下那行"请用黑色水笔填写"是**对填写方式的要求**。你写在横线上的"张三""138xxxx"是**实际内容**。

对应到 HTML 里：

- **标签（Tag）** = 表格的项目名称，比如 `<h1>` 表示"这是一级标题"，`<p>` 表示"这是一个段落"。它是用尖括号包起来的关键词。
- **属性（Attribute）** = 对标签的额外说明。比如 `<a href="https://example.com">` 里的 `href="https://example.com"` 就是属性，告诉浏览器"这个链接指向哪里"。就像表格上的"请用黑色水笔填写"。
- **元素（Element）** = 标签 + 属性 + 内容，合在一起才是一个完整的东西。就像填好的表格：项目名、填写说明、你写的内容，三者合一。

### 标签的样子

```html
<!-- 大多数标签都长这样：开始标签 + 内容 + 结束标签 -->
<p>这是一段文字</p>

<!-- 结束标签比开始标签多一个斜杠 / -->
```

开始标签是 `<p>`，结束标签是 `</p>`，中间夹着内容。这就叫**有内容标签**。

### 空标签 -- 单身的标签

有些标签没有"内容"这一说，比如图片标签，它只是告诉浏览器"在这里放一张图"，不需要在中间夹文字。这种叫**空标签**（也叫自闭合标签）：

```html
<!-- 换行标签：在这里断行 -->
<br>

<!-- 水平分割线 -->
<hr>

<!-- 图片标签：src 是必须的属性，告诉浏览器图片在哪 -->
<img src="photo.jpg" alt="我的照片">
```

空标签不需要结束标签，自己就是完整的元素。

### 嵌套 -- 俄罗斯套娃的哲学

标签可以一层套一层，就像俄罗斯套娃，大娃套小娃：

```html
<div>
  <h2>文章标题</h2>
  <p>这是文章的第一段。</p>
</div>
```

这里 `<div>` 是外层大盒子，里面装着 `<h2>` 和 `<p>` 两个小盒子。

嵌套有几个铁规矩：

1. **后进先出**：先打开的标签要后关闭。就像脱衣服--先脱外套，再脱里面的毛衣。
2. **不能交叉**：`<a><b>文字</a></b>` 是错的，必须 `<a><b>文字</b></a>`。
3. **缩进**：嵌套时用缩进（按 Tab 键），能让你一眼看清谁包着谁。计算机不在乎缩进，但**你自己在乎**。

## 动手试试

打开记事本（或 VS Code），输入下面这段内容，保存为 `test.html`，然后用浏览器打开它：

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>我的第一个网页</title>
</head>
<body>
  <h1>你好，世界！</h1>
  <p>这是我写的第一段 <strong>HTML</strong> 内容。</p>
  <p>换一行<br>接着写。</p>
</body>
</html>
```

试着改一改里面的文字，刷新浏览器看看变化。改错了别怕，不会炸电脑的。

## 本节小结

HTML 是用标签来描述网页内容结构的语言，标签可以带属性，可以嵌套--**记住"填表格"和"俄罗斯套娃"两个比喻就够了**。

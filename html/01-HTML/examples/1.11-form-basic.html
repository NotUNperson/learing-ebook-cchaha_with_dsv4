表单（Form）基础

表单是用户输入数据的区域。登录、搜索、评论——都靠它。

一、GET 方式——明信片

搜索表单用 GET，数据会出现在浏览器地址栏上，像明信片一样公开可见。

试试看：随便输入几个字，点击"搜索"。浏览器地址栏会变成 /search?q=你输入的字。

因为这里没有真实的后端服务器，所以会显示找不到页面——但你能看到 URL 变化，这就是 GET 方式的特征。

二、POST 方式——密封信封

登录表单用 POST，密码不会出现在 URL 上，像密封信封一样安全。

三、label 的两种写法

方式一（推荐）：用 for 和 id 关联

方式二：把 input 包在 label 里面

两种写法效果一样——点击"姓名"就能聚焦输入框。方式一更灵活（label 和 input 可以分开放）。

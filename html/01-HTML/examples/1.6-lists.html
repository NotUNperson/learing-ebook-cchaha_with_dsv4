HTML 列表的三种形态

一、无序列表 ul —— 像购物清单

下面是我的购物清单，每项前面的小圆点是浏览器自动加上的。

  • 鸡蛋
  • 牛奶
  • 全麦面包
  • 苹果

特点：ul 表示这些项是并列关系，顺序不重要。先买鸡蛋还是先买牛奶无所谓。

二、有序列表 ol —— 像操作步骤

下面是泡面的正确步骤，数字序号也是浏览器自动加的。

  1. 撕开包装，取出面饼和调料包
  2. 把面饼放入碗中，倒入调料
  3. 倒入沸水至刻度线
  4. 盖上盖子，等待 3 分钟
  5. 搅拌均匀，开吃！

特点：ol 表示步骤有先后，顺序不能乱。你不可能先吃再泡。

三、描述列表 dl —— 像词典

下面是一个迷你编程词典：

HTML
超文本标记语言（HyperText Markup Language），用来描述网页的结构和内容。
CSS
层叠样式表（Cascading Style Sheets），用来控制网页的外观、颜色、布局。
JavaScript
一种运行在浏览器里的脚本语言，用来给网页添加交互：点击按钮、弹出提示、更新数据等。

特点：dl 是一个"词条-释义"对列表。一个词条（dt）可以有多个释义（dd）。

四、嵌套列表 —— 像文章大纲

列表中套列表，形成层级结构：

  • 第一章：HTML 基础
    • 1.1 HTML 是什么
    • 1.2 文档骨架
    • 1.3 head 元信息
  • 第二章：CSS 入门
    • 2.1 选择器
    • 2.2 颜色与字体
  • 第三章：JavaScript 基础
    • 3.1 变量与类型
    • 3.2 函数
    • 3.3 DOM 操作

注意：内层的 ul 是放在外层 li 的里面的。结构是 ul > li > ul > li，不能写成 ul > ul。

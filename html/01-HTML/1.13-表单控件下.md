# 1.13 表单控件（下）—— 选择类输入

## 本节你会学到什么

- 单选按钮 radio：同一个 name 才能互斥，像单选题
- 复选框 checkbox：每个独立勾选，像多项选择题
- 下拉框 select + option：像折叠起来的单选题
- datalist：半输入半选择，输入时会弹出建议列表
- checked 和 selected：设置默认选中状态

---

## 正文

### 不是所有输入都要敲键盘

上一节讲的都是"敲字"的输入框。但很多场景下，让用户"选择"比"输入"更好：

- 性别：男 / 女（单选）
- 爱好：篮球 / 音乐 / 游戏 / 旅游（多选）
- 省份：从 34 个省里挑一个（下拉框）

这些选择类控件，能让用户更快、更准确地完成输入。

### radio：单选题

```html
<p>请选择你的性别：</p>
<label>
  <input type="radio" name="gender" value="male"> 男
</label>
<label>
  <input type="radio" name="gender" value="female"> 女
</label>
```

radio 就是"单选题"。关键规则：**同一道题的 radio 必须用相同的 name 属性，这样才能互斥。** 如果两个 radio 的 name 不同，它们就可以同时被选中，那就成了多选题了。

checked 属性让某个选项默认选中：

```html
<input type="radio" name="gender" value="male" checked> 男
```

### checkbox：多选题

```html
<p>请选择你的爱好（可多选）：</p>
<label>
  <input type="checkbox" name="hobby" value="basketball"> 篮球
</label>
<label>
  <input type="checkbox" name="hobby" value="music"> 音乐
</label>
<label>
  <input type="checkbox" name="hobby" value="gaming"> 游戏
</label>
```

checkbox 就像考卷上的"多选题"--每个选项独立，可以勾多个，也能全部不勾。

checkbox 也可以加 checked：

```html
<input type="checkbox" name="hobby" value="music" checked> 音乐
```

### select + option：下拉框

```html
<label for="city">所在城市：</label>
<select id="city" name="city">
  <option value="">请选择城市</option>
  <option value="beijing">北京</option>
  <option value="shanghai">上海</option>
  <option value="guangzhou">广州</option>
  <option value="shenzhen">深圳</option>
</select>
```

select 就像折叠起来的单选题。当选项很多时（比如 34 个省份、全世界的国家），用 radio 会把页面拉得很长，select 折叠起来就清爽多了。

selected 属性设置默认选项：

```html
<option value="shanghai" selected>上海</option>
```

**option 的 value 和显示文字可以不同：** `value="beijing"` 是提交给服务器的值（英文、短），而 "北京" 是用户看到的文字。这在实际开发中经常用到。

### datalist：半输入半选择

```html
<label for="browser">常用浏览器：</label>
<input type="text" id="browser" name="browser" list="browser-list">
<datalist id="browser-list">
  <option value="Chrome">
  <option value="Firefox">
  <option value="Safari">
  <option value="Edge">
  <option value="Opera">
</datalist>
```

datalist 是 input 的"建议清单"。用户开始打字时，浏览器会弹出匹配的建议项。用户可以选择一项，也可以坚持自己输入。

需要 input 的 `list` 属性值和 datalist 的 `id` 属性值一致，它们就关联起来了。

把 datalist 想成浏览器的搜索框：你可以直接输入，浏览器也会给你推荐可能的搜索词。

## 动手试试

做一份"用户调查表"，包含：
1. 性别：radio（单选）
2. 年龄段：radio（如 18 以下 / 18-30 / 31-50 / 50 以上）
3. 喜欢的编程语言：checkbox（多选，HTML / CSS / JavaScript / Python / C）
4. 所在城市：select（下拉框，至少 5 个城市）
5. 专业：input + datalist（至少 5 个选项，如计算机、数学、英语等）
6. 全部要有 label

---

## 本节小结

radio 是单选题（同名互斥），checkbox 是多选题，select 是折叠起来的单选，datalist 是带建议的输入框。选择类比输入类更不容易出错。

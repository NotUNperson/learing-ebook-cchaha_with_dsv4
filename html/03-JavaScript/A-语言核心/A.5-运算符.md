# A.5 运算符

## 本节你会学到什么

- 算术运算符：`+` `-` `*` `/` `%` `**`
- 赋值运算符：`=` `+=` `-=` `*=` `/=` `%=`
- 自增自减：`++` `--` 前缀和后缀的区别
- JS 中 `+` 的"双面性"--遇到字符串变成拼接
- JS 除法结果是小数，没有 C 语言的整数除法
- 运算符优先级概览

## 正文

### 一、算术运算符

大部分算术运算符和 C 语言一样：

```javascript
let a = 10;
let b = 3;

console.log(a + b);   // 13  加法
console.log(a - b);   // 7   减法
console.log(a * b);   // 30  乘法
console.log(a / b);   // 3.3333333333333335  除法——注意！结果是小数！
console.log(a % b);   // 1   取余
console.log(a ** b);  // 1000  幂运算（ES6 新增，10 的 3 次方）
```

**几个和 C 不一样的地方**：

1. **除法结果永远是小数（浮点数）**。JS 没有整数除法。`10 / 3` 得到 `3.333...`，而不是 `3`。这和 C 语言完全不同！

2. **`**` 是幂运算符**。`2 ** 10` 表示 2 的 10 次方 = 1024。C 语言里你需要用 `pow(2, 10)` 函数。

3. **`%` 取余**。 `10 % 3 = 1`，和 C 一样。但 JS 中 `%` 对负数也可以操作。

### 二、赋值运算符

赋值运算符和 C 语言几乎一样：

```javascript
let x = 10;

x += 5;   // 等价 x = x + 5  → 15
x -= 3;   // 等价 x = x - 3  → 12
x *= 2;   // 等价 x = x * 2  → 24
x /= 4;   // 等价 x = x / 4  → 6
x %= 4;   // 等价 x = x % 4  → 2
x **= 3;  // 等价 x = x ** 3 → 8
```

### 三、自增自减 -- `++` 和 `--`

和 C 语言一模一样。前缀（先加后用）和后缀（先用后加）的区别要和 C 保持一致：

```javascript
let n = 5;

// 后缀：先返回原值，再加 1
let a = n++;   // a = 5, n = 6

// 前缀：先加 1，再返回新值
let b = ++n;   // n = 7, b = 7
```

```javascript
// 一个帮助理解的例子
let i = 1;
console.log(i++);  // 输出 1（先用，再加）
console.log(i);    // 输出 2

let j = 1;
console.log(++j);  // 输出 2（先加，再用）
console.log(j);    // 输出 2
```

在 `for` 循环里写 `i++` 还是 `++i` 没区别（因为没用到返回值），但在表达式中用到返回值时就要注意了。

### 四、`+` 运算符的"双面性"——JS 的一大特色

这是上一节提到过的重要概念，这里再加深理解。

`+` 在 JS 中有两个角色：
- 当两边都是数字时：做加法
- 当有一个是字符串时：做字符串拼接

```javascript
console.log(10 + 20);        // 30 -- 都是数字，正常加法
console.log("10" + "20");    // "1020" -- 都是字符串，拼接
console.log("10" + 20);      // "1020" -- 有一个字符串，20 被转成 "20" 然后拼接
console.log(10 + 20 + "30"); // "3030" -- 先算 10+20=30，然后 30+"30"="3030"
console.log("10" + 20 + 30); // "102030" -- 先算 "10"+20="1020"，然后 "1020"+30="102030"
```

> 生活类比：`+` 是一个多功能工人。如果两边都是数字，他就做算术；如果看到字符串，他就切换成"贴双面胶"模式，把它们粘在一起。

### 五、运算符优先级

JS 的运算符优先级和 C 语言基本一致。数学课上学的"先乘除后加减"也同样适用。不需要背全表，记住这些常用规则：

1. **先乘除后加减**（和数学一样）
2. **可以用圆括号 `()` 改变优先级**（推荐！别考验自己的记忆力）
3. 赋值运算符优先级很低

```javascript
console.log(2 + 3 * 4);     // 14  不是 20，因为 * 优先于 +
console.log((2 + 3) * 4);   // 20  括号改变优先级

// 不清晰时加括号
let result = (a + b) / (c - d);
```

完整的优先级表格可以在 [MDN](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Operators/Operator_precedence) 上查看。

## 与 C 语言的对比

C 语言的 `/` 运算符在操作两个整数时会做整数除法（`10 / 3` 得到 `3`，小数部分被丢弃），而 JS 的除法**永远返回浮点数结果**（`10 / 3` 得到 `3.333...`）。想要类似 C 的"整除"效果，需要用 `Math.floor()` 或 `Math.trunc()`。另外，C 语言没有 `**` 幂运算符，需要用 `pow()` 函数；JS 的 `**` 和 Python 的风格更接近。`+` 的双面性（数字加法 vs 字符串拼接）是 JS 特有的，C 语言没有类似的自动类型驱动的行为改变。

## 本节小结

- 算术运算符：`+` `-` `*` `/` `%` `**`，和 C 基本一样
- JS 除法永远是小数，没有整数除法
- `**` 是 ES6 的幂运算符
- `++`/`--` 前后缀含义和 C 完全一致
- `+` 在 JS 中有双重身份：遇到字符串变成拼接
- 不记得优先级就用括号，代码清晰比炫技重要

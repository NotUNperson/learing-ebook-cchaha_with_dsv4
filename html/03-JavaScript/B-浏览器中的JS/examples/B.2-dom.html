五、从 C 语言视角看 DOM

如果用 C 语言来实现 DOM 树的节点结构，你可能会这样写：

// C 语言版本的 DOM 节点（伪代码）
typedef struct DomNode {
    char* tagName;              // "p", "div", "h1" ...
    char* textContent;          // 文本内容
    int   nodeType;             // 1=元素, 3=文本, 8=注释
    struct DomNode* parentNode; // 父节点
    struct DomNode* firstChild; // 第一个子节点
    struct DomNode* lastChild;  // 最后一个子节点
    struct DomNode* nextSibling;// 下一个兄弟
    // ... 还有很多属性
} DomNode;

// 遍历子节点（C 语言方式）
DomNode* child = parent->firstChild;
while (child != NULL) {
    printf("%s\n", child->tagName);
    child = child->nextSibling;
}

JS 中遍历子节点则简单得多： parent.childNodes.forEach(node => console.log(node.tagName)); 浏览器已经帮你做好了所有的"malloc"和"free"——JS 程序员只需要用现成的 API。

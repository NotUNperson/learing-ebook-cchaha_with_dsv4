四、Response 对象解析方法

方法返回值适用场景
response.json()JS 对象/数组JSON API（最常见）

response.blob()Blob 对象图片、文件下载

五、fetch 请求流程图

fetch(url)
  |
  +-- [网络成功]
  |     |
  |     +-- response.ok === true  →  response.json() → 使用数据
  |     |
  |     +-- response.ok === false →  手动 throw Error → .catch()
  |
  +-- [网络失败]
        |
        +-- 自动 reject → .catch()

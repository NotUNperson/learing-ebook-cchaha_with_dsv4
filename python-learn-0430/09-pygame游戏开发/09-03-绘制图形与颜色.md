## 正文
### RGB 颜色 —— 三种颜料的混合游戏

回忆一下小时候调水彩颜料——把红色和黄色混在一起变成橙色，把红色和蓝色混在一起变成紫色。电脑屏幕也是这个道理，但它只用三种"原色光"来混合：红（Red）、绿（Green）、蓝（Blue），简称 RGB。

在 pygame 中，颜色用三个数字表示，每个数字范围是 0 到 255：

- `(255, 0, 0)` —— 纯红色（红灯全开，绿灯蓝灯全关）
- `(0, 255, 0)` —— 纯绿色
- `(0, 0, 255)` —— 纯蓝色
- `(255, 255, 255)` —— 纯白色（三种灯全开，最亮）
- `(0, 0, 0)` —— 纯黑色（三种灯全关，没有光）
- `(255, 255, 0)` —— 黄色（红+绿）
- `(128, 128, 128)` —— 灰色（三种灯各开一半）

你可以把每个颜色想象成调音台上的三个推子：红推子、绿推子、蓝推子。推到 255 就是"最大"，拉到 0 就是"关掉"。

### fill() 填充背景 —— 先把画布刷干净

在画任何图形之前，我们通常要先把整个窗口"刷"一遍：

```python
screen.fill((30, 30, 50))  # 深蓝紫色背景
```

这就像画油画之前，先用底色刷一遍画布。不然你的游戏画面会像"透明胶片"一层层叠上去，前一帧的旧画面会残留。

### 画矩形 —— draw.rect()

```python
# 画一个填充的红色矩形
pygame.draw.rect(screen, (255, 0, 0), (50, 50, 150, 100))
#                                  颜色        x,  y,  宽,  高

# 画一个只有边框的绿色矩形
pygame.draw.rect(screen, (0, 255, 0), (250, 50, 150, 100), 5)
#                                                          边框粗细
```

四个参数的含义：
- `screen`：把图形画到哪个"画布"上。
- 颜色：一个 RGB 元组。
- 矩形位置和大小：`(左上角x, 左上角y, 宽度, 高度)`，或者一个 `pygame.Rect` 对象。
- 边框粗细（可选）：不填或填 0 表示填充整个矩形；填一个正整数值表示只画边框，这个数字就是边框的像素宽度。

注意：pygame 的坐标原点在**左上角**（0, 0），x 轴向右，y 轴向下。这和数学课上常见的坐标轴（原点在左下角，y 轴向上）不一样。

### 画圆形 —— draw.circle()

```python
# 画一个填充的黄色圆形
pygame.draw.circle(screen, (255, 255, 0), (125, 250), 60)
#                                      圆心坐标  半径

# 画一个只有边框的青色圆环
pygame.draw.circle(screen, (0, 255, 255), (325, 250), 60, 3)
#                                                        边框粗细
```

圆形的参数是"圆心"和"半径"，比矩形更直观。最后的 width 参数同样是：不填 = 填充，填数字 = 边框粗细。

### 画椭圆 —— draw.ellipse()

椭圆比较有意思——它是内接于一个矩形的。也就是说你给一个矩形，椭圆就是"刚好塞进去的椭圆"。

```python
pygame.draw.ellipse(screen, (255, 0, 255), (400, 200, 180, 100))
#                                         内接矩形 x, y, 宽, 高
```

想象一个长方形框，在里面画一个刚好碰到四条边的椭圆。如果矩形是正方形，椭圆就是正圆；如果矩形是长方形，椭圆就是扁的。

### 画线条 —— draw.line()

```python
pygame.draw.line(screen, (255, 255, 255), (50, 400), (300, 550), 3)
#                                         起点      终点      粗细
```

### 画多边形 —— draw.polygon()

```python
pygame.draw.polygon(screen, (128, 0, 128),
                    [(500, 400), (600, 550), (400, 550)])
#                   三个顶点的坐标列表 → 三角形
```

给一组顶点坐标，pygame 会按顺序连成一个多边形。三角形给三个点，五边形给五个点，以此类推。

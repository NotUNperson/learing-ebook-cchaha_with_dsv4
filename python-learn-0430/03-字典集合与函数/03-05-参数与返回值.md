# 03-05 参数与返回值

## 本节你会学到什么
- 理解参数是什么以及如何使用位置参数
- 使用默认参数让函数更灵活
- 使用 return 语句让函数返回结果

## 正文
### 参数 -- 自动售货机的投币口

你去自动售货机买饮料：投 5 块钱进去（这是**输入**），机器吐出一瓶可乐（这是**输出**）。

函数的参数就是"投币口" -- 你调用函数时"投"进去的数据。函数拿到参数后，在内部处理，然后可能通过 `return` **返回**一个结果给你。

```python
# 自动售货机函数
def buy_drink(money):
    if money >= 5:
        return "可乐"
    else:
        return "钱不够！"

drink = buy_drink(5)
print(drink)  # 可乐

drink = buy_drink(3)
print(drink)  # 钱不够！
```

- `money` 是**参数**（parameter）-- 函数定义时写的"占位符"
- `5` 和 `3` 是**实参**（argument）-- 调用时实际传入的值
- `return` 后面的东西是**返回值** -- 函数处理完还给你的结果

### 位置参数和默认参数

```python
# 两个位置参数 -- 必须按顺序传入
def introduce(name, age):
    print(f"我叫{name}，今年{age}岁")

introduce("小明", 15)  # 正确：我叫小明，今年15岁
# introduce(15, "小明")  # 顺序错了！变成"我叫15，今年小明岁"

# 默认参数 -- 如果调用时不传，就用默认值
def introduce(name, age=18):
    print(f"我叫{name}，今年{age}岁")

introduce("小红", 16)  # 我叫小红，今年16岁 -- 传了就用传的
introduce("小刚")      # 我叫小刚，今年18岁 -- 没传就用默认值
```

有默认值的参数必须放在没有默认值的参数**后面**，这是 Python 的规定。

### return -- 把结果"还"回来

`return` 就像一个快递员：函数处理完后，把结果"打包递还"给调用它的人。

```python
def add(a, b):
    return a + b

result = add(3, 5)  # result 拿到了 8
print(result * 2)    # 16 -- 可以继续使用返回值

# 没有 return 的函数，默认返回 None
def say_hello():
    print("你好")

result = say_hello()  # 打印"你好"
print(result)         # None -- 因为没有 return 任何东西
```

注意区分：`print` 是**显示**在屏幕上给你看，`return` 是**交出**一个值给程序继续用。它们是两件完全不同的事。

## 本节小结
参数是函数的入口（投币口），return 是函数的出口（出货口），两者配合让函数变得灵活强大。

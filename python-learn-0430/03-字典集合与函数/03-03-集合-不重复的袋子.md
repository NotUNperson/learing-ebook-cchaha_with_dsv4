## 正文
### 集合是什么 -- 像书包里掏出来的东西

想象你往书包里放东西：一支笔、一本书、一块橡皮、又一支笔。当你把书包里的东西倒在桌上时，你会发现：笔只有一支 -- 重复的那支不会变成两支。而且倒出来的东西没有先后顺序，你不知道哪支笔是先放进去的。

这就是集合的特点：
1. **不重复** -- 相同的东西只存一份
2. **没有顺序** -- 不能像列表那样用下标 `[0]`、`[1]` 去取
3. **用大括号 `{}` 表示** -- 但要注意，空集合只能用 `set()`，因为 `{}` 被字典占用了

```python
# 创建集合
bag = {"笔", "书", "橡皮", "笔"}  # 放了两支"笔"
print(bag)  # {"笔", "书", "橡皮"} -- 重复的"笔"自动去掉了
print(len(bag))  # 3，不是 4
```

### 集合的增删操作

```python
fruits = {"苹果", "香蕉", "橙子"}

# 添加元素
fruits.add("葡萄")
print(fruits)  # {"苹果", "香蕉", "橙子", "葡萄"}

# 删除元素 -- remove 在元素不存在时会报错
fruits.remove("香蕉")

# 安全删除 -- discard 在元素不存在时也不会报错
fruits.discard("西瓜")  # 西瓜本来就不在集合里，什么都没发生
```

### 集合的运算：交集、并集、差集

这是集合最强大的地方。想象有两个朋友，一个喜欢吃某些菜，另一个喜欢吃另一些菜。

- **交集** `&`：两人都喜欢的菜
- **并集** `|`：全部菜（两人喜欢的所有菜，去重）
- **差集** `-`：只有第一个人喜欢的菜（第二个人不喜欢的）

```python
a_foods = {"鱼香肉丝", "宫保鸡丁", "麻婆豆腐"}
b_foods = {"宫保鸡丁", "麻婆豆腐", "糖醋里脊"}

# 两人都喜欢的菜（交集）
print(a_foods & b_foods)  # {"宫保鸡丁", "麻婆豆腐"}

# 所有菜去重（并集）
print(a_foods | b_foods)  # {"鱼香肉丝", "宫保鸡丁", "麻婆豆腐", "糖醋里脊"}

# 只有 a 喜欢的菜（差集）
print(a_foods - b_foods)  # {"鱼香肉丝"}
```

## 本节小结
集合就像一个会自动去重的袋子，适合用来做"去重"和"求共同部分"这类操作。

# 03-06 变量的作用域

## 本节你会学到什么
- 理解局部变量和全局变量的区别
- 知道在函数内外创建变量有什么不同
- 简单了解 global 关键字

## 正文
### 什么是作用域 -- 教室里的东西 vs 书包里的东西

想象你在教室里：

- 教室的黑板、讲台、空调，**全班同学都能看到和使用** -- 这就像**全局变量**。
- 你自己书包里的文具盒、水杯，**只有你自己能用** -- 这就像**局部变量**。

在 Python 里，作用域（scope）就是决定"某个变量在哪里能被看到、被使用"的规则。

```python
# 这是全局变量 -- 在整个文件里都能用
school_name = "阳光中学"

def show_school():
    # 函数内部可以访问全局变量
    print(school_name)  # 能访问到

show_school()  # 输出：阳光中学
print(school_name)  # 在外面也能访问
```

### 局部变量 -- 函数内部的东西

```python
def my_function():
    my_secret = "只有这里看得到"  # 局部变量
    print(my_secret)  # 函数内部可以访问

my_function()  # 正常工作
# print(my_secret)  # 报错！外面看不到函数里的变量
```

函数内部定义的变量就像你书包里的东西。函数执行结束，这些变量就被"销毁"了，外面根本不知道它们存在过。

### 同名变量不会互相影响

```python
x = 10  # 全局变量 x

def change_x():
    x = 5  # 这是一个新的局部变量 x，和外面的 x 不是同一个！
    print("函数内 x =", x)

change_x()  # 输出：函数内 x = 5
print("函数外 x =", x)  # 输出：函数外 x = 10 -- 全局 x 没有变
```

这就像：教室里的黑板上有"值日生：张三"（全局变量），你在自己笔记本上也写了"值日生：李四"（局部变量）。你改的是你自己的笔记本，黑板上的字不会因此变化。

### global 关键字 -- 想在函数里改全局变量

如果你真的想在函数内部修改全局变量（直接作用于外面的变量），需要用 `global` 关键字声明：

```python
counter = 0  # 全局变量

def increase():
    global counter  # 告诉 Python：我要修改外面那个 counter
    counter = counter + 1

increase()
increase()
print(counter)  # 2 -- 外面的 counter 真的变了
```

不过，**尽量少用 global**。函数最好通过参数接收输入，通过 return 返回结果，而不是去动外部的变量。这样代码更容易理解和维护。

## 动手试试
1. 定义一个全局变量 `version = "1.0"`
2. 定义函数 `show_version()`，在函数内部打印 `version`（不传参）
3. 定义函数 `set_version()`，在函数内部用 `global` 将 `version` 改成 `"2.0"`
4. 依次调用 `show_version()`、`set_version()`、`show_version()`，观察两次输出有什么不同

## 本节小结
函数内部的变量和外部互不影响；要改外部的变量需要用 global，但尽量少用。

# 03-02 字典的增删改查

## 本节你会学到什么
- 使用 keys()、values()、items() 遍历字典
- 使用 get() 安全地访问字典中的值
- 使用 pop() 和 del 删除键值对

## 正文
### 遍历字典 -- 像翻通讯录看所有人

有时候你不是要找某一个人的号码，而是想"把通讯录里所有人都看一遍"。字典提供了三种看的方式：

- 只看名字（键）：用 `keys()`
- 只看号码（值）：用 `values()`
- 名字和号码一起看：用 `items()`

```python
contacts = {"妈妈": "1380000", "爸爸": "1390000", "老师": "1360000"}

# 只看所有人的名字
for name in contacts.keys():
    print(name)  # 妈妈 爸爸 老师

# 只看所有人的号码
for phone in contacts.values():
    print(phone)  # 1380000 1390000 1360000

# 名字和号码一起看
for name, phone in contacts.items():
    print(f"{name} 的电话是 {phone}")
```

### get() -- 给查询加个"备用方案"

之前你学的方式是 `字典[键]`，但键不存在时会直接报错。就像你翻电话本找一个不存在的名字，书不会回答你。

`get()` 方法更温柔：如果键存在就返回值，如果不存在就返回你指定的"默认值"。这就像你问"有小明吗？有就告诉我号码，没有就说'未记录'"。

```python
contacts = {"妈妈": "1380000", "爸爸": "1390000"}

# 用 get 查询 -- 第二个参数是"找不到时返回什么"
print(contacts.get("妈妈", "未记录"))   # 1380000
print(contacts.get("小明", "未记录"))   # 未记录

# 对比：用方括号访问不存在的键会直接报错
# print(contacts["小明"])  # KeyError!
```

### pop() 和 del -- 删除联系人

```python
contacts = {"妈妈": "1380000", "爸爸": "1390000", "老师": "1360000"}

# pop() -- 删除并返回被删除的值
teacher_phone = contacts.pop("老师")
print(teacher_phone)  # 1360000
print(contacts)       # {"妈妈": "1380000", "爸爸": "1390000"}

# del -- 直接删除，不返回任何东西
del contacts["爸爸"]
print(contacts)  # {"妈妈": "1380000"}
```

## 动手试试
1. 创建一个字典 `fruits`，存储几种水果的价格，比如 `{"苹果": 5, "香蕉": 3, "橙子": 4}`
2. 用 `for` 循环和 `items()` 遍历所有水果，打印"XX 每斤 YY 元"
3. 用 `get()` 查询"西瓜"的价格，如果不存在则打印"本店没有西瓜"
4. 用 `pop()` 删除"香蕉"，打印删掉的价格和剩余的水果

## 本节小结
keys、values、items 分别遍历键、值、键值对；get 安全查询；pop 和 del 负责删除。

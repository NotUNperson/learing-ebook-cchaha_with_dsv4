## 正文
### 什么是字典 -- 用电话本做类比

想象你有一个电话本：你要找"小明"的电话号码，你会在电话本里按名字查找，得到一串号码。这个动作在编程里就叫"通过键查值"。

反过来，你不会说"我要找 138xxxx 这个号码是谁的"然后从头翻到尾 -- 电话本的设计目的就是"按名字查号码"。

Python 的字典就是这样一种结构：它存储的是**成对**的东西 -- 一个叫**键**（key），一个叫**值**（value）。你可以把键想象成名字，值想象成电话号码。只要你给出键，字典立刻把对应的值还给你。

字典用大括号 `{}` 表示，每一对"键:值"用冒号分隔，像这样：

```python
phone_book = {"小明": "13800138000", "小红": "13900139000"}
```

这里 `"小明"` 是键，`"13800138000"` 是值，它们组成一对。

### 创建字典和访问值

```python
# 创建一个字典 -- 就像写好一份电话本
scores = {"语文": 90, "数学": 85, "英语": 92}

# 通过键来查值 -- 就像翻电话本找某个人的号码
print(scores["语文"])   # 输出: 90
print(scores["数学"])   # 输出: 85
```

如果用一个字典里不存在的键去访问，Python 会报错。这和你在电话本里找一个不存在的人一样 -- 当然找不到。

```python
# print(scores["体育"])  # 会报错！因为字典里没有"体育"这个键
```

### 添加和修改键值对

```python
scores = {"语文": 90, "数学": 85}

# 添加新的一对 -- 就像在电话本里写上新朋友的名字和号码
scores["英语"] = 92
print(scores)  # {"语文": 90, "数学": 85, "英语": 92}

# 修改已有的值 -- 就像某人的号码换了，你要涂掉旧的写新的
scores["数学"] = 88
print(scores)  # {"语文": 90, "数学": 88, "英语": 92}
```

语法很简单：`字典名[键] = 值`。如果键不存在，就添加；如果键已经存在，就覆盖旧的值。

## 本节小结
字典就像电话本，用键来找值，一对一对地存储数据。

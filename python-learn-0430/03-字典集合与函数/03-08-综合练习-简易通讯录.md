## 正文
### 从"学零件"到"造东西"

前面我们分别学了字典、函数、循环。单独看它们，就像学会了使用锯子、锤子和钉子。但真正的乐趣在于把它们组合起来，做出一个真正有用的东西。

这一节，我们做一个**命令行通讯录**。功能很简单：
- 查看所有联系人
- 添加联系人
- 查找联系人
- 删除联系人
- 退出程序

### 整体思路

用一个字典来存储所有联系人：

```python
contacts = {}  # 空的通讯录
# 添加后长这样：{"张三": "138xxx", "李四": "139xxx"}
```

每个功能写成一个函数，主程序用一个 `while True` 循环显示菜单，让用户选择操作。

### 核心代码框架

```python
def show_menu():
    """显示菜单"""
    print("\n===== 简易通讯录 =====")
    print("1. 查看所有联系人")
    print("2. 添加联系人")
    print("3. 查找联系人")
    print("4. 删除联系人")
    print("5. 退出")
    print("======================")

def show_all(contacts):
    """显示所有联系人"""
    if not contacts:  # 如果字典是空的
        print("通讯录为空")
        return
    for name, phone in contacts.items():
        print(f"  {name}: {phone}")

def add_contact(contacts):
    """添加联系人"""
    name = input("请输入姓名: ")
    phone = input("请输入电话: ")
    contacts[name] = phone
    print(f"已添加 {name}")

# 主程序
contacts = {}
while True:
    show_menu()
    choice = input("请选择操作 (1-5): ")

    if choice == "1":
        show_all(contacts)
    elif choice == "2":
        add_contact(contacts)
    elif choice == "5":
        print("谢谢使用，再见！")
        break
    else:
        print("请输入 1-5 之间的数字")
```

你可以按照这个思路，把剩余的"查找"和"删除"功能补上，做出一个完整的小程序。

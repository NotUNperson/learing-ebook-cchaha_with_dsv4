# 00-02 安装 Python 和运行开发环境

## 本节你会学到什么
- 理解 Python 是什么、为什么要安装它
- 在 Windows 上安装 Python
- 打开 IDLE 并运行第一段代码

## 正文

### 安装 Python 就像买新手机

想一想你拿到一部新手机的过程：

1. 你去店里买手机（**下载 Python**）
2. 回家拆包装、开机、做初始设置（**安装 Python**）
3. 装好之后，你可以打电话、发微信、拍照（**使用 Python 写程序**）

安装 Python 也是同样的三步：下载、安装、使用。安装好之后，你的电脑就"学会"了 Python 这门语言，你可以用它来写程序了。

### 下载和安装 Python

**第一步：下载**

1. 打开浏览器，访问 Python 官网：https://www.python.org
2. 鼠标移到页面顶部的 "Downloads"，它会自动推荐适合你系统的版本
3. 点击那个黄色的大按钮下载安装包（版本号可能是 3.11 或 3.12，都行）

**第二步：安装**

1. 双击你下载的安装包（一个 .exe 文件）
2. **重点**：在弹出的窗口底部，一定要勾选 "Add Python to PATH"（把 Python 加到系统路径里）——这就像把厨房建在客厅旁边，方便随时进去做饭
3. 点击 "Install Now"（立即安装）
4. 等着进度条走完，出现 "Setup was successful" 就说明装好了

> 如果不勾选 "Add Python to PATH"，就像把厨房建在楼顶——使用起来会很麻烦。如果不小心忘了勾，重新运行安装包修复一下就行。

### 认识两个"操作面板"

安装好 Python 之后，你的电脑上多了两个重要的工具：

**IDLE（Python 自带的编辑器）**

IDLE 就像一个**记事本**，但它是专门写 Python 代码的。打开方式：

- 按键盘上的 Win 键，输入 "IDLE"，点击出现的 Python IDLE 图标
- 打开后你会看到一个白色的窗口，上面写着几行字，最后一行是 `>>>`
- 这个 `>>>` 叫做**提示符**，就像在跟你说"我准备好了，你要我做什么？"

在 `>>>` 后面输入：

```python
print("Hello, 世界！")
```

按回车。你会看到计算机立刻回复你：

```
Hello, 世界！
```

恭喜！你已经跟计算机"说上话"了。

**命令行（命令提示符 / PowerShell）**

命令行就像一个**纯文字的操作面板**。没有鼠标点击，只有键盘输入命令。打开方式：

- 按 Win 键，输入 "cmd"（命令提示符）或 "PowerShell"
- 打开后是一个黑色窗口，你也可以在里面输入 `python` 进入 Python 环境
- 输入 `exit()` 可以退出 Python 环境

在 IDLE 和命令行里写 Python，效果是一样的。IDLE 有颜色高亮，更适合写长一点的代码；命令行适合快速试一两行代码。

### 验证安装是否成功

在 IDLE 的 `>>>` 后面输入以下内容，确认每行都没报错：

```python
import sys
print(sys.version)
print("安装成功！")
```

如果看到了版本号（比如 "3.12.0"）和 "安装成功！"，说明一切就绪。

## 动手试试

1. 打开 Python 官网，下载并安装 Python（如果还没装的话）
2. 打开 IDLE，在 `>>>` 后面输入 `print("我的第一个程序")`，按回车，看输出
3. 在 `>>>` 后面输入 `100 + 200`，按回车，看计算结果

## 本节小结

安装 Python 就像给电脑装一个"翻译官"，让它能听懂 Python 语言。IDLE 是你和这个翻译官对话的窗口。

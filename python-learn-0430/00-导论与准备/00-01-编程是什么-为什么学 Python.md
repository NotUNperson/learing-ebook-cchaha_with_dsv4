# 00-01 编程是什么，为什么学 Python

## 本节你会学到什么
- 理解编程的本质是什么
- 了解 Python 这门语言的特点
- 明确学习 Python 能做什么

## 正文

### 编程就像给朋友下厨做菜

想象这样一个场景：你的好朋友要来你家吃饭，你需要做一顿饭。

你首先需要一份**菜谱**——上面写着每一步该怎么做：先洗菜、再切菜、然后开火、放油、翻炒、加调料。这份菜谱就像我们写的**程序**，告诉计算机"一步一步做什么"。

然后你需要**食材**——青菜、肉、调料。这些食材就像**数据**，是程序要处理的东西。

最后，你需要一个**厨师**来执行菜谱——就是你，或者你的家人。这个厨师就是**计算机**，它按照程序的指令一步步执行，把数据加工成结果。

所以，**编程就是写菜谱（程序），让计算机（厨师）处理数据（食材），得到你想要的结果（做好的菜）**。

### 为什么选 Python 这门语言

编程语言有很多种，就像做菜的菜系有川菜、粤菜、日本料理等等。Python 就好比是**家常菜**——简单、好上手、人人能做。

Python 有几个突出的优点：

- **读起来像英文**：Python 的代码非常接近人类的语言，你看一眼大概就能猜到它在干什么。
- **能干的事情多**：做网站、分析数据、写自动化脚本、做人工智能，Python 都能搞定。
- **学起来快**：你不需要先理解复杂的概念才能开始写代码，今天就能写出有用的程序。

用做菜的比喻来说：有的语言像法餐，规规矩矩，讲究繁多；Python 像家常小炒，上手快，味道好，还能天天用。

### 编程到底能做什么

你可能会问："我学会编程后能干什么？" 其实编程已经渗透到生活的方方面面：

- **自动化重复工作**：比如你每天要从网上下载一份报表，整理成 Excel。学会 Python 之后，写一个脚本，一键搞定。
- **做网站**：你每天用的微博、淘宝、B站，背后都是程序在运行。Python 也可以用来写网站的后端。
- **数据分析**：公司里每天产生大量数据，Python 可以帮你从数据中发现规律，做出决策。
- **人工智能**：现在热门的 AI，比如聊天机器人、图像识别，Python 是最主流的语言。

打个比方：学会了编程，就像学会了写字。你不会问"学会写字能干什么"，因为写字能写信、写报告、写小说——而编程也一样，它是一种**创造的工具**。

### 学习路线概览

我们这整套教程会带你从零开始，一步步成为能独立写程序的人：

1. **模块 0**：先搭好环境，写出第一行代码，理解最基本的"变量"概念。
2. **模块 1**：学习 Python 中的各种"数据类型"——数字、文字、列表等等。
3. **模块 2**：学会控制程序的走向——什么时候执行什么代码。
4. **后续模块**：函数、面向对象、文件处理、做游戏、做网站。

每一步都像搭积木，前面的小块是后面大楼的基础。不用着急，稳扎稳打。

## 动手试试

1. 打开你的浏览器，搜索"Python 可以用来做什么"，浏览 3 个搜索结果，建立对 Python 能力的感性认识。
2. 拿出一张纸，写下你学 Python 最想做的一件事（比如"自动下载小说"、"分析我的花销"），贴在显示器旁边——这会是你坚持学下去的动力。
3. 翻到下一节，准备好安装 Python。

## 本节小结

编程就是用计算机能懂的语言写一份"菜谱"，让计算机帮你完成工作。Python 是这份菜谱中最容易上手的一种写法。

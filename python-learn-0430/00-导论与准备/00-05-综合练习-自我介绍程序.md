## 正文

### 前四节学了什么

回顾一下我们走过的路：

1. **编程就像做菜**：程序是菜谱，计算机是厨师
2. **print() 让计算机说话**：把你想显示的内容写进括号里
3. **变量是贴标签的盒子**：存放数据，方便使用

现在我们把它们组合起来，做一个小程序。

### 什么是"组合使用"

你想想做一道番茄炒蛋：你需要知道怎么切番茄、怎么打鸡蛋、怎么开火——这些单独都会。但你要把它们按正确的顺序组合起来，才能做出一道菜。

编程也一样。单独会用 `print()`，单独会创建变量——现在把它们按顺序放在一起，就是一个小程序。

### 程序是"流水线"

程序里的代码，就像工厂的**流水线**——从上到下，一行一行地执行。先执行第一行，再执行第二行，再执行第三行……

```python
print("第一行")
print("第二行")
print("第三行")
```

输出：

```
第一行
第二行
第三行
```

永远不会变成 "第二行\n第一行\n第三行" —— 顺序是确定的。

### 动手做一个自我介绍程序

下面是一个完整的自我介绍程序。它做了这几件事：
1. 用变量存储你的个人信息
2. 用 print() 把信息展示出来
3. 加一些计算（比如出生年份）

```python
# 自我介绍程序
# 先创建变量，存放个人信息
name = "小明"
age = 10
city = "北京"
hobby = "打篮球"
birth_year = 2024 - age   # 用当前年份减去年龄，算出出生年份

# 再用 print() 把这些信息展示出来
print("========== 自我介绍 ==========")
print("大家好，我叫", name)
print("我今年", age, "岁")
print("我来自", city)
print("我的爱好是", hobby)
print("我出生在", birth_year, "年")
print("很高兴认识大家！")
print("================================")
```

**逐行解释：**

- `name = "小明"` — 创建一个叫 name 的盒子，放进"小明"
- `age = 10` — 创建一个叫 age 的盒子，放进数字 10
- `birth_year = 2024 - age` — 用 2024 减去年龄，算出生年份，存到变量里
- `print(...)` — 把变量的内容展示出来，变量名和说明文字用逗号隔开

当你运行这个程序时，计算机会按顺序执行每行代码，最终在屏幕上显示完整的自我介绍。

### 改一下，变成你自己的

你只需要把变量值改成自己的就可以：

```python
name = "你的名字"
age = 你的年龄
city = "你的城市"
hobby = "你的爱好"
```

其他代码完全不用动。这就是变量的好处——**改内容不改形式**。

## 动手试试

1. 把上面的自我介绍代码敲进 IDLE 或一个 .py 文件里，换上你自己的信息，运行看效果
2. 增加两个变量（比如 `food` 喜欢吃什么、`dream` 梦想是什么），并在输出中打印出来
3. 在程序末尾再加上一行，打印一条你自己写的结束语

## 本节小结

把变量和 print() 按顺序组合起来，就能做出第一个有用的程序。程序从上到下依次执行，就像流水线一样确定。

# 00-04 什么是变量

## 本节你会学到什么
- 理解变量是什么
- 学会创建变量和给变量赋值
- 掌握变量命名的规则
- 了解变量可以重新赋值

## 正文

### 变量就像贴了标签的快递盒

想象你在搬家。你有很多东西要打包——书、衣服、厨具等等。

你拿出一个空的**快递盒**，把书放进去，然后在盒子外面贴一张**标签**，写上"书箱"。以后你要找书的时候，看到"书箱"这个标签，就知道里面装的是书。

在 Python 里，这就叫**变量**：

```python
书箱 = "三体小说"
```

- `书箱` 是变量的**名字**（标签）
- `"三体小说"` 是变量的**值**（盒子里的东西）
- `=` 是**赋值符号**，意思是"把右边的东西装进左边的盒子里"

注意：编程里的 `=` 不是"等于"，而是"赋值"——"把右边的东西放到左边这个变量里"。这个理解很重要！

### 变量名就是标签

你可以给变量起各种各样的名字：

```python
name = "小明"        # 名字
age = 10             # 年龄（数字不用引号）
score = 98.5         # 分数（小数也可以）
```

起名字有一些规则，就像给小孩起名不能叫"董事长"一样：

**可以这样做：**
- 用字母、数字、下划线 `_` 组合
- 可以包含中文（但不推荐，因为有些环境可能不兼容）
- 建议用英文单词或拼音

**不能这样做：**
- 不能以数字开头（`1name` 是错的，`name1` 可以）
- 不能包含空格（`my name` 是错的，用 `my_name` 代替）
- 不能用 Python 已经占用的特殊词，比如 `print`、`if`、`for`

```python
# 正确的命名
student_name = "小红"     # 用下划线连接多个单词
age2 = 20                 # 数字可以在中间或末尾
_score = 100              # 下划线开头也可以

# 错误的命名（不要这样写）
# 1name = "小明"           # 不能以数字开头
# my name = "小明"         # 不能有空格
# print = "小明"           # print 是 Python 的关键词，不能用
```

### 盒子的东西可以换

变量的值可以随时改变，就像你可以把盒子里的书拿出来，换成衣服：

```python
box = "书"       # 现在盒子里是书
print(box)       # 输出：书

box = "衣服"     # 换成衣服了
print(box)       # 输出：衣服
```

第一次放东西进去叫**赋值**，之后改内容叫**重新赋值**。变量只记住**最新放进去的**东西。就像快递盒的标签没变，但里面的东西换了——你看到标签"box"，但不知道里面现在是书还是衣服，除非你打开看（也就是用 print 打印出来）。

### 变量可以用来计算

变量不仅能存东西，还能参与计算：

```python
price = 50             # 单价
quantity = 3           # 数量
total = price * quantity  # 总价 = 单价 x 数量
print("总价是", total)   # 输出：总价是 150
```

这就像你先把单价和数量分别写在两张纸条上，要用的时候拿出来做乘法，结果写在第三张纸条上。

## 本节小结

变量就是一个**贴了标签的盒子**——标签是变量名，盒子里的东西是变量的值。你可以随时把东西放进去、拿出来、或者换掉。

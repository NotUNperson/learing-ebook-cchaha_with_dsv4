# 00-03 第一行代码：print() 输出

## 本节你会学到什么
- 用 print() 让计算机显示文字和数字
- 理解函数的基本概念
- 掌握 print() 的几种常用写法

## 正文

### print() 就是让计算机"开口说话"

想象你身边有一个朋友，你可以对他说"你好"，他就会回应你。计算机也是一样——`print()` 就是你让计算机"说出"你想表达内容的方式。

你写在 `print()` 括号里的东西，计算机就会原样显示在屏幕上。这就像你跟朋友说话，对方听到并复述出来。

```python
print("你好")
```

上面这行代码的意思是：让计算机说出"你好"两个字。引号里面的内容就是你要它说的"话"。

### print() 是 Python 的"内置喇叭"

在编程里，`print()` 叫做一个**函数**。函数就像家里的一件**家电**——你给它输入（插上电、放进去食材），它就会输出（微波炉热好饭、洗衣机洗好衣服）。

`print()` 这个"家电"的功能很简单：你给它什么，它就显示什么。

- **输入**：括号里放的内容（要显示的文字、数字等）
- **输出**：在屏幕上显示出来

### 让计算机说不同的"话"

**显示文字（字符串）**

```python
print("Hello World")      # 双引号
print('你好，世界')        # 单引号也可以
print("今天天气真好！")    # 中文完全支持
```

注意：引号必须是英文的 `""` 或 `''`，不能用中文的 `""` 或 `''`。用中文引号会报错——就像你把中文键盘当英文键盘用，打出来的字对方不认识。

**显示数字（不需要引号）**

```python
print(2024)         # 直接写数字
print(3.14)         # 带小数点的也支持
print(100 + 200)    # 甚至可以写算式，计算机会先算出结果
```

当你写 `print(100 + 200)` 时，计算机会先做算术：100 + 200 = 300，然后显示 300。引号里的东西是"原样输出"，不加引号的数字和算式是"先算再输出"。

**一次显示多个内容**

```python
print("Python", "很好学", 2024)
# 输出：Python 很好学 2024
```

用逗号 `,` 分隔多个内容，print() 会把它们显示在同一行，中间自动加空格。这就像一次说了好几句话，中间喘口气。

**显示计算结果**

```python
print("100 + 200 =", 100 + 200)
# 输出：100 + 200 = 300
```

逗号前面是引号里的文字（原样输出），逗号后面是算式（先算再输出）。一个 print 里面可以混合文字和计算。

`print()` 是你和计算机沟通的最基本桥梁——把你想说的话写进去，计算机就在屏幕上显示出来。

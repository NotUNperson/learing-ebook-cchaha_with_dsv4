## 正文
### 读取文件 -- 翻开一本书阅读

你翻开一本书，开始阅读。这个过程分成三步：

1. **打开书** -- 翻开封面
2. **阅读内容** -- 用眼睛扫过一行行文字
3. **合上书** -- 看完放回去

Python 读文件也是这三步：

1. `open()` -- 打开文件（相当于翻开书）
2. `read()` 或 `readlines()` -- 读取内容（相当于阅读）
3. `close()` -- 关闭文件（相当于合上书）

```python
# 第一步：打开文件（'r' 表示 read，只读模式）
file = open("example.txt", "r", encoding="utf-8")

# 第二步：读取内容
content = file.read()  # 一次性读完全部内容
print(content)

# 第三步：关闭文件
file.close()
```

为什么一定要关文件？就像你看完图书馆的书要还回去一样 -- 如果不还，别人就没法借了。在程序里，不关文件可能占着资源，甚至导致数据没保存完整。

### readlines() -- 逐行读

如果你读的是一篇文章，想把每一行分别处理（比如只打印包含某个关键词的行），用 `readlines()` 更方便：

```python
file = open("poem.txt", "r", encoding="utf-8")

lines = file.readlines()  # 返回一个列表，每一项是一行
for line in lines:
    print(line, end="")  # 行末已经有\n了，所以 end=""

file.close()
```

`readlines()` 返回一个列表，文件的每一行是列表的一个元素。这样你就可以用之前学过的列表操作来处理每一行。

### 模式参数说明

```python
open("文件名", "模式", encoding="utf-8")
```

- 第一个参数：文件路径
- 第二个参数：打开模式（'r' = 只读，'w' = 写入，'a' = 追加）
- encoding="utf-8"：指定编码，确保中文不会乱码

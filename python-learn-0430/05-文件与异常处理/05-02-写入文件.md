# 05-02 写入文件

## 本节你会学到什么
- 理解写入模式和追加模式的区别
- 用 'w' 模式覆盖写入文件
- 用 'a' 模式追加写入文件

## 正文
### 写文件 -- 在笔记本上写字

想象你有两个笔记本：
- 一个是**全新的本子**，打开之后你从头开始写，之前的内容全部清空 -- 这是 `'w'`（write）模式
- 另一个是你**正在用的本子**，你翻开最后一页接着写，前面的内容保留 -- 这是 `'a'`（append）模式

### 'w' 模式：覆盖写入（清空重写）

```python
# 'w' 模式：如果文件存在，清空内容后从头写；如果不存在，创建新文件
file = open("note.txt", "w", encoding="utf-8")
file.write("第一行内容\n")
file.write("第二行内容\n")
file.close()
```

每次用 `'w'` 模式打开文件，文件里的旧内容就会被清空。就像你撕掉笔记本上所有写过字的纸，从第一页重新开始写。

**小心**：`'w'` 模式会清空已有内容。如果你只是想加几行，别用 `'w'`！

### 'a' 模式：追加写入（接着写）

```python
# 第一次运行：创建文件，写两行
file = open("note.txt", "a", encoding="utf-8")
file.write("第一行\n")
file.write("第二行\n")
file.close()

# 第二次运行：在文件末尾接着写，不会删之前的
file = open("note.txt", "a", encoding="utf-8")
file.write("第三行\n")
file.write("第四行\n")
file.close()
# 现在 note.txt 里有四行
```

`'a'` 模式就像你每天写日记，不会把昨天的内容删掉，而是翻到最后一页继续写。

### write() 和 writelines()

```python
# write()：写一个字符串
file = open("fruits.txt", "w", encoding="utf-8")
file.write("苹果\n")
file.write("香蕉\n")
file.close()

# writelines()：一次性写一个列表里的所有字符串
fruits = ["橙子\n", "葡萄\n", "西瓜\n"]
file = open("fruits.txt", "a", encoding="utf-8")
file.writelines(fruits)
file.close()
```

注意：`write()` 和 `writelines()` 都**不会自动加换行符**，你需要自己在字符串末尾加 `\n`。

## 动手试试
1. 用 `'w'` 模式创建一个 `my_note.txt`，写入你的名字和年龄
2. 用 `'a'` 模式追加一行"我今天学习了 Python 文件操作"
3. 再用 `read()` 读取这个文件，确认内容是否正确
4. 再跑一次 `'w'` 模式的写代码（写不同的内容），然后再读取，观察旧内容是否被清空了

## 本节小结
'w' 清空重写（撕掉旧纸），'a' 末尾追加（接着写），write 不会自动换行要手动加 \n。

## 正文
### with 语句 -- 自动门

你在商场门口见过自动门吗？你走近时它自动打开，你走进去后它自动关上。你不需要伸手关门。

Python 的 `with` 语句就像自动门：
- 进入 `with` 代码块，文件**自动打开**
- 离开 `with` 代码块，文件**自动关闭**
- 即使中间出了错，门也会关上！

### 旧方式 vs with 方式

```python
# 旧方式：需要手动 close，容易忘
file = open("test.txt", "r", encoding="utf-8")
content = file.read()
print(content)
file.close()  # 经常有人忘了写这一行！

# with 方式：自动关闭，绝不会忘
with open("test.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)
# 出了这个缩进块，文件已经自动关闭了
```

用 `with`，你不需要写 `file.close()`。Python 会在你离开 `with` 的缩进块时自动帮你关闭文件 -- 万无一失。

### with 语句的完整语法

```python
with open("文件名", "模式", encoding="utf-8") as 变量名:
    # 在这个缩进块里使用文件
    变量名.read()
    变量名.write(...)
# 出了缩进块，文件自动关闭
```

`as` 后面的变量名可以随便起（常用的是 `f` 或 `file`），它代表被打开的文件对象。

### 读和写都用 with

```python
# 用 with 读文件
with open("input.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()
    for line in lines:
        print(f"读到: {line}", end="")

# 用 with 写文件
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("用 with 写入的内容\n")
    f.write("简单又安全\n")

# 用 with 追加文件
with open("output.txt", "a", encoding="utf-8") as f:
    f.write("追加一行，文件仍然会自动关闭\n")
```

### 从今天开始，只用 with

学到这里，你应该已经认识到了 `with` 的好处。给你一个建议：今后操作文件，**永远用 `with`，不要手动 `open()` + `close()`**。代码更短，更安全，不会出错。

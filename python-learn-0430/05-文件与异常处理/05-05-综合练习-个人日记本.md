### 我们要做什么

做一个命令行日记本程序，功能包括：
- **写日记** -- 输入日期和内容，保存到文件
- **查看所有日记** -- 列出所有日记的标题（日期）
- **按日期查看** -- 查看某一天的日记全文
- **退出** -- 保存并退出

日记存在一个文本文件里，格式如下：

```
=== 2024-04-28 ===
今天开始学习 Python 的文件操作
感觉很有成就感！

=== 2024-04-29 ===
学了 with 语句，文件操作变得好简单
```

### 核心：读取日记文件

```python
def load_diary():
    """从文件读取日记，返回字典 {日期: 内容}"""
    diary = {}
    try:
        with open(DIARY_FILE, "r", encoding="utf-8") as f:
            current_date = None
            current_content = []
            for line in f:
                line = line.strip()
                if line.startswith("=== ") and line.endswith(" ==="):
                    # 遇到新日期，先保存上一天的
                    if current_date:
                        diary[current_date] = "\n".join(current_content)
                    current_date = line[4:-4]  # 提取日期
                    current_content = []
                else:
                    current_content.append(line)
            # 保存最后一篇
            if current_date:
                diary[current_date] = "\n".join(current_content)
    except FileNotFoundError:
        pass  # 文件不存在，返回空字典
    return diary
```

### 核心：保存日记文件

```python
def save_diary(diary):
    """把日记字典写入文件"""
    with open(DIARY_FILE, "w", encoding="utf-8") as f:
        for date, content in diary.items():
            f.write(f"=== {date} ===\n")
            f.write(content + "\n\n")
```

### 主菜单循环

```python
def main():
    diary = load_diary()
    while True:
        print("\n===== 我的日记本 =====")
        print("1. 写日记")
        print("2. 查看所有日记")
        print("3. 按日期查看")
        print("4. 退出")
        choice = input("请选择: ")

        if choice == "1":
            write_entry(diary)
        elif choice == "2":
            show_all(diary)
        elif choice == "3":
            show_entry(diary)
        elif choice == "4":
            save_diary(diary)
            print("日记已保存，再见！")
            break
```

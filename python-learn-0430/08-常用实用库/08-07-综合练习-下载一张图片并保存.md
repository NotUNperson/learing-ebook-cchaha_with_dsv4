## 正文

### 需求描述

我们来做一个实用的小程序：**下载一张网络图片到本地**。看似简单，但它综合运用了我们学到的 requests、pathlib 和 datetime。

具体功能：
1. 输入一个图片 URL
2. 自动用当前日期创建子文件夹（如 `downloads/2026-04-30/`）
3. 下载图片二进制数据
4. 保存到创建的文件夹中
5. 打印下载摘要（状态码、文件大小、保存路径、耗时）

### 完整代码

```python
"""
图片下载器
从给定 URL 下载图片到本地，自动创建日期子目录。
"""

import requests
from pathlib import Path
from datetime import datetime
import time

def download_image(url, save_dir="downloads"):
    """
    下载一张图片到本地。

    参数:
        url: 图片的 URL
        save_dir: 保存的根目录

    返回:
        保存后的文件路径（Path 对象），失败返回 None
    """
    print(f"开始下载: {url}")

    # ---- 1. 准备保存目录 ----
    today = datetime.now().strftime("%Y-%m-%d")
    target_dir = Path(save_dir) / today
    target_dir.mkdir(parents=True, exist_ok=True)

    # ---- 2. 从 URL 提取文件名 ----
    filename = url.split("/")[-1].split("?")[0]
    if not filename or "." not in filename:
        filename = f"image_{datetime.now().strftime('%H%M%S')}.jpg"
    filepath = target_dir / filename

    # ---- 3. 下载 ----
    start_time = time.time()

    try:
        response = requests.get(url, timeout=30, stream=True)
        response.raise_for_status()  # 检查 HTTP 错误

        # 检查内容类型是否真的是图片
        content_type = response.headers.get("Content-Type", "")
        if "image" not in content_type:
            print(f"警告：Content-Type 是 '{content_type}'，可能不是图片文件")

        # 写入文件
        with open(filepath, "wb") as f:
            f.write(response.content)

        elapsed = time.time() - start_time
        file_size = len(response.content)

        # ---- 4. 打印摘要 ----
        print(f"\n下载成功！")
        print(f"  状态码：{response.status_code}")
        print(f"  文件大小：{file_size} 字节 ({file_size/1024:.1f} KB)")
        print(f"  保存路径：{filepath.resolve()}")
        print(f"  耗时：{elapsed:.2f} 秒")

        return filepath

    except requests.exceptions.Timeout:
        print(f"下载失败：请求超时")
        return None
    except requests.exceptions.HTTPError as e:
        print(f"下载失败：HTTP 错误 - {e}")
        return None
    except Exception as e:
        print(f"下载失败：{e}")
        return None


# ---- 测试 ----
if __name__ == "__main__":
    # 这是一个测试用的公开图片 URL（一张 1px 的透明 PNG）
    test_url = "https://www.python.org/static/img/python-logo.png"
    download_image(test_url)
```

### 代码解析

**要点 1：日期子目录**
```python
today = datetime.now().strftime("%Y-%m-%d")
target_dir = Path(save_dir) / today
target_dir.mkdir(parents=True, exist_ok=True)
```
每天下载的图片放到当天的子目录下，比如 `downloads/2026-04-30/`。这样日积月累也不会乱。

**要点 2：从 URL 提取文件名**
```python
filename = url.split("/")[-1].split("?")[0]
```
URL 格式通常是 `https://example.com/path/to/photo.jpg?size=large`。用 `/` 分割取最后一段，再用 `?` 分割去掉参数部分，得到 `photo.jpg`。

如果 URL 中没有可识别的文件名（比如 `https://example.com/generate`），自动生成一个带时间戳的名字。

**要点 3：二进制下载**
```python
response = requests.get(url, timeout=30, stream=True)
# ...
with open(filepath, "wb") as f:
    f.write(response.content)
```
`response.content` 是二进制数据（bytes），不是文本。打开文件时用 `"wb"`（write binary）模式。

对于大文件（> 几 MB），加 `stream=True` 参数然后分块读更省内存：
```python
with open(filepath, "wb") as f:
    for chunk in response.iter_content(chunk_size=8192):
        f.write(chunk)
```

**要点 4：容错处理**
- `timeout=30`：防止网络问题导致程序永远卡住
- `raise_for_status()`：把 4xx/5xx 状态码转成异常
- `try/except`：捕获各种异常，优雅地打印错误信息
- 检查 `Content-Type`：确认返回的确实是图片

### 扩展：下载多张图片

批量下载也不是问题，加一个循环就行：

```python
image_urls = [
    "https://example.com/photo1.jpg",
    "https://example.com/photo2.jpg",
    "https://example.com/photo3.jpg",
]

for url in image_urls:
    download_image(url)
    time.sleep(1)  # 礼貌地等 1 秒再请求下一张
```

`time.sleep(1)` 是对服务器的基本礼貌：不要太快频繁请求，否则可能被服务器当成攻击而封 IP。

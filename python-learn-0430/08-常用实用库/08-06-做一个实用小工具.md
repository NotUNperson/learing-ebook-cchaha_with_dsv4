## 正文

### 思路：把学过的库组合起来

学了一堆库，单独用每个都很简单。但真正厉害的是**把它们组合起来解决实际问题**。

想象你有一堆积木：长方块（pathlib）、齿轮（datetime）、骰子（random）。单看每个都不起眼，但搭在一起就能垒出一个城堡。

这一节我们就用 datetime + pathlib + random 做一个实用小工具：**文件夹整理器**。它会扫描一个文件夹里的文件，按文件类型分类放到子文件夹里，还能给文件夹生成唯一的名字（防止重名）。

### 工具需求分析

我们的工具要做这几件事：

1. 扫描目标目录下的所有文件
2. 按文件后缀分类（`txt`、`jpg`、`py` 等）
3. 为每类文件创建一个子文件夹
4. 把文件移动到对应的子文件夹
5. 生成一个带时间戳的操作日志

### 完整代码

```python
"""
文件夹整理器
按文件后缀分类整理文件夹，生成时间戳日志。
"""

from pathlib import Path
from datetime import datetime
import shutil  # 用于移动文件

def organize_folder(target_dir):
    """整理文件夹：按后缀分类文件"""
    target = Path(target_dir)

    if not target.exists():
        print(f"目录 {target} 不存在！")
        return

    # 获取所有文件（不包含子目录）
    files = [f for f in target.iterdir() if f.is_file()]
    print(f"找到 {len(files)} 个文件")

    moved_count = 0
    log_lines = []

    for file_path in files:
        # 获取后缀（去掉点，如 .txt -> txt）
        suffix = file_path.suffix.lstrip(".").lower()
        if not suffix:
            suffix = "无后缀"

        # 创建分类目录
        category_dir = target / suffix
        category_dir.mkdir(exist_ok=True)

        # 移动文件
        new_path = category_dir / file_path.name
        # 如果目标位置已有同名文件，加上序号
        if new_path.exists():
            stem = file_path.stem
            for i in range(1, 100):
                new_name = f"{stem}_{i}{file_path.suffix}"
                new_path = category_dir / new_name
                if not new_path.exists():
                    break

        shutil.move(str(file_path), str(new_path))
        log_lines.append(f"[{datetime.now().strftime('%H:%M:%S')}] {file_path.name} -> {suffix}/")
        moved_count += 1
        print(f"  移动：{file_path.name} -> {suffix}/")

    # 写入整理日志
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    log_file = target / f"整理日志_{timestamp}.txt"
    log_file.write_text("\n".join(log_lines), encoding="utf-8")
    print(f"\n整理完成！移动了 {moved_count} 个文件")
    print(f"日志已保存：{log_file.name}")

# 测试（请先创建一个测试文件夹）
# organize_folder("./test_files")
```

### 代码解析

**涉及到的库和知识点**：

| 代码 | 用到什么 | 作用 |
|------|--------|------|
| `Path(target_dir)` | pathlib | 把目录路径变成 Path 对象 |
| `target.iterdir()` | pathlib | 遍历目录下的所有内容 |
| `f.is_file()` | pathlib | 判断是不是文件 |
| `file_path.suffix` | pathlib | 获取文件后缀 |
| `category_dir.mkdir(exist_ok=True)` | pathlib | 创建分类子目录 |
| `shutil.move()` | shutil（内置） | 移动文件 |
| `datetime.now().strftime(...)` | datetime | 生成时间戳，用于日志文件名和日志内容 |
| `\n`.join(log_lines)` | 字符串方法 | 把日志行拼成完整文本 |
| `log_file.write_text()` | pathlib | 把日志写入文件 |

**关键设计点**：

1. **后缀获取**：`file_path.suffix.lstrip(".").lower()` —— 先用 `.suffix` 获取 `.txt`，再用 `lstrip(".")` 去掉点，用 `lower()` 统一小写（防止 `.TXT` 和 `.txt` 被分成两类）。

2. **防重名**：移动文件时，如果目标位置已经有同名文件，就加序号 `_1`、`_2`……直到不重名为止。

3. **时间戳日志**：用 `strftime("%Y%m%d_%H%M%S")` 生成类似 `整理日志_20260430_153045.txt` 的文件名，确保每次整理都产生不同的日志文件。

### 另一个小工具：随机文件名生成器

很多时候我们需要生成一些不重复的临时文件名，比如测试时的临时文件、缓存文件等。这里再展示一个结合了 datetime + random 的小例子：

```python
from datetime import datetime
from pathlib import Path
import random
import string

def random_filename(prefix="tmp", suffix=".txt", with_time=True):
    """
    生成随机文件名。

    参数:
        prefix：前缀（比如 "temp"、"cache"）
        suffix：后缀（比如 ".txt"、".log"）
        with_time：是否包含时间戳

    返回:
        类似 "tmp_20260430_1530_a3x9k2.txt" 的文件名
    """
    # 时间戳部分
    time_part = ""
    if with_time:
        time_part = datetime.now().strftime("%Y%m%d_%H%M") + "_"

    # 随机部分（6 位字母数字混合）
    chars = string.ascii_lowercase + string.digits
    random_part = "".join(random.choices(chars, k=6))

    filename = f"{prefix}_{time_part}{random_part}{suffix}"
    return filename

# 测试生成 5 个文件名
for _ in range(5):
    print(random_filename("cache", ".tmp"))
```

运行结果类似：
```
cache_20260430_1530_x7k2m3.tmp
cache_20260430_1530_a9n4p1.tmp
cache_20260430_1530_z1q8w5.tmp
cache_20260430_1530_b3r6y9.tmp
cache_20260430_1530_m5t7v2.tmp
```

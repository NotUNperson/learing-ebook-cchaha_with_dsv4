# 04-02 定义类和创建对象

## 本节你会学到什么
- 用 class 关键字定义一个类
- 创建类的实例（对象）
- 访问对象的属性

## 正文
### 定义类 -- 就像设计一份学生档案表

学校要给学生建档，老师会先设计一份表格，表格上有这些栏目：

- 姓名：________
- 年龄：________
- 班级：________

这份空白表格就是**类**。它定义了"每一个学生应该有哪些信息"，但还是一片空白，没有填具体内容。

用 Python 代码写出来：

```python
class Student:
    """学生类 -- 定义了学生有哪些属性"""
    pass  # pass 是"暂时什么都不做"的占位符
```

这个类现在还什么都不能做。我们需要让它能存数据。

### 给类添加属性

最简单的办法，是在创建对象之后手动给它添加属性：

```python
class Student:
    pass

# 创建两个学生对象 -- 就像拿出两份空白表格
s1 = Student()
s2 = Student()

# 分别填写 -- 就像在表格里写内容
s1.name = "张三"
s1.age = 15
s2.name = "李四"
s2.age = 14

# 访问属性 -- 用 对象.属性名
print(s1.name)  # 张三
print(s2.name)  # 李四
```

注意：`s1` 和 `s2` 都是 Student 类型的对象，但它们是两个**完全不同**的个体。改 s1 的属性不会影响 s2。

### 对比：类 vs 实例

```python
class Dog:
    pass

# 三只不同的狗 -- 三个不同的实例
dog1 = Dog()
dog1.name = "旺财"
dog1.breed = "金毛"

dog2 = Dog()
dog2.name = "小黑"
dog2.breed = "泰迪"

dog3 = Dog()
dog3.name = "阿黄"
dog3.breed = "土狗"

print(f"{dog1.name} 是 {dog1.breed}")  # 旺财 是 金毛
print(f"{dog2.name} 是 {dog2.breed}")  # 小黑 是 泰迪
```

- `Dog` 是类 -- 一个概念上的"狗"
- `dog1`、`dog2`、`dog3` 是三个实例（对象）-- 三只具体的狗
- `.name`、`.breed` 是属性 -- 每只狗自己的特征

这和我们前面学的字典有点像，但类的功能远比字典强大 -- 后面你会看到。

## 本节小结
class 定义类（模板），`对象 = 类名()` 创建实例，`对象.属性` 访问属性。

## 正文
### __init__ -- 宝宝出生时自动办的"出生登记"

在中国，宝宝一出生，医院就会做"出生登记"：登记姓名、性别、出生时间、体重等信息。这个过程是**自动发生的**，不需要你额外去"启动"它。

`__init__` 方法就像这个"出生登记"：每当你创建一个对象（就像宝宝出生），Python 会**自动调用** `__init__` 方法，帮你把对象的初始信息填好。

注意 `__init__` 的写法：前后各有两个下划线。这是 Python 的约定，表示"这是一个特殊方法，Python 会在特定时刻自动调用它"。

### 对比：有 __init__ 和没有 __init__

```python
# 没有 __init__ —— 手动一个个赋值，麻烦
class Student:
    pass

s1 = Student()
s1.name = "张三"
s1.age = 15
# 如果有 10 个属性你要写 10 行...

# 有 __init__ —— 创建时一把搞定
class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

s1 = Student("张三", 15)   # 一行搞定！
s2 = Student("李四", 14)
print(s1.name, s1.age)     # 张三 15
```

### self 是什么 -- "我自己"的

`self` 是 Python 对象编程里最重要的概念之一。最简单的理解方式：**self 就是"我自己"**。

```python
class Student:
    def __init__(self, name, age):
        self.name = name  # self.name 意思是"我自己的 name"
        self.age = age    # self.age 意思是"我自己的 age"

# 创建两个对象
s1 = Student("张三", 15)
s2 = Student("李四", 14)

# s1 的 self 就是 s1 自己；s2 的 self 就是 s2 自己
# 所以 s1.name 和 s2.name 是不同的
print(s1.name)  # 张三
print(s2.name)  # 李四
```

你可以理解为：
- 定义类时写 `self.name = name`，意思是"每个对象都要记住自己名字"
- `s1.name` 问的是 s1 的名字，`s2.name` 问的是 s2 的名字
- **self 就像每个人说"我"** -- 张三说"我"指的是张三，李四说"我"指的是李四

### __init__ 也可以有默认参数

```python
class Student:
    def __init__(self, name, age=16, grade="一年级"):
        self.name = name
        self.age = age
        self.grade = grade

s1 = Student("张三")                     # age 和 grade 用默认值
s2 = Student("李四", 14, "二年级")       # 全部自定义
print(s1.name, s1.age, s1.grade)  # 张三 16 一年级
print(s2.name, s2.age, s2.grade)  # 李四 14 二年级
```

## 本节小结
__init__ 是对象出生时自动执行的初始化方法；self 就是"我自己"，用来区分不同对象。

# 04-05 继承：父与子

## 本节你会学到什么
- 理解什么是继承以及为什么要用继承
- 定义子类，让它自动拥有父类的属性和方法
- 使用方法重写和 super() 定制子类的行为

## 正文
### 继承 -- 继承家业

想象一个家族企业。老爸开了一家餐馆，积累了一些资产和经营方法。儿子继承了这家餐馆：

- 老爸有的东西（房产、菜谱、员工），儿子**自动就有了** -- 不用从头再来
- 儿子还可以在老爸的基础上**发展新业务**（比如增加外卖服务）
- 如果某个业务儿子做得不一样（比如老爸用现金收银，儿子改用扫码支付），儿子可以**覆盖**老的方法

在 Python 里：
- **父类**（parent class）= 老爸的餐馆
- **子类**（child class）= 儿子的餐馆，自动继承父类的一切
- **方法重写**（override）= 儿子用自己的方式做事

### 继承的基本语法

```python
# 父类 -- 老爸的餐馆
class Restaurant:
    def __init__(self, name):
        self.name = name

    def describe(self):
        print(f"餐馆名称: {self.name}")

    def open(self):
        print(f"{self.name} 开门营业了")

# 子类 -- 儿子的餐馆，继承父类
class FastFood(Restaurant):  # 括号里写父类名
    pass  # 什么都没写，但已经拥有了父类的所有方法！

# 子类自动拥有父类的方法
kfc = FastFood("肯德基")
kfc.describe()  # 餐馆名称: 肯德基  （继承来的）
kfc.open()      # 肯德基 开门营业了  （继承来的）
```

注意 `class FastFood(Restaurant):` -- 括号里的 `Restaurant` 就是父类。这一行就完成了继承。

### 子类添加自己的东西

```python
class FastFood(Restaurant):
    def __init__(self, name, style):
        super().__init__(name)  # 调用父类的 __init__，让父类处理 name
        self.style = style       # 子类新增的属性

    def deliver(self):           # 子类新增的方法
        print(f"{self.name} 提供{self.style}外卖服务")

kfc = FastFood("肯德基", "24小时")
kfc.describe()  # 继承自父类
kfc.deliver()   # 子类自己的方法
```

`super().__init__(name)` 的意思是"调我老爸的 `__init__` 方法，让他帮我处理 name"。`super()` 就是"我老爸"的意思。

### 方法重写 -- 儿子有自己的做法

```python
class FastFood(Restaurant):
    def __init__(self, name, style):
        super().__init__(name)
        self.style = style

    # 重写父类的 describe 方法
    def describe(self):
        print(f"{self.name} ({self.style})")
        print("欢迎光临！")

kfc = FastFood("肯德基", "快餐")
kfc.describe()  # 用的是子类自己的版本，不是父类的
```

子类定义了和父类同名的方法，就会"覆盖"父类的版本。这就像儿子决定用自己的方式经营餐馆，而不是完全照搬老爸的。

## 动手试试
1. 定义一个父类 `Animal`，有 `__init__(self, name)` 和方法 `speak()`（打印"动物在叫"）
2. 定义子类 `Cat`，继承 `Animal`，重写 `speak()` 方法，打印"喵喵喵"
3. 定义子类 `Dog`，继承 `Animal`，重写 `speak()` 方法，打印"汪汪汪"
4. 分别创建 `Cat` 和 `Dog` 的对象，调用 `speak()`
5. 尝试在 Cat 里调用 `super().__init__(name)` 传递名字，让动物有自己的名字

## 本节小结
继承让子类自动拥有父类的功能；super() 调用父类方法；重写可以让子类有不同的行为。

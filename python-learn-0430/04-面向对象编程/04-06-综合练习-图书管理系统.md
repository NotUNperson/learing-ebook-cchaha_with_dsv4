## 正文
### 需求分析

做一个简单的图书管理系统，支持：
- 添加新书到图书馆
- 显示所有藏书
- 借出书籍
- 归还书籍

### 设计 Book 类

每本书需要记录：
- 书名（title）
- 作者（author）
- 是否已借出（is_borrowed）

```python
class Book:
    def __init__(self, title, author):
        self.title = title
        self.author = author
        self.is_borrowed = False  # 新书默认没借出

    def borrow(self):
        """借书"""
        if not self.is_borrowed:
            self.is_borrowed = True
            return f"《{self.title}》借出成功"
        else:
            return f"《{self.title}》已被借出"

    def return_book(self):
        """还书"""
        if self.is_borrowed:
            self.is_borrowed = False
            return f"《{self.title}》归还成功"
        else:
            return f"《{self.title}》本来就在馆内"

    def __str__(self):
        """打印这本书时显示什么（类似于 toString）"""
        status = "已借出" if self.is_borrowed else "在馆"
        return f"《{self.title}》 {self.author} | {status}"
```

### 用列表管理所有书

```python
library = []  # 一个空列表，用来装所有的 Book 对象

def add_book(library, title, author):
    book = Book(title, author)
    library.append(book)
    print(f"添加成功: {book}")

def show_all(library):
    if not library:
        print("图书馆里一本书都没有")
        return
    print(f"\n===== 图书馆藏书 ({len(library)} 本) =====")
    for i, book in enumerate(library, 1):
        print(f"{i}. {book}")
```

### 主菜单循环

```python
library = []
while True:
    print("\n===== 图书管理系统 =====")
    print("1. 添加书籍")
    print("2. 查看所有书籍")
    print("3. 借出书籍")
    print("4. 归还书籍")
    print("5. 退出")

    choice = input("请选择: ")

    if choice == "1":
        title = input("书名: ")
        author = input("作者: ")
        add_book(library, title, author)
    elif choice == "2":
        show_all(library)
    # ... 借书和还书的逻辑类似
    elif choice == "5":
        print("再见！")
        break
```

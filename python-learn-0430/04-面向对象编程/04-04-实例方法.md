### 方法 -- 遥控器上的按钮

你的电视遥控器上有一排按钮：电源键、音量加、音量减、换台。每个按钮对应一个功能。你不需要知道电视内部电路怎么工作 -- 你只要按按钮就行。

对象的**方法**就是遥控器上的按钮。它是定义在类内部的函数，专门用来操作这个对象的数据。

- **属性** = 对象的特征（电视的尺寸、品牌、当前频道）
- **方法** = 对象能做什么（开机、调音量、换台）

```python
class TV:
    def __init__(self, brand):
        self.brand = brand
        self.channel = 1   # 默认频道是 1
        self.volume = 10   # 默认音量是 10

    def change_channel(self, new_channel):
        """换台"""
        self.channel = new_channel
        print(f"切换到第 {self.channel} 频道")

    def volume_up(self):
        """音量加"""
        self.volume += 1
        print(f"音量: {self.volume}")

# 创建一台电视
tv = TV("小米")
tv.change_channel(5)  # 换到第 5 频道
tv.volume_up()         # 音量调到 11
tv.volume_up()         # 音量调到 12
```

注意：每个方法的第一参数都是 `self`。这和 `__init__` 一样 -- `self` 代表调用这个方法的那个对象。

### 方法里用 self 访问属性和其他方法

```python
class Dog:
    def __init__(self, name):
        self.name = name
        self.hunger = 5  # 饥饿度 0-10，越大越饿

    def bark(self):
        """叫一声"""
        print(f"{self.name}: 汪汪！")

    def eat(self):
        """吃东西，减少饥饿度"""
        if self.hunger > 0:
            self.hunger -= 1
            print(f"{self.name} 吃饱了一点 (饥饿度: {self.hunger})")
        else:
            print(f"{self.name} 已经饱了")

    def status(self):
        """汇报自己的状态"""
        print(f"{self.name}: 饥饿度={self.hunger}")
        if self.hunger > 7:
            self.bark()  # 方法里调用自己的另一个方法

dog = Dog("旺财")
dog.status()
dog.eat()
dog.eat()
dog.status()
```

通过 `self.hunger` 访问属性，通过 `self.bark()` 调用同一个对象的另一个方法。`self` 让对象内部的各个部分可以互相访问。

### 多个对象的方法互不影响

```python
dog1 = Dog("旺财")
dog2 = Dog("小黑")

dog1.eat()  # 旺财吃了一口
dog1.eat()  # 旺财又吃了一口
# 小黑一口都没吃，完全不受影响
dog1.status()  # 旺财: 饥饿度=3
dog2.status()  # 小黑: 饥饿度=5
```

每个对象的方法操作的是**自己**的数据，不会影响到其他对象。这就像你按自己家的遥控器，不会把邻居家的电视换台。

## 本节小结
方法是定义在类里的函数，第一个参数必须是 self；对象通过方法操作自己的数据。

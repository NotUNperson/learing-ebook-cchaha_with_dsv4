# 07-01 认识小海龟：会画画的箭头

## 本节你会学到什么

- 让小海龟前进和后退

## 正文

### 用"遥控小汽车"类比 turtle

想象你有一辆遥控小汽车。你拿着遥控器，按"前进"，小车就在地上走一段；按"后退"，小车就往回退；按"左转"，小车就转个弯。小汽车走过的路径会在地面上留下轮胎印，形成一条线。

**Python 的 turtle 库，就是这样一辆"遥控小汽车"**，只不过它在屏幕上跑。屏幕是它的操场，它是一支会移动的画笔（形状是一只小海龟），你发命令让它前进、后退、转弯，它走过的路线就会画出线条来。

turtle 是 Python 内置的标准库，不需要额外安装，直接 `import` 就能用。它原本是为了教小孩子学编程而发明的（Logo 语言的 Python 版），用来入门图形编程再合适不过了。

### 创建画布和让小海龟动起来

创建一个 Python 文件，输入以下代码：

```python
import turtle

# 创建一个画布窗口
window = turtle.Screen()
window.title("我的第一幅 turtle 画")
window.bgcolor("white")

# 创建一只小海龟
t = turtle.Turtle()
t.shape("turtle")  # 把箭头形状改成海龟形状
t.color("blue")    # 设置线条颜色为蓝色

# 让小海龟动起来
t.forward(100)    # 前进 100 像素
t.backward(50)    # 后退 50 像素

# 保持窗口打开
turtle.done()
```

### 逐行解释

```python
import turtle
```
从家里把 turtle 工具箱拿出来。

```python
window = turtle.Screen()
```
创建一张画布（窗口），就像在地上铺一张白纸。

```python
window.title("我的第一幅 turtle 画")
window.bgcolor("white")
```
给窗口起个名字，设置背景颜色为白色。`bgcolor` 可以是颜色名字（如 `"white"`、`"lightblue"`）或十六进制颜色码（如 `"#FF0000"` 红色）。

```python
t = turtle.Turtle()
```
创建一个 Turtle 对象，也就是"小海龟"。`t` 是我们给它的名字，相当于拿到了遥控器。你可以用任何变量名，但 `t` 是惯例写法。

```python
t.shape("turtle")
```
默认形状是箭头，改成 `"turtle"` 就变成一只小海龟的样子，更形象。可选形状还有 `"arrow"`、`"circle"`、`"square"`、`"triangle"`、`"classic"`。

```python
t.color("blue")
```
设置小海龟画线的颜色，蓝色。

```python
t.forward(100)
```
这是核心命令：让海龟朝它**当前朝向**的方向前进 100 个单位（像素）。`forward` 也可以简写为 `fd`。

```python
t.backward(50)
```
让海龟**倒退** 50 像素，不改变朝向，就是退着走。也可以简写为 `bk` 或 `back`。

```python
turtle.done()
```
告诉 turtle "我画完了，窗口可以留着看"。如果少了这一行，画完窗口会立刻关闭，你什么都看不到。也可以用 `turtle.mainloop()`，效果一样。

### 小海龟的坐标系统

turtle 的窗口是一个平面直角坐标系：

- 窗口正中心是原点 `(0, 0)`
- 往右是 x 轴正方向，往上是 y 轴正方向
- 小海龟**默认在原点，头朝右（东）**

所以 `t.forward(100)` 就是从原点向右画一条 100 像素长的线段。

## 动手试试

1. 运行上面的完整代码，观察海龟画了什么。
2. 修改 `forward` 和 `backward` 的参数值（比如改成 200 和 80），看看线段长度怎么变化。
3. 试着把 `t.shape("turtle")` 改成 `t.shape("arrow")`，对比一下。
4. 把 `t.color("blue")` 改成 `"red"`、`"green"` 试试。
5. 把 `window.bgcolor("white")` 改成 `"lightyellow"`（浅黄色），观察背景变化。

## 本节小结

`import turtle` 引入画图工具，`Turtle()` 创建画笔，`forward()`/`backward()` 让它前进后退 —— 就像操控一辆会留下轨迹的遥控小车。

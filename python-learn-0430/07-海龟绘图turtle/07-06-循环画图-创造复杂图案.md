## 正文

### 用"万花筒"类比循环画图

小时候玩过万花筒吗？几片碎彩色纸屑，通过镜面反射重复成绚丽的对称图案。

turtle 的循环画图就是电子版的万花筒：**你写一次"画一个形状，转一个角度"，然后让计算机重复执行几十次，复杂的图案就自动出现了。**

这种技术叫做"算法艺术"：简单的规则重复多次，产生令人惊叹的视觉效果。

### 基础模式：画一个形状 + 转一点角度 + 重复很多次

先看一个最简单的旋转图形：

```python
import turtle

t = turtle.Turtle()
t.speed(0)  # 最快（无动画）

for _ in range(36):
    t.forward(100)
    t.backward(100)
    t.right(10)    # 36 * 10 = 360，正好一圈

turtle.done()
```

这段代码干了什么？每次循环画一条线（前进 100 再退回原点），然后转 10 度。重复 36 次后，36 条线从同一个中心点散开，就像太阳的光芒。

### 画旋转正方形

把"画线"换成"画正方形"：

```python
t.speed(0)
t.pensize(2)

for _ in range(36):
    for _ in range(4):        # 内层循环：画正方形
        t.forward(80)
        t.right(90)
    t.right(10)               # 外层旋转 10 度

turtle.done()
```

你会看到 36 个正方形，每个比前一个稍微旋转了一点，围绕中心排成一个大圆。这就是嵌套循环的威力：**内层画形状，外层控制旋转。**

### 加颜色效果更棒

配合颜色渐变：

```python
colors = ["red", "orange", "yellow", "green", "blue", "purple"]

t.speed(0)
t.pensize(2)

for i in range(36):
    t.pencolor(colors[i % len(colors)])  # 循环取颜色
    for _ in range(4):
        t.forward(80)
        t.right(90)
    t.right(10)

turtle.done()
```

`i % len(colors)` 是取余数，确保即使循环 36 次，也能循环使用列表里的 6 种颜色。

### 画螺旋线

螺旋线是最经典的 turtle 图案之一。核心思路：**每次前进的距离逐渐增大。**

```python
t.speed(0)
t.pensize(2)

for i in range(100):
    t.forward(i * 3)    # 越走越长
    t.right(45)         # 每次转 45 度

turtle.done()
```

把 `right(45)` 改成 `right(59)` 或 `right(89)`，会得到不同风格的螺旋。试试不同的角度值，你会惊叹于数学规律产生的美感。

### 彩色螺旋

```python
t.speed(0)
turtle.bgcolor("black")  # 黑色背景，让颜色更醒目

colors = ["red", "yellow", "blue", "green", "purple", "orange"]

for i in range(200):
    t.pencolor(colors[i % len(colors)])
    t.forward(i * 2)
    t.right(59)       # 59 度产生优美的螺旋

turtle.done()
```

### 画星星图案

```python
t.speed(0)
t.pensize(2)
t.fillcolor("yellow")
t.begin_fill()

for _ in range(5):
    t.forward(100)
    t.right(144)    # 五角星的转弯角度是 144 度
t.end_fill()

turtle.done()
```

为什么五角星是 144 度？五角星的五个顶点均匀分布在圆周上，跟正五边形有关。公式是 180 - 180/5 = 144。画完 5 条边，总转弯 = 5 * 144 = 720 = 两整圈。

更通用的星星公式：`angle = 180 - 180/n`（n 是顶点数）。比如六角星：`180 - 180/6 = 150` 度。

# 07-05 填充颜色

## 本节你会学到什么
- 用 `fillcolor()` 设置填充颜色
- 用 `begin_fill()` 和 `end_fill()` 包围填充区域
- 画有填充色的几何图形

## 正文

### 用"涂色本"类比填充

小时候玩的涂色本还记得吗？一个图案有黑色的轮廓线，你用蜡笔把轮廓内部的区域涂上颜色。

turtle 的填充机制一模一样：
1. 先用线条画出封闭图形（轮廓）
2. 用 `fillcolor()` 选好蜡笔颜色
3. 在画之前喊一声"开始涂色"（`begin_fill()`）
4. 画完轮廓后喊一声"涂完"（`end_fill()`）
5. 电脑自动帮你把封闭区域填满颜色

### 三步实现填充

```python
t.fillcolor("yellow")      # 1. 选一支黄色蜡笔
t.begin_fill()             # 2. 开始涂色
# 画一个封闭图形
for _ in range(4):
    t.forward(100)
    t.right(90)
t.end_fill()               # 3. 涂完
```

运行后你会看到：一个正方形，**边框是笔的颜色，内部填满了黄色**。

### 关键规则

1. `begin_fill()` 和 `end_fill()` 必须是**成对出现**的，就像括号一样。
2. 填充颜色 `fillcolor` 和笔画颜色 `pencolor` 是**分开设置**的。你可以设置红边框 + 黄填充，也可以蓝边框 + 绿填充。
3. 图形必须是**封闭的**（起点和终点重合），否则填充可能溢出或填充出错。
4. `t.color(pen_color, fill_color)` 可以一次设置两种颜色：
   ```python
   t.color("red", "yellow")  # 红边框，黄填充
   ```
   跟下面两行等价：
   ```python
   t.pencolor("red")
   t.fillcolor("yellow")
   ```

### 填充圆形和三角形

```python
# 填充的圆形
t.fillcolor("lightblue")
t.begin_fill()
t.circle(80)
t.end_fill()

# 填充的三角形
t.fillcolor("orange")
t.begin_fill()
for _ in range(3):
    t.forward(100)
    t.right(120)
t.end_fill()
```

### bgcolor() 设置背景色

你也可以改变整个画布的背景颜色：

```python
window = turtle.Screen()
window.bgcolor("lightyellow")  # 浅黄色背景
```

上面这个是设置 Screen 的背景。用 `turtle.bgcolor("lightyellow")` 也可以，它们是同一个窗口。

### 综合演示：画一个彩色靶子

```python
import turtle

t = turtle.Turtle()
t.speed(5)

colors = ["red", "orange", "yellow", "green", "blue"]

for i, color in enumerate(colors):
    t.penup()
    t.goto(0, -30 * (i + 1))  # 每层往下挪一点
    t.pendown()
    t.fillcolor(color)
    t.begin_fill()
    t.circle(30 * (i + 1))    # 半径逐渐变大
    t.end_fill()

turtle.done()
```

这段代码画了一个由红到蓝的五层同心圆靶子。`enumerate` 知识点：它同时提供索引 `i` 和值 `color`，`i` 从 0 开始。

### 如果不闭合图形会怎样？

试试这段代码：

```python
t.fillcolor("red")
t.begin_fill()
t.forward(100)
t.right(90)
t.forward(100)
t.right(90)
t.forward(100)
# 故意不画第四条边
t.end_fill()
```

你会发现：turtle 会**自动用一条直线连接终点到起点，然后填充这个三角形区域**。这不是 bug，是 turtle 的"自动补全"机制。但这并不代表你可以不画封闭图形 —— 自动补全的形状可能不是你想要的。

## 动手试试

1. 画一个红色填充的五角星（或先用三角形练手）。
2. 画三个不同填充色的正方形，并排排列（提示：用 `penup` + `goto` 换位置）。
3. 画一个"停车标志"：红色填充的八边形，加一条白边（粗边框）。
4. 试试 `turtle.bgcolor("black")` 把背景变黑，然后用亮色填充画图形 —— 探索不同背景下的视觉效果。

## 本节小结

`fillcolor()` 选颜色，`begin_fill()` 开始填，`end_fill()` 结束填 —— 就像给涂色本里的轮廓上色一样简单。

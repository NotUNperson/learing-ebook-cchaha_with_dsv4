## 正文

### 思路：把复杂图案拆成"零件"

画复杂图案就像搭积木：先把每个零件做好，再组装起来。

我们要画的花可以拆成：
1. 一个圆形花心
2. 多个花瓣（用 `circle()` 的半圆弧可以画出花瓣）
3. 一根花茎
4. 两片叶子

关键是：**用一个花瓣，旋转重复围成一圈，就是一朵花！**

### 画一朵花

```python
import turtle

t = turtle.Turtle()
t.speed(5)

# ---- 1. 画花瓣 ----
def draw_petal():
    """画一个花瓣：用 circle 画两个对称的半圆"""
    t.fillcolor("pink")
    t.begin_fill()
    t.circle(60, 90)     # 四分之一圆
    t.left(90)
    t.circle(60, 90)     # 又一个四分之一圆
    t.left(90)
    t.end_fill()

# ---- 2. 画一朵完整的花（6 个花瓣围成一圈） ----
t.penup()
t.goto(0, 100)
t.pendown()

for _ in range(6):
    draw_petal()
    t.right(360 / 6)     # 转 60 度画下一个花瓣

# ---- 3. 画花心 ----
t.penup()
t.goto(0, 40)
t.pendown()
t.fillcolor("yellow")
t.begin_fill()
t.circle(25)
t.end_fill()

# ---- 4. 画花茎 ----
t.penup()
t.goto(0, 40)
t.pendown()
t.pencolor("green")
t.pensize(5)
t.setheading(270)    # 朝下
t.forward(200)

# ---- 5. 画叶子 ----
def draw_leaf():
    """画一片叶子"""
    t.fillcolor("lightgreen")
    t.begin_fill()
    t.circle(30, 90)
    t.left(90)
    t.circle(30, 90)
    t.left(90)
    t.end_fill()

# 左边叶子
t.penup()
t.goto(0, -60)
t.setheading(210)     # 朝左下方
t.pendown()
draw_leaf()

# 右边叶子
t.penup()
t.goto(0, -120)
t.setheading(330)     # 朝右下方
t.pendown()
draw_leaf()

t.hideturtle()    # 隐藏海龟，只看画
turtle.done()
```

### 逐段解析

**花瓣函数 `draw_petal()`**：
- `circle(60, 90)` 画半径为 60 的四分之一圆弧
- `left(90)` 调整方向
- 再画一个 `circle(60, 90)`，两个四分之一圆弧拼成一个心形花瓣
- 用粉色填充

**for 循环画 6 个花瓣**：
- 每次调用 `draw_petal()` 后右转 60 度
- 6 次正好一圈，6 片花瓣围成花朵

**花心**：一个黄色实心圆，放在花朵中心位置。

**花茎**：一根朝下的绿色粗线。`setheading(270)` 确保方向朝下。

**叶子函数 `draw_leaf()`**：原理和花瓣一样，两个四分之一圆弧拼成叶子形状，填充浅绿色。

**`t.hideturtle()`**：画完之后把海龟隐藏起来，只留作品，画面更干净。对应 `t.showturtle()` 可以重新显示。

### 拓展：画五角星

```python
def draw_star(size, color_name):
    """画一个填充了颜色的五角星"""
    t.fillcolor(color_name)
    t.begin_fill()
    for _ in range(5):
        t.forward(size)
        t.right(144)
    t.end_fill()
```

你可以把这个函数加到花的图案旁边，或者画一面五星红旗的感觉。

### 关于代码结构

随着图形越来越复杂，把重复的部分抽成函数（`draw_petal`、`draw_leaf`、`draw_star`）会让代码更清晰。这也是编程的一个重要思维方式：**分解问题，化整为零**。

回顾一下你现在掌握的 turtle 技能树：

| 类别 | 方法 |
|------|------|
| 移动 | `forward()` `backward()` `goto()` |
| 转向 | `right()` `left()` `setheading()` |
| 画笔 | `pencolor()` `pensize()` `penup()` `pendown()` |
| 填充 | `fillcolor()` `begin_fill()` `end_fill()` |
| 图形 | `circle()` `dot()` |
| 控制 | `speed()` `hideturtle()` `clear()` |
| 画布 | `bgcolor()` `title()` `done()` |

你已经是一个合格的 turtle 画师了！

# 07-03 画笔控制

## 本节你会学到什么
- 改变画笔颜色 `pencolor()`
- 改变画笔粗细 `pensize()`
- 抬笔 `penup()` 和落笔 `pendown()`
- 控制绘制速度 `speed()`

## 正文

### 用"换不同颜色的笔写字"类比画笔控制

想象你在纸上画画，你有：
- 不同颜色的笔（红的、蓝的、绿的）
- 不同粗细的笔（细的勾线、粗的涂色）
- 把笔抬起来换位置（不画线）
- 手速快慢

turtle 也提供了这些控制。小海龟就像手里握着笔，你可以随时**换颜色、换粗细、抬笔、改变速度**。

### pencolor() 和 pensize()

```python
t.pencolor("red")    # 换成红色笔
t.pensize(5)         # 换成 5 像素粗的笔
```

- `pencolor()`：设置笔的颜色。可以传颜色名字（`"red"`、`"blue"`、`"pink"`），也可以传十六进制码（`"#FF0000"`）。还可以传 RGB 元组，但需要先 `turtle.colormode(255)` 切换色彩模式。
- `pensize()`：设置笔的粗细，单位是像素。默认是 1，数字越大线条越粗。也可以简写为 `width()`。

```python
t.pencolor("#FF0000")  # 红色（十六进制）
t.pensize(3)           # 3 像素粗
t.forward(100)         # 画一条粗红线
```

### 用"抬起笔换位置"类比 penup() 和 pendown()

画画的时候，如果你想换到纸的另一端继续画，你会**把笔抬起来**，移到新位置，再**把笔放下去**。不抬笔的话，移动过程就会画出一条你不想要的线。

turtle 里：
- `t.penup()`：抬笔，移动不画线（简写 `t.up()` 或 `t.pu()`）
- `t.pendown()`：落笔，移动又开始画线（简写 `t.down()` 或 `t.pd()`）

经典用法：移动到新位置开始画新东西。

```python
t.penup()
t.goto(-100, 0)    # 抬笔状态下走到新位置，不画线
t.pendown()
t.forward(200)     # 落笔后从新位置开始画
```

### speed() 设置绘制速度

```python
t.speed(1)   # 最慢
t.speed(3)   # 慢
t.speed(6)   # 中速
t.speed(10)  # 最快
t.speed(0)   # 没有动画，瞬间完成
```

数字范围 1 到 10（整数）。`speed(0)` 是"无动画"，海龟直接出现在终点，画线瞬间完成。画复杂图案时可以用 0，调试时可以用慢速看清海龟的动作。

也可以使用字符串：`"slowest"`、`"slow"`、`"normal"`、`"fast"`、`"fastest"`，但数字更常用。

### 综合演示：用不同颜色和粗细写个"Z"

```python
import turtle

t = turtle.Turtle()
t.speed(5)

# 第一条：粗红线
t.pencolor("red")
t.pensize(5)
t.forward(100)
t.backward(100)

# 抬笔换位置
t.penup()
t.goto(0, -50)
t.pendown()

# 第二条：细蓝线
t.pencolor("blue")
t.pensize(2)
t.forward(100)

turtle.done()
```

### 其他画笔相关方法

| 方法 | 作用 |
|------|------|
| `t.color("red")` | 同时设置笔颜色和填充颜色 |
| `t.fillcolor("blue")` | 只设置填充颜色（下一节讲） |
| `t.reset()` | 清空画布，海龟回到原点，所有设置恢复默认 |
| `t.clear()` | 只清空画布，海龟位置和设置不变 |

## 动手试试

1. 用不同粗细和颜色画几条平行线（提示：用 `penup` + `goto` 换行，`pendown` 后画线）。
2. 尝试用 `speed(1)` 和 `speed(10)` 分别运行同一段绘图代码，感受速度差异。
3. 画一个"田"字格：
   - 先画外围正方形
   - 再抬笔移动到中间画一横一竖
4. 用 `pencolor` 的 RGB 模式试试：
   ```python
   turtle.colormode(255)
   t.pencolor(255, 100, 50)  # 橙红色
   t.forward(100)
   ```

## 本节小结

`pencolor()` 换颜色，`pensize()` 调粗细，`penup()`/`pendown()` 控制画不画线 —— 画笔在手，随心所欲。

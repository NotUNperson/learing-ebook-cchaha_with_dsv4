# 07-04 画基本图形

## 本节你会学到什么
- 用循环画正多边形（正方形、三角形、六边形）
- 用 `circle()` 画圆形和圆弧
- 理解"重复次数 x 角度 = 一周"的规律

## 正文

### 用"搭积木"类比画图形

搭积木时，复杂的造型其实都是一块块基础积木拼起来的。正方形就是 4 块"前进 + 转 90 度"积木的组合，三角形就是 3 块"前进 + 转 120 度"积木。

海龟画图的思路一样：**任何一个正多边形，都可以拆成"重复 N 次：前进一段距离，转一个角度"。**

### 画正方形

```python
import turtle

t = turtle.Turtle()

for _ in range(4):       # 重复 4 次
    t.forward(100)       # 走 100
    t.right(90)          # 转 90 度
```

就这么 4 行逻辑代码，一个正方形就出来了。为什么转 90 度？因为正方形每个角是 90 度，画完四条边总转弯是 360 度（一整圈）。

### 画三角形

```python
for _ in range(3):       # 重复 3 次
    t.forward(100)       # 走 100
    t.right(120)         # 转 120 度
```

等等，为什么三角形转 120 度而不是 60 度？

这是一个容易踩的坑。注意：我们转的是**每一段走完后改变方向的角度**，不是三角形的内角。三角形内角是 60 度，但海龟转弯需要 180 - 60 = 120 度（或者说，转的是外角，即 360/3 = 120 度）。

**通用规律**：画正 N 边形时，每次转的角度 = 360 / N。

| 图形 | N | 每次转的角度 |
|------|---|------------|
| 正三角形 | 3 | 360/3 = 120 |
| 正方形 | 4 | 360/4 = 90 |
| 正五边形 | 5 | 360/5 = 72 |
| 正六边形 | 6 | 360/6 = 60 |
| 正八边形 | 8 | 360/8 = 45 |

所以画任意正多边形的通用代码是：

```python
n = 6  # 边数
for _ in range(n):
    t.forward(100)
    t.right(360 / n)
```

把 n 换成几，就画出正几边形。试试 n=30，你会发现出来的几乎就是一个圆了！

### circle() 画圆和圆弧

如果要画圆，有更简单的方法 —— `circle()`：

```python
t.circle(100)   # 画一个半径为 100 的圆
```

`circle(radius)` 以海龟当前位置为参照，画一个指定半径的圆。圆心在海龟的左侧（逆时针方向画圆）。

`circle()` 还可以画圆弧：

```python
t.circle(100, 180)  # 画一个半径为 100 的半圆（180 度）
t.circle(100, 90)   # 画一个四分之一圆（90 度）
t.circle(100, 360)  # 完整圆（和 t.circle(100) 一样）
```

第二个参数是圆弧的**角度**（也叫做 extent 参数），默认是 360（整圆）。

`circle()` 还有一个可选的第三个参数 `steps`，用来画内切正多边形：

```python
t.circle(100, 360, 6)   # 画一个内切于半径 100 的圆的正六边形
```

### 综合演示：画一个机器人脸

```python
import turtle

t = turtle.Turtle()
t.speed(5)
t.pensize(3)

# 头部（大圆形）
t.penup()
t.goto(0, -100)
t.pendown()
t.circle(100)         # 脸

# 左眼（小圆形）
t.penup()
t.goto(-40, 30)
t.pendown()
t.dot(20, "blue")     # dot() 画一个实心圆点

# 右眼
t.penup()
t.goto(40, 30)
t.pendown()
t.dot(20, "blue")

# 嘴巴（半圆弧）
t.penup()
t.goto(-30, -20)
t.pendown()
t.right(90)
t.circle(30, 180)     # 半圆

turtle.done()
```

这里用了一个新方法 `dot(size, color)`，画一个实心的圆点，不需要抬笔落笔。

## 动手试试

1. 用循环画出正五边形和正六边形。
2. 画一个"一箭穿心"的图案：先画一个正方形，再用 `circle(50, 360, 3)` 在它内部画一个内切三角形。
3. 试试 `circle(50, 270)` —— 画一个四分之三圆，观察圆弧的走向。
4. 挑战：画一个"贪吃蛇"的样子（几个正方形连成一串，就像贪吃蛇的身体）。

## 本节小结

正多边形 = 循环（前进 + 转 360/N 度）；`circle()` 画圆和圆弧，简单又强大。

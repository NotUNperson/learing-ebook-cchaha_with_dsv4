# 07-02 移动与转向

## 本节你会学到什么
- 用 `right()` 和 `left()` 让小海龟转向
- 用 `setheading()` 设置绝对朝向
- 用 `goto()` 移动到指定坐标位置

## 正文

### 用"走迷宫时转弯"类比转向

还记得上一节的遥控小汽车吗？只会直走和后退显然到不了目的地。需要加上**转弯**才行。

在 turtle 里，`right()` 和 `left()` 就是转弯命令。它们让海龟在原地上旋转指定的角度。

- `t.right(90)`：**右转** 90 度（顺时针）
- `t.left(90)`：**左转** 90 度（逆时针）

角度可以传入任意值，比如 `right(45)` 右转 45 度，`left(180)` 左转 180 度（调头）。

注意：转弯时**不画线**，只是改变方向。线是在前进/后退的时候才画。

### right() 和 left() 示例

```python
import turtle

t = turtle.Turtle()

t.forward(100)    # 向右画 100 像素
t.right(90)       # 右转 90 度（现在头朝下）
t.forward(100)    # 向下画 100 像素
t.left(90)        # 左转 90 度（现在头又朝右了）
t.forward(100)    # 再向右画 100 像素

turtle.done()
```

这段代码画了一个倒着的 L 形：先向右，再向下，再向右。

### 用"指南针"类比 setheading()

在现实生活中，你可以用指南针来导航：东是 90 度，南是 180 度，西是 270 度，北是 0 度（或 360 度）。

`setheading()` 就是给海龟设定一个**绝对方向**，不管你之前朝向哪里，调完之后一定朝那个方向。

```python
t.setheading(0)    # 朝右（东）
t.setheading(90)   # 朝上（北）—— 注意 turtle 的默认 90 度是朝上
t.setheading(180)  # 朝左（西）
t.setheading(270)  # 朝下（南）
```

`setheading()` 可以简写为 `seth()`。

**turtle 里的角度规则**（和数学课上一样）：
- 0 度 = 朝右（东方）
- 90 度 = 朝上（北方）
- 180 度 = 朝左（西方）
- 270 度 = 朝下（南方）

`right(90)` 相当于在当前角度上加 90 度，`left(90)` 相当于减 90 度。

### 用"导航到地图上的点"类比 goto()

小海龟不仅可以按角度走路，还可以直接"跳"到某个坐标位置。

`t.goto(x, y)` 让海龟**直线移动到**坐标为 `(x, y)` 的位置。移动过程中会画线。

```python
t.goto(100, 100)   # 从当前位置直接走到 (100, 100)
t.goto(0, 0)       # 走回原点
```

### 综合演示：画一个直角三角形

结合 `forward`、`right` 和 `goto`：

```python
import turtle

t = turtle.Turtle()
t.speed(3)

# 方式一：用车载导航（转向+前进）
t.forward(200)      # 底边
t.left(90)
t.forward(150)      # 竖边
t.goto(0, 0)        # 直接走回起点（斜边）

turtle.done()
```

`t.speed(3)` 设置绘制速度，数值从 1（最慢）到 10（最快），0 表示瞬间完成（无动画）。不设置时默认中等速度。

### 查看当前位置

有两个辅助方法，不画东西，只告诉你信息：

- `t.position()` 或 `t.pos()`：返回当前坐标 `(x, y)`
- `t.heading()`：返回当前朝向角度

```python
print(t.pos())      # 比如 (0.00, 0.00)
print(t.heading())  # 比如 0.0
```

## 本节小结

`right()`/`left()` 相对转向，`setheading()` 绝对转向，`goto()` 直接飞到指定坐标 —— 小海龟的移动能力就此齐全。

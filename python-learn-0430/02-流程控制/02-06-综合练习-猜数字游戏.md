## 正文

### 想一想：你和朋友玩过的猜数字

你和朋友玩过这样的游戏吗——一个人心里想一个数字，另一个人猜。每猜一次，出题的人会告诉你"猜大了"还是"猜小了"，直到猜对为止。

现在我们要用 Python 来实现这个游戏。计算机会"想"一个数字，然后让你猜。

### 让计算机"想"一个数字

首先需要 `random` 模块——它是 Python 自带的一个"随机工具箱"：

```python
import random
secret = random.randint(1, 100)   # 在 1 到 100 之间随机挑一个整数
```

`random.randint(1, 100)` 就像你在 1 到 100 之间随便想一个数——每次都可能不一样。

### 游戏的整体思路

把游戏逻辑用中文写出来：

1. 计算机想一个 1-100 的随机数
2. 玩家猜一个数
3. 如果猜对了 → 恭喜！游戏结束
4. 如果猜大了 → 提示"猜大了"，回到第 2 步
5. 如果猜小了 → 提示"猜小了"，回到第 2 步

这显然是一个 `while` 循环——因为不知道玩家要猜多少次才能对，只知道"猜对了就停"。用 `break` 退出。

### 完整代码

```python
import random

# 1. 计算机想一个数
secret = random.randint(1, 100)
print("我想了一个 1 到 100 之间的整数，你猜猜看？")

guess_count = 0    # 记录猜的次数

# 2. 循环：反复让玩家猜
while True:
    guess = int(input("请输入你猜的数字："))    # input 返回的是字符串，用 int() 转成数字
    guess_count = guess_count + 1
    
    if guess == secret:
        print("恭喜你，猜对了！")
        print("你一共猜了", guess_count, "次")
        break                          # 猜对了，退出循环
    elif guess > secret:
        print("猜大了，再试一次")
    else:
        print("猜小了，再试一次")
```

### 逐行解释

- `import random` — 导入随机数工具箱
- `secret = random.randint(1, 100)` — 生成 1-100 的随机数
- `guess = int(input(...))` — 让玩家输入，`input()` 拿到的是字符串，用 `int()` 转成整数
- `guess_count = guess_count + 1` — 每猜一次，计数器 +1
- `if guess == secret:` — 猜对了的出口
- `elif guess > secret:` — 猜大了
- `else:` — 只有猜小了的情况（因为等于和大于都判断过了）

### 知识回顾

这短短的十几行代码，用到了我们到目前为止学的所有核心知识：

- **变量**：`secret`、`guess`、`guess_count`
- **数字运算**：`guess_count + 1`
- **比较运算**：`==`、`>`
- **if-elif-else**：判断猜对、猜大、猜小
- **while 循环**：反复猜
- **break**：猜对后退出
- **print()**：输出提示信息
- **import**：引入外部模块

你已经有能力写出一个完整的、可以互动的小游戏了。

### 进阶：加更多功能

学会基础版之后，你可以尝试加一些功能：

**限制次数：**

```python
max_guesses = 7  # 最多猜 7 次

while guess_count < max_guesses:
    # ...猜测逻辑
    if guess_count >= max_guesses:
        print("次数用完了！答案是", secret)
```

**提示剩余次数和范围：**

```python
print("你还有", max_guesses - guess_count, "次机会")
```

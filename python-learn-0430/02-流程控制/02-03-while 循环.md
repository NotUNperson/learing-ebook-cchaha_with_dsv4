## 正文

### 吃零食吃到饱

你面前摆着一袋薯片，你的行动逻辑是：

> **只要** 没吃饱，就继续吃一片。

用 Python 写出来：

```python
full = False     # 还没饱

while not full:                      # "只要还没饱"
    print("吃一片薯片……")
    # 吃了几片后可能就饱了
```

这个 `while` 就是**"当……的时候，一直做……"**。

### while 的结构

```python
while 条件:
    # 只要条件为 True，就不停地执行这里的代码
    # 一遍执行完，再检查条件，如果还是 True，再来一遍
```

和 `if` 很像：
- `while` 后面有**冒号**
- 下面的代码要**缩进**
- 只要条件为 True，缩进的代码就被反复执行

**关键区别**：`if` 只执行一次；`while` 会反复执行，直到条件变成 False。

### 一个最简单的 while 循环

```python
count = 1

while count <= 5:
    print("第", count, "次")
    count = count + 1       # 计数器 +1

print("循环结束！")
```

输出：

```
第 1 次
第 2 次
第 3 次
第 4 次
第 5 次
循环结束！
```

逐行理解：
1. `count = 1` — 从 1 开始数
2. `while count <= 5:` — 如果 count 小于等于 5，就执行下面缩进的代码
3. `print(...)` — 打印当前是第几次
4. `count = count + 1` — 计数器加 1（这是关键！）
5. 回到第 2 步，再次检查条件
6. 当 count = 6 时，`6 <= 5` 是 False，循环停止，执行后面的 `print("循环结束！")`

### 计数器的作用

`count = count + 1` 这行代码非常重要。如果没有它，`count` 永远是 1，`count <= 5` 永远是 True，循环就永远不会停——这叫**死循环**。

死循环就像一只在跑步机上永远跑步的仓鼠——跑步机不停，它就停不下来。

```python
# 危险！死循环示例（不要运行！）
# count = 1
# while count <= 5:
#     print("我不会停……")
#     # 忘了写 count = count + 1
#     # 循环永远不会停！
```

如果不小心写了死循环，按 **Ctrl + C** 可以强制停止程序。

### while 的典型使用场景

**场景 1：用户输入密码，直到输对为止**

```python
correct_password = "123456"
user_input = ""

while user_input != correct_password:
    user_input = input("请输入密码：")
    if user_input != correct_password:
        print("密码错误，请重试！")

print("密码正确，登录成功！")
```

这里 `input()` 是让用户在键盘上输入内容的内置函数。程序会停在那里等你输入。

**场景 2：倒计时**

```python
timer = 5

while timer > 0:
    print(timer, "...")
    timer = timer - 1      # 每循环一次减 1

print("时间到！")
```

输出：

```
5 ...
4 ...
3 ...
2 ...
1 ...
时间到！
```

### 用 while 做累加

```python
# 计算 1+2+3+...+100 的和
total = 0         # 初始和为 0
n = 1             # 从 1 开始

while n <= 100:
    total = total + n   # 把当前数字加到总和里
    n = n + 1           # 准备下一个数字

print("1 到 100 的和是：", total)   # 5050
```

这个例子很经典——用循环做重复计算。你不需要手动加 100 次。

## 本节小结

`while` 让程序能反复执行一段代码，直到条件不再满足。关键是要让条件最终变成 False，否则会陷入死循环。

# 02-05 break 和 continue

## 本节你会学到什么
- 用 break 提前退出循环
- 用 continue 跳过当前一次循环
- 理解两者的区别和适用场景

## 正文

### break：提前下班

你在公司上班，规定是下午 6 点下班。但如果手头的事干完了，领导说"活干完了就可以走"。你下午 4 点干完了活，直接走人——这就叫 **break（提前退出）**。

```python
for hour in range(9, 19):      # 9 点到 18 点（模拟上班时间）
    print(hour, "点：工作中...")
    if hour == 16:
        print("  活干完了，下班！")
        break                    # 提前退出循环！
```

输出：

```
9 点：工作中...
10 点：工作中...
11 点：工作中...
12 点：工作中...
13 点：工作中...
14 点：工作中...
15 点：工作中...
16 点：工作中...
  活干完了，下班！
```

注意：17 点、18 点没有出现——`break` 让循环在 16 点就停了。

### continue：跳过这盘菜

你去吃自助餐，一盘一盘地拿。轮到了一盘你不喜欢的苦瓜——你**跳过**它，继续拿下一盘。这就叫 **continue（跳过本次，继续下次）**。

```python
dishes = ["红烧肉", "苦瓜", "糖醋排骨", "青菜", "水煮鱼"]

for dish in dishes:
    if dish == "苦瓜":
        print(dish, "—— 跳过，不吃！")
        continue               # 跳过苦瓜，继续下一盘
    print(dish, "—— 好吃，吃掉了！")
```

输出：

```
红烧肉 —— 好吃，吃掉了！
苦瓜 —— 跳过，不吃！
糖醋排骨 —— 好吃，吃掉了！
青菜 —— 好吃，吃掉了！
水煮鱼 —— 好吃，吃掉了！
```

`continue` 不会停止整个循环，只是跳过"当前这一轮"，直接开始下一轮。

### break vs continue 对比

用一句话记住它们的区别：

- `break`：我不干了，整个循环到此结束。（就像辞职）
- `continue`：这盘菜我不吃，但下一盘还要继续吃。（就像挑食）

### break 的常见用途

**1. 从列表里找到某个东西后就停止**

```python
# 在一堆水果里找"橘子"，找到了就停
fruits = ["苹果", "香蕉", "橘子", "葡萄", "西瓜"]

for fruit in fruits:
    print("正在翻找：", fruit)
    if fruit == "橘子":
        print("找到橘子了！不用找了。")
        break
```

输出：

```
正在翻找： 苹果
正在翻找： 香蕉
正在翻找： 橘子
找到橘子了！不用找了。
```

注意："葡萄"和"西瓜"没有被翻——找到橘子就 `break` 了。

**2. while 死循环中的"安全出口"**

```python
while True:              # True 永远为 True，看起来像是死循环
    answer = input("输入 quit 退出：")
    if answer == "quit":
        print("再见！")
        break            # 但这里有一个 break 出口！
    print("你输入了：", answer)
```

`while True` 加上内部的 `break` 是一种常见的写法——循环本身是无限的，但内部有一个"逃生门"。

### continue 的常见用途

**1. 过滤掉不想处理的元素**

```python
# 只打印偶数，跳过奇数
for i in range(1, 11):       # 1 到 10
    if i % 2 != 0:           # 如果是奇数（%2 不等于 0）
        continue              # 跳过
    print(i, "是偶数")
```

输出：

```
2 是偶数
4 是偶数
6 是偶数
8 是偶数
10 是偶数
```

**2. 忽略特殊情况**

```python
# 处理学生成绩，跳过请假没有成绩的人
scores = [85, -1, 92, -1, 78, 90]   # -1 表示请假

for score in scores:
    if score == -1:
        print("请假同学，跳过")
        continue
    print("成绩：", score)
```

### 一张表记住区别

| | break | continue |
|---|---|---|
| 作用 | 彻底结束整个循环 | 跳过本次循环，继续下一次 |
| 之后的循环还跑吗？ | 不跑了 | 继续跑 |
| 类比 | 下班走人 | 跳过不喜欢吃的菜 |
| 用在 | 找到目标就停、出错就停 | 过滤、忽略特殊情况 |

## 动手试试

2. 用 while True 做一个"猜密码"的小程序：密码是 "open"，输入正确就 break，否则提示"密码错误"
3. 用 for 循环遍历 1 到 10，用 continue 跳过 3 的倍数，打印其他数字

## 本节小结

`break` 和 `continue` 让你能精准控制循环的走向：`break` 提前整个结束，`continue` 跳过当前这轮。它们是循环里的"方向盘"。

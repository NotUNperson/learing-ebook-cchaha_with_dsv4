# 02-02 elif 和 else 多分支

## 本节你会学到什么
- 理解多分支选择的逻辑
- 掌握 if-elif-else 完整结构
- 知道如何处理"以上都不满足"的情况

## 正文

### 考试成绩分等级

你从小到大，每次考试都有等级划分：

- 90 分以上：A（优秀）
- 80-89 分：B（良好）
- 70-79 分：C（中等）
- 60-69 分：D（及格）
- 60 分以下：F（不及格）

这种"多个选项选一个"的情况，叫**多分支判断**。Python 用 `elif` 和 `else` 来实现。

### if-elif-else 就像导航地图

用导航时，导航会给你几个选项：走高速、走国道、走小路。你选一个就走。`if-elif-else` 也一样：从上到下检查，**第一个满足的就执行，后面全跳过**。

```python
score = 85

if score >= 90:
    print("等级：A")
elif score >= 80:
    print("等级：B")
elif score >= 70:
    print("等级：C")
elif score >= 60:
    print("等级：D")
else:
    print("等级：F")
```

运行结果：`等级：B`（因为 85 >= 80，走到第二个分支就停了）

### 逐行理解

- `if score >= 90:` —— 第一个判断：大于等于 90 吗？
- `elif score >= 80:` —— 如果上一个不满足，再判断：大于等于 80 吗？
- `elif score >= 70:` —— 如果上一个还不满足，再判断……
- `else:` —— 以上**全都不满足**时，执行这个

几个关键规则：

1. **开头必须是 `if`**，不能一上来就 `elif` 或 `else`
2. **`elif` 可以有多个**，一个接一个地检查
3. **`else` 是可选的**，放在最后，兜底——相当于"以上都不成立"
4. **只执行第一个满足的**——85 同时满足 >=80 和 >=70，但只会执行 `>=80` 那一个，因为它是第一个命中的

### else 就像一个"兜底"的篮子

```python
age = 15

if age >= 18:
    print("可以投票")
else:
    print("还不到投票年龄")
```

`else` 是"否则"的意思。它没有自己的条件——只要前面所有 `if` / `elif` 都不满足，它就执行。

### 完整的结构模板

```python
if 条件1:
    # 条件 1 满足时执行
elif 条件2:
    # 条件 1 不满足，但条件 2 满足时执行
elif 条件3:
    # 条件 1 和 2 都不满足，但条件 3 满足时执行
else:
    # 以上所有条件都不满足时执行
```

### 生活中的多分支例子

**看时间决定吃什么：**

```python
hour = 14

if hour < 8:
    print("吃早餐")
elif hour < 12:
    print("吃午餐")
elif hour < 19:
    print("吃晚餐")
else:
    print("该吃夜宵了")
```

**看身高决定票价：**

```python
height = 130   # 厘米

if height < 120:
    print("免票")
elif height < 150:
    print("半票")
else:
    print("全票")
```

### 多个 if vs if-elif-else

比较两种写法：

```python
# 写法 1：多个独立的 if（每个都检查）
score = 85
if score >= 90:
    print("A")
if score >= 80:
    print("B")
if score >= 70:
    print("C")
# 输出：B 和 C（两个都打印！因为每个 if 都独立检查）

# 写法 2：if-elif（只执行第一个满足的）
score = 85
if score >= 90:
    print("A")
elif score >= 80:
    print("B")
elif score >= 70:
    print("C")
# 输出：B（只打印一个！命中就停了）
```

大多数"分等级"的场景用 `if-elif-else`；如果每个条件彼此独立、互不影响，用多个 `if`。

## 本节小结

`if-elif-else` 让程序在多个选项中选一个执行。`elif` 可以堆多个，`else` 是兜底的"否则"。从上往下检查，命中即停。

## 正文

### 老师逐个点名

上课时，老师拿着花名册，**一个一个点名**：

> "张三" → "到"
> "李四" → "到"
> "王五" → "到"

老师不需要知道名单上有多少人，也不需要计数器——她只要从头到尾，每个人点一次就行。

在 Python 里，这就是 `for` 循环：

```python
students = ["张三", "李四", "王五"]

for name in students:
    print(name, "→ 到")
```

输出：

```
张三 → 到
李四 → 到
王五 → 到
```

### for 循环的结构

```python
for 变量 in 可遍历的东西:
    # 缩进的代码
    # 每次循环，变量会依次变成"可遍历的东西"中的每一个元素
```

- `for` —— "对于"
- `name` —— 一个变量名，每次循环会被赋予列表中的下一个元素
- `in` —— "在……里面"
- `students` —— 你要遍历的列表（或字符串等）
- `:` —— 冒号，后面是缩进的循环体

翻译成人话：**对于 students 列表中的每一个名字，执行缩进里的代码。**

### for 和 while 的区别

| | while | for |
|---|---|---|
| 何时用 | 你不知道要跑多少次，只知道"什么时候停" | 你知道有多少个东西，要一个个处理 |
| 类比 | 吃到饱为止 | 挨个点名 |
| 需要计数器 | 通常需要自己写 | 不需要，自动一个个取 |

### range()：生成一系列数字

`for` 经常和 `range()` 配合使用。`range()` 是一个"数字生成器"：

```python
# range(5) 生成：0, 1, 2, 3, 4（注意：到 5 之前停止，不包括 5）
for i in range(5):
    print(i)
# 输出：0 1 2 3 4

# range(2, 5) 生成：2, 3, 4（从 2 开始，到 5 之前停止）
for i in range(2, 5):
    print(i)
# 输出：2 3 4

# range(1, 10, 2) 生成：1, 3, 5, 7, 9（从 1 开始，每次加 2，到 10 之前停止）
for i in range(1, 10, 2):
    print(i)
# 输出：1 3 5 7 9
```

`range(起始, 结束, 步长)` 三个参数：
1. 起始值（不写的话默认 0）
2. 结束值（**不包含**它自己）
3. 步长（每次加多少，不写的话默认 1）

`range()` 就像你在本子上写等差数列——从几开始，每次加几，写到几为止（但不包括最后一个）。

### for 遍历字符串

字符串也可以被 for 遍历——每次取一个字符：

```python
word = "Python"

for char in word:
    print(char)
# 输出：
# P
# y
# t
# h
# o
# n
```

这就像把单词的每个字母拆开，一个一个看。

### 用 for 做累加

还记得上一节用 while 算 1 到 100 的和吗？用 for + range 更简洁：

```python
total = 0
for i in range(1, 101):     # 1 到 100（range 不包含 101）
    total = total + i
print("1 到 100 的和：", total)   # 5050
```

不需要手写 `i = i + 1`，range 自动帮你生成下一个数字。

### for 循环最常见的用法

```python
# 打印九九乘法表的一行（2 的倍数）
for i in range(1, 10):
    print("2 x", i, "=", 2 * i)
```

```python
# 给列表中每个元素加前缀
fruits = ["苹果", "香蕉", "橘子"]
for fruit in fruits:
    print("我喜欢吃", fruit)
```

```python
# 统计列表中数字的总和
numbers = [3, 1, 4, 1, 5, 9, 2]
total = 0
for n in numbers:
    total = total + n
print("总和：", total)
```

## 本节小结

`for` 循环适合"逐个处理"的场景——列表中的每个元素、range 生成的每个数字。它不需要你手动管理计数器，自动一个一个取。

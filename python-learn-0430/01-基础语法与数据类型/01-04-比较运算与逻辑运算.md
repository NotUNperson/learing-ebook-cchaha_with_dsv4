# 01-04 比较运算与逻辑运算

## 本节你会学到什么
- 使用比较运算符判断大小和相等
- 理解比较运算的结果是 True 或 False
- 使用 and、or、not 组合多个条件

## 正文

### 比较运算就像"比大小"

你和朋友玩扑克牌，总会比较谁的牌大：
- "我的 10 比你的 8 大" —— 这就是**比较**
- "一样大" —— 这就是**判断相等**

Python 里也有同样的操作，而且结果非常干脆：只有 **True（真）** 或 **False（假）** 两种答案。

| 符号 | 含义 | 生活例子 |
|------|------|----------|
| `>` | 大于 | 身高 170 > 160？是（True） |
| `<` | 小于 | 体重 50 < 60？是（True） |
| `>=` | 大于等于 | 考试 60 分就算及格：60 >= 60？是（True） |
| `<=` | 小于等于 | 身高 120 才能免费乘车 |
| `==` | 等于 | 你的答案和标准答案一样吗？ |
| `!=` | 不等于 | 你和同桌的分数不一样吗？ |

**特别注意：`==` 两个等号是"判断相等"，`=` 一个等号是"赋值"！刚入门最容易搞混的就是这个。**

```python
age = 10        # 一个等号：把 10 装进 age 这个盒子
age == 10       # 两个等号：问"age 等于 10 吗？"答案：True
age == 20       # 问"age 等于 20 吗？"答案：False
```

### True 和 False：程序里的"是"和"否"

`True` 和 `False` 是 Python 里的两种特殊值，叫做**布尔值**（Boolean，以数学家 George Boole 命名）。你不需要记住这个名词，只需要知道：

- `True` = 是、对、真的
- `False` = 否、错、假的

```python
print(5 > 3)       # 5 大于 3 吗？True（是）
print(5 < 3)       # 5 小于 3 吗？False（否）
print(100 == 100)  # 100 等于 100 吗？True（是）
print("abc" == "ABC")   # 大小写不同，False
```

### 逻辑运算：把多个条件串起来

有时候一个条件不够，需要多个条件一起判断。比如：

- "如果天晴 **而且** 是周末，我就去公园" —— 两个条件都满足才去
- "如果饿了 **或者** 到饭点了，我就吃饭" —— 满足一个就去吃

Python 用三个词来做这种组合判断：

**and（与/而且）—— 所有条件都满足，结果才是 True**

```python
weather = "晴"
day = "周末"
print(weather == "晴" and day == "周末")   # True：两样都满足
print(weather == "晴" and day == "周一")   # False：天气满足但日子不对
```

这就像你妈说"写完作业**而且**把房间收拾了才能看电视"——少做一个都不行。

**or（或/或者）—— 满足任意一个条件就是 True**

```python
is_hungry = True
is_meal_time = False
print(is_hungry or is_meal_time)   # True：饿了就算，管它到不到饭点
```

这就像你进超市可以说"有会员卡**或者**有优惠券都可以打折"——有一样就行。

**not（非/取反）—— 把 True 变 False，把 False 变 True**

```python
print(not True)    # False
print(not False)   # True
print(not 5 > 3)   # 5>3 是 True，取反就是 False
```

这就像你说"我没写作业"，`not 写了` = `没写`，把真假反过来。

### 组合使用

你可以把比较运算和逻辑运算组合起来：

```python
age = 15
height = 165
# 判断"年龄大于 12 且身高大于 150"
print(age > 12 and height > 150)   # True

# 判断"年龄小于 10 或身高大于 170"
print(age < 10 or height > 170)   # False（两个都不满足）
```

## 动手试试

1. 判断 `100` 是否大于 `99` 且 `200` 是否小于 `300`（用 `and` 连接两个比较）
2. 创建一个变量 `password = "123456"`，判断它的长度是否大于等于 6（用 `len()` 和 `>=`）
3. 判断 `"hello"` 是否等于 `"Hello"`，然后对结果用 `not` 取反

## 本节小结

比较运算（`>`、`<`、`==`、`!=` 等）用来判断大小和相等，结果是 True 或 False。逻辑运算（`and`、`or`、`not`）用来组合多个判断条件。

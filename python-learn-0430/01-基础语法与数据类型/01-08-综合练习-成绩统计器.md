## 正文

### 想一想：学期末你在做什么

期末考试结束后，老师手里有一份全班同学的成绩单：85、92、78、90、88、76、95、83。

老师要算三样东西：
- **平均分**：大家整体考得怎么样
- **最高分**：谁考得最好
- **最低分**：谁考得最差

如果用手算，要先把所有分数加起来，数一数有多少人，做除法……还要找最高、最低。但用 Python，几行代码就搞定。

### 用列表存储成绩

首先，把成绩放进一个列表：

```python
scores = [85, 92, 78, 90, 88, 76, 95, 83]
```

成绩是一个数字列表，每个数字代表一位同学的成绩。

### Python 的内置"计算器"

Python 自带几个非常有用的函数，专门用来处理列表里的数字：

**sum() —— 求总和**

```python
total = sum(scores)
print("总分：", total)   # 687
```

就像你把购物小票上的每一项价格加起来，`sum()` 帮你把列表里的所有数字加在一起。

**max() —— 找最大值**

```python
highest = max(scores)
print("最高分：", highest)   # 95
```

**min() —— 找最小值**

```python
lowest = min(scores)
print("最低分：", lowest)   # 76
```

### 计算平均分

有了总和和人数，平均分就是除法：

```python
count = len(scores)              # 一共有多少人
average = sum(scores) / count    # 总分除以人数
print("平均分：", average)        # 85.875
```

### 完整的成绩统计器

把以上内容组合起来：

```python
# 全班成绩
scores = [85, 92, 78, 90, 88, 76, 95, 83]

# 统计
total = sum(scores)
count = len(scores)
average = total / count
highest = max(scores)
lowest = min(scores)

# 输出报告
print("========== 成绩统计报告 ==========")
print("全班人数：", count)
print("总分：", total)
print("平均分：", average)
print("最高分：", highest)
print("最低分：", lowest)
print("===================================")
```

输出：

```
========== 成绩统计报告 ==========
全班人数： 8
总分： 687
平均分： 85.875
最高分： 95
最低分： 76
===================================
```

这就是一个完整的"数据进，结果出"的小程序——你把成绩列表放进去，它给你一份统计报告。

### 再加一点花样

除了基础统计，你还可以把成绩排序后显示：

```python
scores.sort()
print("从低到高：", scores)        # [76, 78, 83, 85, 88, 90, 92, 95]

scores.sort(reverse=True)
print("从高到低：", scores)        # [95, 92, 90, 88, 85, 83, 78, 76]
```

你还可以统计及格人数（60 分以上）。这部分需要用到下一模块的 `if` 判断，先留个悬念。

### 这个程序告诉我们什么

回顾模块 1 学到的所有知识，几乎都体现在这个程序里了：

- **数字**：成绩是整数
- **列表**：用列表存多个成绩
- **列表操作**：`len()`、`sort()`
- **算术运算**：总和除以人数
- **变量**：用变量存中间结果
- **print()**：输出统计报告

你已经有能力用 Python 解决一个实际问题了。

2. 在统计报告里再加一项：用最高分减去最低分，得出"分差"（成绩差距有多大）

## 本节小结

把成绩放进列表，用 `sum()`、`max()`、`min()`、`len()` 四个函数就能做出一个成绩统计器。这些函数组合起来，就能让数据"开口说话"。

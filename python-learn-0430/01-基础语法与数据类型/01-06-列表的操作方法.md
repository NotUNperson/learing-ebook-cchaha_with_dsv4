# 01-06 列表的操作方法

## 本节你会学到什么
- 用 append() 向列表末尾添加元素
- 用 insert() 在指定位置插入元素
- 用 remove() 和 pop() 删除元素
- 用 sort() 对列表排序

## 正文

### 列表就像你的抽屉，需要经常整理

你书桌的抽屉不是永远不变的：买新文具要放进去，坏掉的笔要扔掉，东西多了要按大小排列。列表也是一样——它有很多"操作方法"让你随时调整内容。

这些操作方法都是**列表自己的工具**（术语叫"方法"，method），用的时候在列表名后面加 `.` 来调用。

### append()：往末尾加东西

就像你在购物清单的最下面加一行新内容：

```python
fruits = ["苹果", "香蕉"]
fruits.append("橘子")     # 在末尾加上"橘子"
print(fruits)             # ['苹果', '香蕉', '橘子']
```

`append` 这个词的意思是"附加、追加"——永远加在最后面。

### insert()：插队到指定位置

append 只能加在末尾，如果想插到中间呢？用 `insert()`：

```python
fruits = ["苹果", "香蕉", "橘子"]
fruits.insert(1, "葡萄")   # 在索引 1 的位置插入"葡萄"
print(fruits)
# ['苹果', '葡萄', '香蕉', '橘子']
```

`insert(位置, 内容)` —— 第一个参数是"插到哪儿"（索引），第二个参数是"插什么东西"。原来的元素往后挪，给新来的腾位置，就像排队时有人插队一样。

### remove()：按名字删除

你知道要删什么东西，但不知道它在第几个位置：

```python
fruits = ["苹果", "香蕉", "橘子", "香蕉"]
fruits.remove("香蕉")     # 删除找到的第一个"香蕉"
print(fruits)             # ['苹果', '橘子', '香蕉']
```

`remove()` 只删除**第一个**匹配的元素。如果列表里有两个"香蕉"，它只删第一个，第二个还在。这就像你抽屉里有两支一样的笔，你说"把那只笔扔了"，只会扔掉第一眼看到的那支。

如果要删除的东西不在列表里，会**报错**——所以用之前最好确认一下东西确实在列表里。

### pop()：按位置删除（并取出来）

想象你在叠盘子：从最上面拿一个下来。`pop()` 默认从**末尾**取走一个元素：

```python
fruits = ["苹果", "香蕉", "橘子"]
last = fruits.pop()       # 取出最后一个
print(last)               # 橘子
print(fruits)             # ['苹果', '香蕉']
```

`pop()` 会**返回**被删除的元素——也就是说你既可以把它删掉，也可以把它拿出来用。就像从抽屉里抽出一件东西，你可以选择扔掉或者留着用。

你也可以指定位置：

```python
fruits = ["苹果", "香蕉", "橘子"]
second = fruits.pop(1)    # 取出索引 1 的元素（香蕉）
print(second)             # 香蕉
```

### sort()：排排队

把列表里的东西按顺序排列：

```python
numbers = [3, 1, 4, 1, 5, 9, 2]
numbers.sort()
print(numbers)   # [1, 1, 2, 3, 4, 5, 9] — 从小到大

fruits = ["banana", "apple", "cherry"]
fruits.sort()
print(fruits)    # ['apple', 'banana', 'cherry'] — 按字母顺序
```

`sort()` 是**直接修改原列表**的，它不会返回新列表。这就像你直接在抽屉里把东西重新排列，而不是把东西拿出来排好再放回去。

如果想从大到小排：

```python
numbers.sort(reverse=True)
print(numbers)   # [9, 5, 4, 3, 2, 1, 1]
```

### append 和 insert 的区别小结

| 方法 | 作用 | 修改原列表？ |
|------|------|------------|
| `append(x)` | 在末尾添加 x | 是 |
| `insert(i, x)` | 在索引 i 处插入 x | 是 |
| `remove(x)` | 删除第一个值为 x 的元素 | 是 |
| `pop(i)` | 删除并返回索引 i 的元素（默认最后一个） | 是 |
| `sort()` | 从小到大排序 | 是 |

## 本节小结

列表操作方法：`append()` 末尾加，`insert()` 指定位置插，`remove()` 按值删，`pop()` 按位置取走，`sort()` 排顺序。这些都是直接修改原列表的。

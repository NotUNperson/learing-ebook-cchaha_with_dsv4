# 01-03 字符串的常用操作

## 本节你会学到什么
- 用 len() 获取字符串长度
- 用 upper()、lower() 转换大小写
- 用 replace() 替换文字
- 用索引访问字符串中的单个字符

## 正文

### 字符串就像一个"文字工具箱"

你有一支笔，可以在纸上写文字。除此之外，你还有橡皮擦、高亮笔、尺子——这些工具让写文字更方便。

Python 的字符串也自带一个"工具箱"，里面有各种好用的工具。这些工具叫做**方法（method）**，用的时候在字符串后面加一个 `.` 再调用。

### len()：数一数有多少个字

就像你用尺子量一段绳子的长度：

```python
text = "Python 真好玩"
print(len(text))    # 输出：13（注意：空格也算一个字符！）
```

字母、数字、标点符号、空格——在 Python 眼里，每个都算一个字符。`P` 算 1 个，`y` 算 1 个，空格也算 1 个，`真` 算 1 个。

### upper() 和 lower()：大小写转换

`upper()` 把所有字母变成**大写**，`lower()` 把所有字母变成**小写**：

```python
word = "Hello Python"
print(word.upper())   # 输出：HELLO PYTHON
print(word.lower())   # 输出：hello python
```

这就像你写作文时，把标题全部大写表示强调，正文用正常大小写。

注意：`upper()` 和 `lower()` **不会改变原来的字符串**，它们会返回一个新的字符串：

```python
word = "Hello"
loud = word.upper()
print(word)   # 还是 "Hello"，原文没变
print(loud)   # "HELLO"，新的字符串
```

这在编程里叫"**不可变**"——字符串一旦创建就不能修改。所有操作都是生成一个新的字符串。这就像你在复印机上修改复印件，原件永远不会变。

### replace()：把文字换掉

就像用涂改液覆盖错字，写上正确的：

```python
message = "我喜欢吃苹果"
new_message = message.replace("苹果", "香蕉")
print(new_message)   # 输出：我喜欢吃香蕉
print(message)       # 原来的还在，没变：我喜欢吃苹果
```

`replace("旧文字", "新文字")` 在字符串里找到所有"旧文字"，换成"新文字"。同样，它不改变原字符串，而是返回新的。

### 索引：取出字符串中的某一个字

想象一个**停车位**编号：第 0 号车位、第 1 号车位、第 2 号车位……在 Python 里，字符串中的每个字符也都有自己的"车位编号"，这个编号叫做**索引**。

**重要：索引从 0 开始，不是 1！**

```python
text = "Python"
#      012345   <-- 索引编号
print(text[0])   # 输出：P（第 1 个字符，索引是 0）
print(text[1])   # 输出：y（第 2 个字符，索引是 1）
print(text[5])   # 输出：n（第 6 个字符，索引是 5）
```

为什么从 0 开始？这是计算机界的惯例，就像一楼在欧洲叫"Ground Floor"而不是"1st Floor"。刚开始可能不习惯，多用几次就记住了。

**负索引：从尾巴倒着数：**

```python
text = "Python"
print(text[-1])   # 输出：n（倒数第 1 个）
print(text[-2])   # 输出：o（倒数第 2 个）
print(text[-3])   # 输出：h（倒数第 3 个）
```

这就像你从队伍末尾倒着数：最后一个人是 -1，倒数第二个是 -2。

## 动手试试

1. 创建一个字符串变量 `s = " Hello World! "`，用 `len()` 看看它有多长（注意前后空格也算）
2. 把 `"Python"` 的每个字母用索引打印出来（提示：6 个字母，索引 0 到 5）
3. 用 `replace()` 把 `"今天天气真好"` 里的 `"真好"` 换成 `"不太好"`

## 本节小结

字符串自带工具箱：`len()` 量长度，`upper()`/`lower()` 转换大小写，`replace()` 换文字，`[索引]` 取单个字符。记住索引从 0 开始。

# 01-05 列表入门

## 本节你会学到什么
- 理解列表是什么

- 用 len() 获取列表长度

## 正文

### 列表就像你的购物清单

去超市之前，你会拿张小纸条写上要买的东西：

```
- 牛奶
- 面包
- 鸡蛋
- 香蕉
```

这张纸条就是一个**列表**：把一堆东西按顺序排好，写了就不能随意打乱（除非你划掉重写）。

在 Python 里，列表用方括号 `[]` 表示，每一项之间用逗号 `,` 隔开：

```python
shopping_list = ["牛奶", "面包", "鸡蛋", "香蕉"]
print(shopping_list)
# 输出：['牛奶', '面包', '鸡蛋', '香蕉']
```

### 列表是"万能容器"

列表不仅能装文字，什么都能装：

```python
numbers = [1, 2, 3, 4, 5]           # 全是数字
mixed = ["小明", 10, 145.5, True]    # 混合：字符串、整数、浮点数、布尔值
empty = []                           # 空列表（啥都没装）
```

列表就像一个**大抽屉**，每个格子可以放不同的东西。第一格放名字，第二格放年龄，第三格放身高……随便你怎么安排。

### 索引访问：找到列表中的某一项

和字符串一样，列表也用**索引**来访问其中的每一项，**从 0 开始**：

```python
fruits = ["苹果", "香蕉", "橘子", "葡萄"]
#          0        1       2       3     <-- 索引

print(fruits[0])   # 苹果（第 1 个，索引 0）
print(fruits[1])   # 香蕉（第 2 个，索引 1）
print(fruits[3])   # 葡萄（第 4 个，索引 3）
print(fruits[-1])  # 葡萄（倒数第 1 个）
```

想象一个**队列**：你排在队伍里，你是第 1 个人吗？不一定——编程里，排在"最前面"的人编号是 0。这需要一点时间适应，但用多了就习惯了。

### 列表也有长度

```python
fruits = ["苹果", "香蕉", "橘子"]
print(len(fruits))   # 3 个水果
```

`len()` 对列表和字符串都能用——它告诉你"这个容器里有多少个东西"。

### 列表里的元素可以修改

和字符串不同，列表是**可变**的——你可以直接修改其中某个位置的内容：

```python
scores = [85, 90, 78]
scores[1] = 95        # 把索引 1（第 2 个）的成绩改成 95
print(scores)         # [85, 95, 78]
```

这就像你发现购物清单上的"面包"写错了，划掉改成"蛋糕"。列表允许你随时改。

而字符串不行——字符串一旦创建就不能改（还记得吗？所有操作都返回新的字符串）。

## 动手试试

1. 创建一个列表，包含你喜欢的 4 种食物，用 `print()` 打印整个列表
2. 用索引访问第 1 个和第 3 个食物，并打印它们
3. 用 `len()` 看看你的列表有多少个元素，然后用索引 `-1` 访问最后一个

## 本节小结

列表是用 `[]` 创建的有序容器，里面的东西可以按索引（从 0 开始）访问，也可以修改。`len()` 告诉你它包含几个元素。

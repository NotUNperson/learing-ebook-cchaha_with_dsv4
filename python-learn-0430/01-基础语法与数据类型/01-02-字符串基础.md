# 01-02 字符串基础

## 本节你会学到什么
- 理解什么是字符串
- 掌握用单引号和双引号创建字符串
- 学会字符串的拼接和重复

## 正文

### 字符串就是"一串文字"

想象你在一张**便条纸**上写字——"今天下午三点开会"、"记得买牛奶"。这些写在纸上的文字，在 Python 里就叫做**字符串**。

字符串就是一串字符拼在一起。"字符"是啥？就是每个单独的文字、数字、符号——比如 `A`、`你`、`3`、`!` 都是字符。把它们串起来，就是字符串。

在 Python 里，字符串必须用引号包起来（英文引号）：

```python
"Hello"          # 双引号包起来的字符串
'你好'           # 单引号也可以
"Python 真好玩"   # 中英文混合都没问题
```

为什么需要引号？因为计算机要区分**字符串**和**变量名**。看到引号里的东西，计算机知道"这是要原样处理的文字"；没引号的，计算机会去找"有没有这个名字的变量"。

```python
name = "小明"     # "小明" 是字符串（有引号），name 是变量名（没引号）
print(name)       # 输出变量的值：小明
print("name")     # 输出字符串本身：name（注意区别！）
```

### 单引号还是双引号？

大部分情况下都可以，看个人习惯。但有一个小技巧：

**如果字符串里本身包含引号：**

```python
print("小明说: '你好！'")    # 外面用双引号，里面就可以用单引号
print('小明说: "你好！"')    # 外面用单引号，里面就可以用双引号
```

这就像俄罗斯套娃——外层用一种引号，内层用另一种，不会搞混。

### 拼接字符串：把两段文字接起来

就像把两张便条纸用胶带粘在一起：

```python
first_name = "张"
last_name = "小明"
full_name = first_name + last_name   # 用 + 号拼接
print(full_name)    # 输出：张小明

# + 号也可以直接用在 print 里
print("你好，" + "世界！")    # 输出：你好，世界！
```

注意：`+` 用来拼接字符串时，是把两段文字**直接粘起来**，中间不会有空格。如果需要空格，你得自己加：

```python
print("你好，" + " " + "世界！")   # 加了空格
# 更简单的写法
print("你好，" + "世界！")         # 逗号前已经有空格了
```

### 重复字符串：贴一百张一样的小广告

想象你要在一张纸上写满"重要"两个字：

```python
print("重要" * 5)    # 输出：重要重要重要重要重要
print("-" * 30)      # 输出：------------------------------
print("哈哈" * 3)     # 输出：哈哈哈哈哈哈
```

字符串乘以数字，就是把字符串重复那么多次。这个技巧在做分隔线的时候特别好用。

### 字符串里不能直接拼数字

一个常见的错误：

```python
# 这样写会报错！
# print("我有 " + 10 + " 本书")   # 错误！不能把数字直接拼到字符串上
```

字符串 `+` 数字会报错，就像你不能把一张便条纸和一个苹果直接粘在一起。解决办法是用逗号（print 的逗号）或把数字转成字符串（后面会学）：

```python
print("我有", 10, "本书")   # 用逗号就 OK 了
```

## 本节小结

字符串就是用引号包起来的文字。你可以用 `+` 号把它们拼接起来，用 `*` 号把它们重复。（目前版本版本不涉及f-string及str)

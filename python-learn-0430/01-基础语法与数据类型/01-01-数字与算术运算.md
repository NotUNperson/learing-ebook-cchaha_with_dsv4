# 01-01 数字与算术运算

## 本节你会学到什么
- 认识 Python 中的两种数字类型：整数和浮点数
- 掌握加减乘除等基本算术运算
- 理解取余和幂运算

## 正文

### 数字的两种"模样"

你回想一下小学数学课上学过的数字：

- 有些是**整数**：1、2、100、-5，没有小数点的
- 有些是**小数**：3.14、0.5、-2.7，带小数点的

在 Python 里，这两种数字有专门的名称：

- `int`（integer 的缩写）——整数，如 `10`、`-3`、`0`
- `float`（floating-point number 的缩写）——浮点数（带小数点的），如 `3.14`、`-0.5`、`2.0`

> 为什么叫"浮点数"？因为小数点可以"浮动"——3.14 可以写成 0.314 x 10，也可以写成 31.4 x 0.1。小数点的位置是"浮动的"。不用深究这个名字，知道它指的是带小数点的数字就行。

### 算术运算就像超市购物算账

你去超市买东西，收银员要做各种计算：

- **加法**：一包薯片 8 块 + 一瓶可乐 5 块 = 13 块
- **减法**：100 块付出去，找零 87 块
- **乘法**：买了 3 包饼干，每包 6 块 = 18 块
- **除法**：一份零食套餐 20 块，两个人 AA = 每人 10 块

Python 里做这些运算用的符号和数学课一样：

```python
print(8 + 5)     # 加法，输出 13
print(100 - 87)  # 减法，输出 13
print(6 * 3)     # 乘法，输出 18
print(20 / 4)    # 除法，输出 5.0（注意：除法结果总是浮点数）
```

**特别注意两个特殊运算：**

**取余（%）——"分完之后剩多少"**

就像把 10 颗糖分给 3 个小朋友，每人 3 颗，还剩下 1 颗。这剩下的 1 颗就是"余数"。

```python
print(10 % 3)    # 10 除以 3 的余数，输出 1
print(17 % 5)    # 17 除以 5 的余数，输出 2
print(8 % 2)     # 8 能被 2 整除，余数为 0
```

取余非常有用：判断一个数是奇数还是偶数，就用 `% 2` ——余 0 是偶数，余 1 是奇数。

**幂运算（**）——"连乘几次"**

`2 ** 3` 就是 2 的 3 次方：2 x 2 x 2 = 8。这就像细菌繁殖：1 个变 2 个，2 个变 4 个，4 个变 8 个——每次翻倍。

```python
print(2 ** 3)    # 2 的 3 次方，输出 8
print(10 ** 2)   # 10 的平方，输出 100
print(3 ** 4)    # 3 的 4 次方，输出 81
```

### 运算的先后顺序

和数学课一样，Python 也遵循"先乘除，后加减，有括号先算括号"的规则：

```python
print(10 + 2 * 3)      # 先乘后加，2*3=6，10+6=16
print((10 + 2) * 3)    # 先算括号，10+2=12，12*3=36
```

如果不确定谁先谁后，**加括号最安全**。括号里的永远先算。

### 整数和浮点数混合运算

当一个整数遇到一个浮点数时，Python 会自动把结果变成浮点数：

```python
print(10 + 2.5)    # 整数 + 浮点数 = 12.5（浮点数）
print(5 * 2.0)     # 整数 x 浮点数 = 10.0（浮点数）
```

这就像一杯水（整数）倒进半杯水（浮点数）里，结果不会是整杯——肯定是带零头的。

Python 中数字分整数（int）和浮点数（float），用 +、-、*、/、%、** 可以做各种计算，运算顺序和数学课一样。
